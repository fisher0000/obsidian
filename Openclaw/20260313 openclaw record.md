 ### 方式 1：通过 sessions_spawn 创建子任务（推荐）

 Main agent 可以动态创建子 agent 会话来执行特定任务：

 ```javascript

   sessions_spawn({
     runtime: "subagent",
     agentId: "cmc",  // 调用 CMC 专家

     mode: "run",
     runTimeoutSeconds: 300
   })
 ```

 工作流程：

 ```
   用户 → main agent → 任务分解 → sessions_spawn → cmc agent → 返回结果 →
 main 汇总 → 用户
 ```
