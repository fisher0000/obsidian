实验记录审核技能 - 用于自动识别、审核实验记录本图片并录入飞书多维表格。

当用户发送实验记录本图片时，自动执行以下流程：

1. OCR识别图片内容，判断是否为"实验记录本"

2. 提取关键信息：日期、实验地点、实验主题、实验人、页码、复核人

3. 与用户确认识别结果

4. 数据写入对应字段

5. 执行16项审核检查并记录详细结果

6. 将详细审核结果回复给图片发送人

使用场景：

- 收到图片消息且图片抬头为"实验记录本"

- 需要自动审核实验记录合规性

- 需要将实验记录信息归档到多维表格

版本：V03

_# 实验记录审核技能 (V03)_

_## 触发条件_

当收到图片消息时，启动本技能。

_## 执行流程_

_### 阶段1：图片识别与抬头验证_

1. **OCR识别**

- 使用OCR服务识别图片中的文字内容

- 提取所有可见文本

2. **抬头验证**

- 检查识别结果中是否包含"实验记录本"字样

- _**如果不包含**：停止执行，回复"这不是实验记录本，跳过审核"_

- _**如果包含**：继续执行阶段2_

_### 阶段2：信息提取_

从图片中提取以下字段：

| 字段 | 说明 | 示例 |

|------|------|------|

| 日期 | 实验日期 | 2024-03-15 |

| 实验地点 | 实验进行的地点 | 合成实验室A |

| 实验主题 | 本次实验的主题/标题 | 艾普拉唑钠合成工艺优化 |

| 实验人 | 执行实验的人员姓名 | 张三 |

| 页码 | 实验记录页码 | 15 |

| 复核人 | 复核实验记录的人员姓名 | 李四 |


_**提取规则**：_

- 如果某字段无法识别，标记为"未识别"

- 日期格式统一为 YYYY-MM-DD

_### 阶段3：用户确认_

_**只将提取的6项信息**发送给用户：_

  
  
  
实验记录识别结果：  
━━━━━━━━━━━━━━━━━━━━  
日期：2024-03-15  
实验地点：合成实验室A  
实验主题：艾普拉唑钠合成工艺优化  
实验人：张三  
页码：15  
复核人：李四  
━━━━━━━━━━━━━━━━━━━━  
  
请确认以上信息是否正确？  
回复"正确"确认，或回复需要修改的内容。  
  

**处理用户回复**：

- 回复"正确"或"是"或"OK"：进入阶段4

- 回复修改意见：根据意见修改对应字段，重新展示确认，直到用户确认正确

### 阶段4：数据录入

将确认后的数据写入飞书多维表格：

**目标表格**：

- URL: https://j0eukrlohu.feishu.cn/wiki/WZmUwFhkYiIWl0keRo1cPWnynhe?table=tblZ3rCBeqWJdBsp&view=vew8gcCw2H

- 表格ID: tblZ3rCBeqWJdBsp

**字段映射**：

| 提取字段 | 表格字段 | 处理规则 |

|----------|----------|----------|

| 日期 | 日期栏 | 直接填入 |

| 实验地点 | 实验地点栏 | 直接填入 |

| 实验主题 | 实验主题栏 | 直接填入 |

| 实验人 | 实验人栏 | 直接填入 |

| 页码 | 页码栏 | 直接填入 |

| 复核人 | 复核人栏 | 直接填入 |

**注意**：如果某字段识别不出内容，该栏留空。

### 阶段5：审核检查

执行以下16项审核检查：

| 序号 | 审核项 | 检查内容 |

|------|--------|----------|

| 1 | 物料信息 | 物料是否已注明所用的数量、来源以及批号 |

| 2 | 书写工具 | 是否使用蓝色水笔书写 |

| 3 | 环境记录 | 温度、湿度是否已书写 |

| 4 | 基本信息 | 实验主题、实验地点是否已书写 |

| 5 | 日期格式 | 日期格式是否正确(YYYY-MM-DD) |

| 6 | 表格规范 | 绘制表格线条是否工整 |

| 7 | 修改规范 | 书写有误是否正确修改（划改+签名+日期） |

| 8 | 续页标识 | 后接第二页的记录是否有标明 |

| 9 | 骑缝签名 | 黏贴纸张处是否书写骑缝名、日期 |

| 10 | 签名完整 | 操作人、复核人是否已签名 |

| 11 | 空格处理 | 记录是否划掉空格 |

| 12 | 批号规范 | 批号是否编制正确 |

| 13 | 仪器记录 | 是否注明主要仪器设备的型号和编号 |

| 14 | 数据真实 | 实验过程的数据是否如实记录 |

| 15 | 结果完整 | 实验结果和实验结论是否已书写 |

| 16 | 及时填写 | 实验记录是否按时填写（滞后一周内） |

### 阶段6：结果记录与反馈

1. **写入详细审核结果**

- 将详细审核结果（每项的通过/不通过状态及具体问题）写入多维表格的"审核结果栏"

2. **回复图片发送人**

- 将**详细审核结果**完整回复给图片发送人

- 格式示例：

  
  
  
实验记录审核结果：  
━━━━━━━━━━━━━━━━━━━━  
✅ 通过项：  

- 实验主题、实验地点已书写
    

- 日期格式正确
    

- 表格线条工整...
    

❌ 不通过项：  

- 温度、湿度未记录
    

- 未注明仪器设备型号和编号
    

⚠️ 需关注项：  

- 物料信息需进一步核实━━━━━━━━━━━━━━━━━━━━
    

## 工具使用

### OCR识别

- 使用图片OCR服务识别图片文字

### 飞书多维表格操作

- 使用 `feishu_bitable_app_table_record` 工具创建记录

- 使用 `feishu_bitable_app_table_field` 工具查询字段列表

- 表格Token从URL中提取

### 消息交互

- 使用 `message` 工具发送确认消息和审核结果

- 等待用户回复后继续流程

## 异常处理

| 异常情况 | 处理方式 |

|----------|----------|

| OCR识别失败 | 回复"图片识别失败，请重新发送清晰的图片" |

| 非实验记录本 | 回复"这不是实验记录本，跳过审核" |

| 表格写入失败 | 回复"数据保存失败，请检查表格权限" |

| 用户长时间未回复 | 超时提醒后结束流程 |
