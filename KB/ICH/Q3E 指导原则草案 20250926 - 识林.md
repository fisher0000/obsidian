前言</h2><p>浸出物是指在既定的生产和标示的贮藏条件下，从生产组件/系统、包装或给药装置的组件迁移至药品中的化学物质。可提取物是指在特定的实验室试验条件下，从生产组件/系统、包装或给药装置的组件中有意提取的化学物质，这些化学物质是潜在的浸出物。</p><p>本指导原则为浸出物的评估和控制提供了一个整体的框架和流程，进一步扩展了现有 ICH 关于杂质的指导原则，包括 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">新原料药（ICH Q3A）</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3B(R2)_Impurities_in_New_Drug_Products.">新药制剂（ICH Q3B）中的杂质</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3C_R9_Impurities_Guideline_for_Residual_Solvents_20240124">残留溶剂（ICH Q3C）</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3D_R2_Elemental_Impurities_20220426">元素杂质（ICH Q3D）</a> ，以及 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">DNA 反应性（致突变）杂质（ICH M7）</a> 。本指导原则的框架遵循 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q9_R1_Quality_Risk_Management_20230118">ICH Q9</a> 中描述的风险管理原则。本指导原则涉及了材料表征和工艺理解的内容，然而其主要目的是通过评估和控制药品中的浸出物来保障患者安全和产品质量。由于材料工程、新装置、新生产模式和新型治疗方式的快速发展，本指导原则主要在科学和监管领域内提供前瞻性的原则和概念。</p><h2>2. 适用范围</h2><p>本指导原则适用于新的药物制剂（包括细胞和基因治疗产品）中浸出物的风险评估和控制。需要上市许可且符合化学药品或生物制品定义的药械组合产品同样在本指南的适用范围内。</p><p>本指导原则主要关注有机浸出物。尽管元素杂质分析的推荐方法包含在本指导原则的范围内，但元素杂质浸出物的安全性评估应参照 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3D_R2_Elemental_Impurities_20220426">ICH Q3D</a> ，因此不在此指导原则的适用范围内。</p><p>本指导原则同样适用于已批准产品中可能影响浸出物谱或患者暴露量的变更，例如涉及处方、生产工艺、给药剂量和（或）包装系统的变更（即，生命周期管理）。本指导原则不适用于产品因污染或掺假而引入的外源性物质、外来物质或异物。</p><p>本指导原则不适用于草药、动物或植物来源的粗制品。对于这些液体剂型的产品，可适用区域性的要求。</p><p>本指导原则不适用于临床研究阶段使用的产品。然而，当患者存在高风险的情况下，可采用本指导原则的总体原则来支持临床研究。</p><p>原则上，除特殊考虑外，放射性药物不在本指导原则的适用范围内。</p><p>本指导原则不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 的生产或贮藏。关于液体或半固体 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">活性药物成分</a> （APIs）包装组件的特殊考虑，请参见第 3.4.1 节。</p><h2>3. 可提取物和浸出物的风险评估与控制</h2><h2>3.1 总体原则</h2><p>本指南旨在构建一个整体的框架，以识别、评估和控制与浸出物相关的风险，从而确保药品的安全性、有效性和质量属性满足要求。图 1 目的是为产品开发过程中的相关考量提供指导，这些考量贯穿于产品注册前的阶段以及整个 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 管理过程中的持续性质量管理环节。</p><p>可提取物与浸出物（E&amp;L）的质量风险管理流程需要制定全面的策略。该策略应充分运用先验知识，并且深入理解生产/包装组件以及药品的目标属性和关键属性，同时还需充分考虑药品的生产工艺和储存条件。分析化学与安全评估方面专家之间的密切合作对于知识共享以及 E&amp;L 质量 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%A3%8E%E9%99%A9%E7%AE%A1%E7%90%86">风险管理</a> 流程的开发至关重要。每个产品都应启动一个质量风险管理流程，包括其各自的风险评估、风险控制以及生命周期管理流程。</p><h2>3.2 多因素风险矩阵的概念</h2><p>在进行浸出物的总体风险评估和控制时，必须充分考虑风险的多维度特性，既涵盖药品质量方面，也包括安全性方面。就药品质量而言，关键考量维度包括：</p><ul><li><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E4%BA%A7%E8%AE%BE%E5%A4%87">生产设备</a> 或包装组件与处方之间相互作用的可能性，</li></ul><ul><li>可能导致产生浸出物的设备或组件的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%90%86%E5%8C%96%E7%89%B9%E6%80%A7">理化性质</a> ，以及使用前对组件的预处理，</li></ul><ul><li>生产和贮藏条件，包括但不限于表面积与溶液体积比、温度、接触时间、下游去除步骤的邻近程度及其去除潜在浸出物的能力。</li></ul><ul><li>处方的浸出倾向，包括但不限于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">API</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PH">pH 值</a> 、有机共溶剂和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%A1%A8%E9%9D%A2%E6%B4%BB%E6%80%A7%E5%89%82">表面活性剂</a> /螯合剂等。</li></ul><p>安全性评估维度与浸出物可能引发的危害相关，包括与暴露相关的多种因素，例如： <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 、相关患者人群、最大给药剂量、给药频率和/或给药间隔，以及终生累计最大治疗时间等。</p><p>图 2 展示了不同维度（未包含全部）的相对风险。药品浸出物的总体风险是综合考虑所有维度来确定的。</p><p>根据预期的风险并运用先验知识，可采用多种方法来进行风险评估和控制，包括从相关食品接触安全或药典标准/法规的符合性，到更全面的 E&amp;L 表征和安全性风险评估（参见附录 1）。对于口服制剂，当进行充分论证后（如，拟定用途符合区域食品接触相关法规，药物制剂的浸出倾向与所参考区域法规中所列举的情形相似或更低，且所有特定的测试结果均符合可接受标准），符合相关区域食品接触的安全法规可能足以支持聚合物生产设备/系统及包装系统的安全性和质量。</p><p>对于其他 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%89%82%E5%9E%8B">剂型</a> ，或成分、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 和使用限制不符合食品接触法规要求的口服制剂，通常需进行可提取物/浸出物评估。</p><p>上述风险矩阵及风险因素突显了浸出物相关风险评估的复杂性。了解各相关因素的风险水平是风险评估过程的一部分，并有助于指导生产及包装组件的选择以及总体风险评估/控制策略的制定。</p><h2>3.3 风险评估</h2><p>基于风险管理流程（图 1，第 3.1 节）、多维风险矩阵（图 2，第 3.2 节）以及 E&amp;L 风险评估和风险控制的典型工作流程（图 3 和 4，附录 1）的描述，风险评估可以概括为以下三个基本步骤：</p><ul><li><u>第 1 步</u> - <u>危害源识别：</u> 基于先验知识（如组件的使用经验、既往已开展的试验等）和/或可提取物及浸出物试验，识别可能从直接接触面（如生产组件/系统、包装系统及给药装置组件）或间接接触面（如次级包装，半透性组件标签所用的油墨或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%B2%98%E5%90%88%E5%89%82">粘合剂</a> ）迁移至制剂中的潜在浸出物。</li></ul><ul><li><u>第 2 步</u> - <u>风险分析：</u> 量化制剂中浸出物的潜在水平以及患者的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9A%B4%E9%9C%B2%E9%87%8F">暴露量</a> 。</li></ul><ul><li><u>第 3 步</u> - <u>综合风险评价：</u> 评估对产品质量、安全性和有效性的风险，以确定所选生产组件/系统和包装系统是否满足拟定用途。</li></ul><h2>3.4 风险控制</h2><p>如果综合风险评估表明需要降低风险，那么可以采取的措施包括但不限于：更换组件/供应商、对组件进行预清洗、对生产设备进行再冲洗，以及增加额外的纯化/分离步骤。应通过可提取物和/或浸出物研究来确认/验证最终实施的风险减少措施是否充分。</p><p>当组件经确认满足拟定用途后，需对其建立适当的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8E%A7%E5%88%B6%E7%AD%96%E7%95%A5">控制策略</a> ，包括但不限于对于组件 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%8E%A7%E5%88%B6">质量控制</a> 至关重要的常规 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">GMP</a> 规范。控制策略应包括：</p><ul><li>建立全面的质量控制，包括组件的可接受标准、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%96%E6%A0%B7">取样</a> 计划（如适用）。</li></ul><ul><li>与组件供应商签订适当的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E5%8D%8F%E8%AE%AE">质量协议</a> ，其中应包括组件全 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 质量控制的相关内容，特别是涉及任何可能影响可提取物谱的配方和/或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%8F%98%E6%9B%B4">生产工艺的变更</a> 。</li></ul><p>有关 E&amp;L 风险评估和风险控制的典型工作流程参见附录 1，包括生产组件/系统（图 3，附录 1）以及包装系统和给药装置组件（图 4，附录 1）的组件确认。通常，需要对包装和给药装置组件进行可提取物和浸出物研究。在某些特定情况下，如果理由充分，也可以提出其他替代方法。</p><p>包装和给药装置组件浸出物风险识别的原则和规范以及降低风险所采用的策略，同样适用于与制剂相关溶液直接接触的聚合物生产设备组件。可提取物研究的设计应体现最差生产条件（例如，最小批量下最长接触时间、最高温度和压力）。通常认为由于生产组件/系统与制剂的接触时间相对较短，药液体积与组件接触的表面积比值相对较大，制剂中浸出物来源于生产组件/系统的可能性低于包装和给药系统组件。在上游生产工艺步骤中引入的浸出物也许能够通过下游步骤（例如，精制/纯化）进行清除，从而降低浸出物最终引入制剂终产品的风险。这些因素应在选择和确认生产设备以及进行质量调查时进行考虑。</p><p>对于生产组件/系统而言，如果所有可提取物的检出量均小于等于制剂的分析评价阈值（AET），并且未检出 1 类浸出物时（见第 5 节），那么其浸出物风险可被认为极低且可接受。提取研究中使用的分析方法应符合第 4.3 节中所列出的标准。</p><p>当生产组件/系统的可提取物检出量高于 AET 时，可以进行可提取物的定性鉴别和定量检测来评估浸出物风险，前提是需采用与鉴别得到的可提取物结构一致的适宜 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AF%B9%E7%85%A7%E5%93%81">对照品</a> 进行定量检测。如果无法获得结构一致的对照品，则可使用具有相似分析响应的化合物。如果采用上述定量方法测得的可提取物浓度低于相关可接受的安全性水平（见第 6 节），则认为浸出物相关的安全性风险可忽略不计。对于生产设备中检出量大于 AET 的可提取物，也可通过进行浸出物的安全性评估这种替代方案进行确认。</p><p>对于药品包装组件/系统，如果患者安全风险可以通过先验知识进行充分控制（例如，已建立可提取物/浸出物相关性，拟申报制剂与已获批上市制剂浸出倾向相似），或未检出/检出少量大于 AET 但小于其安全性阈值的可提取物（如 3 类浸出物，参见第 6 节），则可考虑提交简化的研究数据包。表 A.1.2（附录 1）提供了与图 2（第 3.2 节）相关的总体风险较低的示例，在进行充分论证的前提下可考虑提交简化数据包。当申请提交简化数据包时，建议与相关区域监管机构/卫生监管部门进行沟通，以确认所用方法可被接受。</p><p>如果鉴别得到的可提取物可能通过化学转化（即，通过化学降解和/或与处方组分相互作用）产生安全性风险更高的化合物，或无法充分鉴别和/或定量测定大于 AET 的所有可提取物峰，则应进行浸出研究来解决这些问题并证明组件的可接受性。</p><h3>3.4.1 特殊考虑</h3><p>对于使用多个组件的生产工艺，尤其是那些由相同或相似材料构成的组件，应评估浸出物累积的风险。</p><p>质量风险评估及其控制策略，如适用时，还应涵盖储存液体 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 或半固体原料药 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%B9%E5%99%A8">容器</a> 的潜在浸出物。</p><p>尽管在冷冻状态下浸出极少，仍应评估冷冻前和解冻后包装组件/系统的潜在浸出风险。</p><p>此外，对于生物制品和生物技术衍生产品，风险识别和风险降低措施还可包括：</p><ul><li>评价具有反应性的浸出物与处方组分之间潜在的相互作用，可能导致对产品质量、安全性和/或有效性产生不良影响。当发现已知的反应性浸出物对产品的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E8%B4%A8%E9%87%8F%E5%B1%9E%E6%80%A7">关键质量属性</a> 产生影响时，应考虑化学修饰的潜在机制（例如变性、聚集或降解）。</li></ul><ul><li>对于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%B6%B2">原液</a> 的生产，浸出物可在最后一个 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BA%AF%E5%8C%96">纯化</a> 步骤中去除。因此，质量风险评估通常重点关注其后续的生产过程。</li></ul><h2>3.5 文件与合规性</h2><p>注册申请应包含可提取物/浸出物研究的合理性论证、相关研究报告、超出 AET 物质的安全性评估以及必要的风险控制策略。为支持生产和包装组件/系统的可接受性而进行的可提取物和浸出物研究应包含在申报资料中（如 <a href="https://lib.shilinx.com/wiki/index.php?title=M4Q(R1)_Quality">ICH M4Q</a> 所述）。应提供充分的浸出物数据，以确保制剂在整个货架期内的安全性和质量满足要求。通常情况下，可以与已有的稳定性数据一并提交浸出物研究结果，在事先征得相关区域监管部门同意后，可在批准后提交额外数据。应按照本指导原则第 3.3 节相关内容，对一次性使用和多次使用的生产组件/系统、初级包装组件和给药装置组件进行质量风险评估。对于半渗透性包装材料，还应视情况对次级包装进行评估。</p><p>应提供可提取物研究和浸出物研究资料，以及相应的评估报告，报告中通常应包括可提取物研究所用的分析方法和提取条件（溶剂、温度、时间、表面积/体积比等）的选择依据，以及浸出物研究中样品制备方法以及 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 。此外，应提供定量分析方法以及方法适用性研究（例如， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A3%80%E6%B5%8B%E9%99%90%E5%BA%A6">检测限</a> （LOD）、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%9A%E9%87%8F%E9%99%90%E5%BA%A6">定量限</a> （LOQ）、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%93%E5%B1%9E%E6%80%A7">专属性</a> 、线性、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%B2%BE%E5%AF%86%E5%BA%A6">精密度</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%87%86%E7%A1%AE%E5%BA%A6">准确度</a> 和重复性）。申报资料中应提供所有大于 AET（见第 5 节）的可提取物和浸出物的信息，包括化学名称、结构、CAS 登记号（如有）和观察到的检出水平。对于浸出物（或用于组件/系统确认而测定的可提取物），应包括第 6 节所述的安全风险评估。</p><p>除质量风险评估外，申报资料中还应酌情提供浸出物与可提取物的相关性内容（参见第 4.6 节）。最后，应提供实施前后收集的数据来证明风险降低措施（例如包装和给药组件/系统的预清洗或生产组件/系统的再冲洗）的合理性。</p><h2>3.6 风险回顾/生命周期管理</h2><p>本节描述了在药品生命周期内可能需要对浸出物谱进行重新评价的变更类型。以下是潜在变更的非详尽列表，以及这些变更对患者浸出物暴露水平潜在影响的说明。因此，可能需要基于新的研究和/或现有信息对这些变更进行考量和科学论证。</p><p><u>新信息：</u> 如果与材料适用性相关的新数据和/或信息表明存在潜在风险，和/或如果获得了关于某浸出物的新的患者安全性信息，则可能需要更新评估。</p><p><u>制剂处方的变更：</u> 制剂的变更可能导致现有与制剂直接接触的生产组件/系统，和/或直接接触的包装，和/或给药装置组件引入的浸出物发生变化。例如， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> / <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%A1%A8%E9%9D%A2%E6%B4%BB%E6%80%A7%E5%89%82">表面活性剂</a> 组成或浓度的变更可能影响浸出物的种类和含量。</p><p><u>与原料药和/或制剂直接接触的包装系统、给药装置或生产组件/系统的变更：</u> 在药品有效期内，当与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> （主要是液体和/或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%B6%E5%93%81">生物制品</a> ）或制剂直接接触的材料在配方、供应商、生产工艺、几何构造或预处理等方面发生变更时，浸出物谱可能改变。此外，对于某些产品，其非直接接触的包装组件也可能引入潜在浸出物。</p><p><u><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%8F%98%E6%9B%B4">生产工艺的变更</a> ：</u> 工艺条件的变更可能导致现有制剂直接接触的材料引入不同的浸出物或导致浸出物含量发生改变。例如，溶剂体系、接触时间、温度、压力、pH 值、清洗/灭菌工艺、表面积/体积比、操作前准备（如冲洗）等条件的变更，可能影响浸出物的组成和含量。</p><p><u>可能影响患者暴露量的变更：</u> 药物的剂量方案、治疗持续时间、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 和患者人群（如，老年患者/儿童患者）等 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%98%E6%9B%B4">变更</a> ，均可能改变患者对先前已知浸出物的暴露量评估值。所有这些变更均可能影响浸出物暴露量评估和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E7%90%86%E5%AD%A6">毒理学</a> 风险评估中所依据的基本前提。</p><p><u>可能影响患者获益风险比的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%80%82%E5%BA%94%E7%97%87">适应症</a> 变更：</u> 例如，适应症由抗肿瘤变更为风湿免疫疾病。</p><h2>4. 化学试验和评估</h2><h2>4.1 先验知识</h2><p>先验知识可包括在进行化学试验之前需获取的有用信息，包括供应商提供的信息以及其他制剂产品和生产工艺的相关信息。这些信息可能包括：</p><ul><li>配方（例如，基础聚合物和共聚物、各添加剂，如增塑剂、加工助剂、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%82%AC%E5%8C%96%E5%89%82">催化剂</a> 、抗氧剂）</li></ul><ul><li>食品接触合规性</li></ul><ul><li>无有意添加特定（如，未经授权）化合物的声明</li></ul><ul><li>药典检测</li></ul><ul><li>可用的可提取物研究</li></ul><ul><li>生物反应性试验</li></ul><ul><li>加工或预处理步骤（如，灭菌、清洁、冲洗、硅化、表面处理）</li></ul><ul><li>既往使用历史，包括其他类似制剂、生产工艺和/或接触条件的历史使用情况</li></ul><h2>4.2 组件选择</h2><p>药品生产商需负责制定符合监管预期的药品生产、包装、储存和递送等相关要求，以向特定患者群体提供安全、有效的药品。特定材料或组件的浸出物风险水平与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%89%82%E5%9E%8B">剂型</a> 相互作用的可能性相关。例如，与浸出倾向较高的剂型（如液体）相互作用的组件，其风险可能高于与浸出倾向极低的剂型（如非冻干固体）相互作用的组件。可通过从供应商处获得的信息（例如，可提取物报告、药典要求符合性等），或补充适宜的额外试验，来进行风险评估以及可提取物/浸出物研究开发，从而证明组件选择的合理性。可提取物、浸出物和模拟浸出物研究的总结见表 A.2.1（附录 2）。</p><h2>4.3 可提取物研究</h2><p>可提取物研究是从受试物中提取化学实体的过程。合理的可提取物研究采用的溶剂及提取条件需与制剂在最差生产或放置条件下的预期浸出倾向相关，并采用多种互补分析技术建立全面的提取物特性。合理的可提取物研究关键特性包括：</p><ul><li>建立和应用制剂特定的 AET，用于指明需进行鉴定并作为潜在浸出物处理的可提取化学实体。对组件或组装系统进行测试，包含可代表最终成品组件或系统使用的加工和处理操作（如，灭菌、成型与制造条件、清洁、硅化）。</li></ul><ul><li>选择适宜的提取介质，包括与制剂处方相关且具有代表性的不同 pH 值和极性的溶剂（如辅料、表面活性剂）。</li></ul><ul><li>代表制剂在生产过程中，或包装组件/系统在货架期内最差条件下（例如，接触面积、温度、持续时间）可能产生的浸出物。</li></ul><ul><li>所采用的分析方法在与提取研究目的相适应的水平上进行充分确认。</li></ul><ul><li>包括针对挥发性、半挥发性和非挥发性有机可提取物及元素可提取物的适宜分析方法。</li></ul><ul><li>可提取物报告详细描述分析方法的内容。</li></ul><p>根据对结构材料和质量的理解，应对潜在 1 类浸出物（参见第 6.2 节“浸出物分类”）进行特定目标检测，酌情开展风险分析。对潜在 1 类浸出物的分析应遵循定量可提取物研究（第 4.3.2 节）或浸出物研究（第 4.4 节）的描述。</p><h3>4.3.1 半定量可提取物研究</h3><p>半定量可提取物研究可适用于随后将进行浸出物研究，以确定材料在预期用途中可接受性的情形。半定量可提取物研究的目的是确定哪些可提取物可能作为浸出物存在于制剂中。半定量可提取物研究的关键特点包括：</p><ul><li>分析方法需采用若干分析中常见的可提取物或浸出物相关对照品进行确认。</li></ul><ul><li>在计算制剂特定 AET 时，使用分析不确定因子（UF；第 5.1 节）。</li></ul><ul><li>采用相关的对照品进行可提取物的定量。</li></ul><p>半定量可提取物如大于 AET，则可作为后续定量可提取物研究或浸出物研究的目标化合物。</p><h3>4.3.2 定量可提取物研究</h3><p>在半定量可提取物研究中观察到的可提取物水平高于 AET 时，如果为了确认生产组件/系统和某些低风险包装组件/系统（分别参见附录 1 表 A.1.1 和 A.1.2），则需进行定量可提取物研究来量化这些特定的可提取物。定量可提取物研究的关键特点包括：</p><ul><li>高于 AET 的可提取物需进行确定的鉴别。</li></ul><ul><li>采用分析响应相同或相似的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AF%B9%E7%85%A7%E5%93%81">对照品</a> ，对已鉴别的高于 AET 的可提取物进行定量测定。</li></ul><ul><li>已鉴别的高于 AET 的可提取物所用的定量分析方法应采用该特定的标准物质进行确认。</li></ul><p>如果经充分鉴别并定量的可提取物超出其规定限度（例如，适用安全性阈值或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">每日允许暴露量</a> （PDE），则应进行浸出物研究，以证明该化合物作为浸出物时低于其规定限度。此外，当存在高于 AET 的可提取物无法进行确定性鉴别时，也可采用浸出物研究来评估其质量风险。</p><h2>4.4 浸出物研究</h2><p>用于支持制剂注册申报的浸出物研究旨在模拟整个拟定有效期和使用期间的实际生产条件及预期贮藏条件。在有效期内和使用期间，应评估多个时间点来表征浸出物变化趋势并估计最大检出量。在稳定性放置期间，对实际包装的制剂开展包装系统的浸出物评估，并可包括加速放置条件。对于包装系统的浸出物研究应包含采用与预期用于市售产品一致的实际包装和给药系统生产的多个制剂稳定性注册批次和/或研发批次。如果无法开展多批次研究，可提出替代方法并说明理由。在浸出物研究中使用与可提取物研究相同批次的组件可能有助于在可提取物和浸出物之间建立更有意义的相关性。应对特定目标浸出物的分析方法进行适当验证，以确认其灵敏度、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%93%E5%B1%9E%E6%80%A7">专属性</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%87%86%E7%A1%AE%E5%BA%A6">准确度</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%B2%BE%E5%AF%86%E5%BA%A6">精密度</a> 。还应使用非目标扫描方法，采用适当的分析技术，以检测浸出物的非预期降解、来自次级包装的浸出物和/或相互作用产物。非目标扫描方法应包含 AET 的应用（见第 5 节），即设定一个水平，高于该水平的浸出化学实体需进行鉴别、定量，并报告进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E7%90%86%E5%AD%A6">毒理学</a> 评估。</p><p>首选采用对照品（如有）进行定量，因为当用于测定适当的响应因子或标准曲线时，对照品有助于更准确和精密地定量测定目标浸出物，其可能是实际制剂中存在的浸出物；在此情况下，分析准确度和精密度较高。</p><h2>4.5 模拟浸出物研究</h2><p>在某些情形下，即便经过全面调查，可能包括对多种不同样本制备技术的系统研究，结合高灵敏度和高选择性的分析方法、技术及仪器设备等，仍可能在技术层面上无法进行制剂浸出物研究。此类情形可能包括与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%A7%E5%AE%B9%E9%87%8F%E6%B3%A8%E5%B0%84%E5%89%82">大容量注射剂</a> （LVP）相关的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A3%80%E6%B5%8B%E9%99%90%E5%BA%A6">检测限</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%9A%E9%87%8F%E9%99%90%E5%BA%A6">定量限</a> 挑战、复杂制剂处方中固有的分析基质显著干扰，或多种此类因素叠加的情形。此时，如经过充分论证合理，使用模拟研究来支持实际制剂浸出物评价可能是可接受的。例如，可以进行模拟研究来补充浸出物研究，以完成浸出物检测中无法实现的目标。当 AET 方面存在挑战时（即分析方法 LOQ &gt; AET），浸出物研究可使用实际可行的检测方法 LOQ 开展，并通过模拟研究填补 LOQ 与 AET 之间的差距。另外，当全面调查确认进行浸出物研究不可行时，可以使用模拟研究代替浸出物研究。</p><p>应当认识到的重要方面是，无论模拟研究的设计和执行如何完善，其结果可能仅近似于制剂浸出物研究的结果，无法完全复制制剂的真实浸出物谱。例如，模拟研究不能也不会解决浸出物与制剂处方组成之间任何潜在的相互作用问题。</p><p>模拟研究是一种替代研究，用于揭示如果进行浸出物研究，可能会检测到的潜在真实浸出物。因此，应对超过模拟研究确定的制剂特定 AET 的模拟浸出物进行鉴别、定量和安全性评估。鉴于模拟研究的目标在于获得与制剂在货架期内产生的实际浸出物谱高度相似的模拟结果，模拟研究中使用的条件与过程应与浸出物研究采用的生产/贮藏条件高度一致，以模拟制剂在生产、有效期储存及（临床）使用制备过程中经历的各项条件。此外，所选用的模拟溶剂应具有与制剂相似的浸出倾向，且模拟生产工艺还应在最差条件下实施。再者，模拟研究可基于制剂有效期放置的条件进行加速研究，从而在较短时间内模拟覆盖整个有效期的浸出物研究结果。</p><p>鉴于模拟研究旨在补充或替代浸出物研究，该研究必须满足浸出物研究的所有质量要求，包括检测方法的确认。经充分论证后，可使用模拟研究作为推荐浸出物研究的替代方法。因此，对于特定制剂开展的模拟浸出研究，其应用目的、合理性论证以及确认，应当建立在科学合理的基础上，并通过适宜的测试与试验作为全面调查的支持依据。在考虑使用模拟研究时，可能需要在实施前咨询相关区域监管机构。</p><h2>4.6 可提取物和浸出物相关性</h2><p>生成可提取物谱的主要目的在于表征并辅助组件选择、识别潜在浸出物、针对目标浸出物开发检测方法，以及建立可提取物与浸出物的相关性。浸出物通常是可提取物的一个子集，在规范实施的可提取物研究中，每种浸出物的浓度通常低于相应可提取物的浓度。</p><p>当 E&amp;L 高于 AET 时，建议评估两者之间定性和定量的相关性。当实际制剂的浸出物能与对应的组件或系统可提取物研究中的可提取物进行定性和定量比较时，也许能够建立浸出物和可提取物之间的相关性。对高风险制剂、变更控制及持续质量控制而言，如果建立了浸出物与可提取物的相关性，则在经过充分论证合理后，可采用常规组件可提取物检测代替稳定性研究期间的常规浸出物检测（如适用）。对于未检出或检出水平高于在提取研究条件下预期水平的浸出物，潜在原因可能包括：可提取物研究设计和/或执行欠佳、浸出物降解形成新化合物、浸出物与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">API</a> 和/或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 间的相互作用产物、从包装中迁移的化学品，以及在有效期放置期间因老化（如暴露于紫外线、高温、氧气）导致材料变化产生的新浸出物。尽管 E&amp;L 的相关性对质量风险评估具有价值和指导意义，并可能有助于组件选择和生命周期管理决策，但最终决定患者安全性风险评价和组件可接受性的仍是浸出物谱。</p><p>如果存在产品 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 内发生的、显著改变可提取物/浸出物特征的任何变更，应快速重新评价可提取物谱/浸出物谱及其相关性。如果在制剂稳定性研究中观察到某种特定浸出物的水平明显高于预期值，（该预期值是基于提取研究计算得出的潜在最大水平，且提取研究所用的组件/系统批次与制剂稳定性批次所用批次相同），则表明提取研究不完整，可能无法就该浸出物建立有效的浸出物-可提取物相关性。</p><h2>5. 分析评价阈值</h2><p>AET 并非一个控制阈值，而是对应于一种浓度阈值，当可提取物或浸出物的浓度高于 AET 时，应对其进行鉴别、定量，并报告进行安全性评估，其构成了可提取物和浸出物风险评估及控制策略的基础。ICH 发布的 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">新原料药中的杂质（ICH Q3A）</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3B(R2)_Impurities_in_New_Drug_Products.">新药物制剂中的杂质（ICH Q3B）</a> 指导原则描述了一系列基于最大日剂量的预定阈值，旨在通过对关键质量属性的充分控制，确保制剂在有效期内的安全性与有效性。与上述指导原则不同，本指导原则建议采用安全性关注阈值（SCT；见第 6 节安全性评估）来首先确定相关研究特定的 AET。</p><p>可提取物研究中应包含 AET 的建立和应用，以便指明需进行检测、鉴别和报告的、可作为制剂潜在浸出物的可提取化学实体。浸出物研究中，高于 AET 的化合物应进行鉴别和定量，以进行适当的安全性评估。对于 1 类浸出物（见附录 4 表 A.4.1），在进行定量测定时应使用化合物特定的安全性限度，而非产品特定的 SCT。</p><p>E&amp;L 研究特定的 AET 取决于剂量相关的考量来进行确定（例如，最大剂量水平、给药频率和治疗持续时间）。根据研究类型（可提取物与浸出物）及评估内容，AET 可用不同计量单位表示。例如，溶液中可提取物的常用计量单位包括：每克组件材料的可提取物重量（如 μg/g），或每毫升提取液体积的可提取物重量（如 μg/mL）。在浸出物研究中，可用每个包装或给药组件/系统的浸出物重量（例如，μg/组件、μg/mL、μg/g、ppm）表示基于整个包装系统或整套生产组件的浸出物 AET。无论 AET 的计量单位如何设定，均等同于某特定研究中的潜在患者等效剂量。AET 计算示例见附录 3。</p><h2>5.1 分析不确定因子</h2><p>当在半定量分析方法中使用 AET 时，应采用适当的不确定因子，以解决因待测物与参比对照品的响应系数差异而可能导致的待测物浓度低估问题。</p><p>在特定可提取物/浸出物研究中，适宜的分析不确定因子数值的确定需综合考虑以下因素：对结构材料的先验知识和理解、潜在可提取物/浸出物可能的化学结构、覆盖响应因子范围的参比对照品的可及性和分析方法的局限性。</p><p>在某些情况下，可接受的方法是乘以不大于 0.5 的不确定因子（UF）。或者，可通过统计分析相关参比化合物的适当响应因子数据库来推导出不确定因子。可提取物/浸出物研究报告中应包括所应用 UF 的合理性说明。</p><h2>6. 安全性评估</h2><h2>6.1 一般原则</h2><p>必须进行基于风险的科学评价，以确认制剂中任何潜在浸出物的水平对患者构成的风险可忽略不计。在此基于风险的总体评价中，安全性评估的重点是对药物制剂中超过预定 SCT 的浸出物的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E7%90%86%E5%AD%A6">毒理学</a> 评价。在此情况下，SCT 被视为一个阈值，低于该值时，浸出物的暴露水平非常低，此时其致突变和非致突变毒性问题均可忽略不计。安全性评估的结果可用于判断材料中 1 类浸出物的水平是否可接受，并可根据需要用于设定制剂中浸出物的质量标准。</p><p>由于 SCT 旨在保护免受致突变和非致突变效应影响，故其必须同时考虑致突变性关注和替代毒性终点相关的问题，并考虑暴露方面更具限制性的因素。因此，除暴露量外，其还取决于暴露的途径和持续时间。对于致突变性问题， <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">ICH M7</a> 中描述的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:TTC">毒理学关注阈值</a> （TTC）被视为适用。对于非致突变毒性终点，本指导原则中采用了 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%95%8C%E5%AE%9A%E9%99%90">界定限度</a> （Qualification Threshold, QT），可将其视为潜在的非致突变毒性作用可以忽略不计的剂量。考虑到 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 和潜在暴露持续时间，将 SCT 设定为特定制剂中 TTC 或 QT 的最低值。通过对约 330 种潜在浸出物 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">每日允许暴露量</a> （PDE）进行回顾，得出口服和注射给药途径的 QT 值。表 1 概述了口服、注射、经皮/透皮和吸入给药暴露途径的全身安全性阈值（μg/天）。此外，还列出了制剂中眼用 <a 浸出物分类</h2><p>各种材料中的潜在浸出物包含多种化学物质，因此具有多种毒理学特性。为了提供一种实用的、基于风险的浸出物安全性评估方法，考虑到某些化合物具有潜在的高毒性，需将其控制在低于既定界定限度的水平。在现行指导原则中，此类化学品被归类为 1 类浸出物。对于致突变致癌物，根据 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">ICH M7</a> 定义的关注队列和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E6%8E%A5%E5%8F%97%E6%91%84%E5%85%A5%E9%87%8F">可接受摄入量</a> （AI）低于 1.5 µg/天的 1 类杂质应被视为 1 类浸出物。同样，某些化合物，如双酚 A（BPA）或苯并(a)芘，可能具有显著的非致突变毒性风险。理论上，当制剂中这些物质的含量达到或低于 QT 限度值时，其带来的患者安全风险仍不可忽视。对于 1 类浸出物，最实用的做法是避免使用可能浸出此类化合物的材料（见第 5 节）。然而，若此类材料或组件的使用确属不可避免，则应采用针对这些物质的特定化合物安全性限度。</p><p>3 类浸出物是指全身毒性效力相对较低的化合物，其衍生的慢性注射 PDE 超过典型浸出物水平（即应用附录 5 所述方法时， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> ≥ 1 mg/天）。若每日暴露水平低于 1mg/天时观察到 3 类浸出物，则无需进行进一步安全性确认。介于这两类之间的是具有潜在毒性的化合物，其毒性可能在浸出物的常见水平下显现（即 2 类浸出物）。附录 4 概述了这三类浸出物。</p><h2>6.3 安全性评估流程</h2><p>对超过 AET 的有机浸出物，应进行鉴别、定量和报告，以供安全性风险评估。需从分析角度证明对化合物结构进行部分或不完全解析的可接受性。如果 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E7%90%86%E5%AD%A6">毒理学</a> 上合理，在某些情况下，提供暂定结构的部分解析可以为安全性评估提供依据。浸出物安全性评估的一般流程见流程图（图 3），包括对致突变性和一般毒性问题的评估。</p><p>*如 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">ICH M7</a> 所述。</p><p>**如果浸出物的每日暴露量&gt;1 mg/天，则应考虑进行 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3B(R2)_Impurities_in_New_Drug_Products.">ICH Q3B</a> 建议的遗传毒性研究（例如，细菌致突变性试验和 <i>体外</i> 染色体畸变试验）。</p><p>在材料和组件选择过程中，应设法识别并避免潜在的 1 类浸出物。若无法避免，则应首先采用较低的化合物特定阈值和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> ，以充分控制其作为浸出物的含量。随后，应根据 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">ICH M7</a> 指导原则评估制剂中高于适用 TTC 的所有浸出物是否具有致突变潜力。被认为具有潜在致突变性的浸出物应控制在 TTC 限度内；若已通过充分的致突变性研究排除了相关风险，则不受此限。</p><p>除致突变性评估外，还应对制剂中高于适用 QT 的所有浸出物进行一般毒性评估。如果有充分数据支持浸出物在患者最大潜在暴露下的安全性，则无需进行进一步毒理学评估（详细信息见附录 5）。相反，如果数据不足以支持浸出物的安全性，则需要采取措施将潜在暴露降至已知可接受水平（材料更换等），或补充毒理学数据以佐证当前暴露水平的安全性，或进行风险/获益评估论证当前暴露水平的合理性。</p><p>需要注意的是，对于缺乏充分数据证明化合物安全性的浸出物，鼓励采用基于高度相似化合物（具有毒理学数据）的交叉参照法。若能识别出具有充分数据支持在观测水平下浸出物安全性的替代物，则无需进一步的安全性风险评估和/或研究。</p><p>如果认为需要生成新的毒理学数据，以支持浸出物暴露的安全性评估，在论证充分的前提下，可考虑采用新方法（NAM），包括计算机模拟模型和体外模型。否则，应考虑进行 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3B(R2)_Impurities_in_New_Drug_Products.">Q3B</a> 中描述的毒理学确认研究，以支持化合物的安全性评估。</p><h2>6.4 给药途径相关注意事项和特殊情况（局部毒性问题）</h2><p>潜在全身毒性的安全性风险评估通常足以支持浸出物暴露的安全性。然而，在某些情形中，由于化合物局部浓度可能导致脆弱组织损伤（例如肺部制剂、眼用制剂及脑内/鞘内/硬膜外制剂），潜在的局部毒性效应可能具有相关性。在相关情况下，毒理学风险评估应评估浸出物对局部组织毒性的潜在影响，以及可能降低减少此类影响的因素（如处方、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 、接触时间和组织损伤恢复）。此外，当考虑潜在的局部毒性时，所使用的 SCT 应为致突变性阈值（即 TTC）、非致突变性阈值（即 QT）和局部毒性阈值（相关浓度转化为最高日暴露水平）中的最低者（按每日暴露量计算）。</p><h3>6.4.1 眼用制剂</h3><p>眼用产品通常为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B1%80%E9%83%A8%E7%94%A8%E8%8D%AF">局部给药</a> ，而其中部分产品通过直接注射至眼部组织给药。目前缺乏数据来描述浸出物与眼部组织接触时的潜在局部毒性。基于历史先例，在缺乏相关数据库的情况下，对于最终待上市的眼用局部给药制剂中浓度超过 20 ppm 的浸出物，需完成化合物特定风险评估以论证其安全性。该浓度限度不适用于与眼部组织短暂接触的冲洗液。注射到眼部组织的产品未设定阈值，应对所有存在的浸出物进行定性安全性评估，即使其浓度低于 20ppm，该评估也可能具有相关性。</p><h3>6.4.2 脑内、鞘内、硬膜外制剂</h3><p>脑内、鞘内和硬膜外制剂可能直接与损伤后修复能力有限的重要中枢神经系统（CNS）组织相互作用，但目前缺乏充分数据来表征直接作用于神经组织或邻近区域的化合物的潜在毒性。 <i>体外</i> 数据表明，对于某些已知具有神经毒性的化合物，化学诱导的生物效应可能在十亿分之一（ppb）的低浓度范围内发生。因此，化合物特定风险评估应考虑观察到的浸出物的局部浓度以及对神经组织（例如，神经元、星形胶质细胞、神经胶质细胞、髓鞘）的潜在局部毒性问题，包括对局部炎症反应可能性的评估。</p><h3>6.4.3 经皮给药制剂</h3><p>对于局部毒性效应，当浸出物为强效或极强效皮肤致敏物时，致敏潜力（见第 6.4.4 节）可能是最敏感的非遗传毒性终点。对于高效价化学品（HPC），皮肤致敏阈值（DST）推导为 1 μg/cm <sup>2</sup> /天。经 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3D_R2_Elemental_Impurities_20220426">ICH Q3D</a> 中描述的经皮和透皮浓度限值（CTCL）计算进行转换后，该阈值对应于经皮给药制剂中 500 ppm 的浓度。因此，对于经皮给药制剂，可以使用与制剂中 500 ppm 浓度相对应的局部毒性阈值，低于该阈值时无需进行包括致敏潜力在内的局部非致突变毒性评估（见表 1）。</p><h3>6.4.4 致敏潜力</h3><p>致敏剂是反复暴露后可能引发超敏反应的化合物。对这些化合物的关注程度主要取决于三方面：该化合物的致敏潜力、暴露途径和暴露个体的易感性。已针对各种暴露途径描述了具有多种作用模式的不同类型的超敏反应，但目前仅经皮途径存在经验证的预测模型。本指导原则涉及诱发致敏潜力的风险，并提供了该风险的局部毒性阈值（如适用）。如果患者对某种化合物敏感，则可能在较低的阈值下引发致敏反应。</p><p><u>经皮暴露</u></p><p>大多数关于致敏潜力的数据来自于经皮 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 。除人体数据外，还开发了计算机模拟、化学分析、体外和体内模型来表征化合物的皮肤致敏潜力。DST 是根据致敏效力得出的。 <sup><b>1,2</b></sup></p><p>如果经皮给予已知浸出物时，其浓度低于相应效力类别的 DST，则可以得出结论，预计不会产生皮肤致敏性，不需要采取进一步措施。如果超过 DST，则应评估关于致敏潜力的可用化合物特定数据。如果没有此类数据，或者当这些数据引起关注时，需要考虑风险缓解措施。措施可能包括更换浸出化合物的组件或降低浸出物水平。</p><p>由于透皮药物也经皮肤给药，因此可以使用相同的方法来评估其致敏潜力风险。对于多日贴剂，假设所有浸出物在一天内迁移，应使用数据证明较慢的迁移速率是合理的。</p><p><u>吸入暴露</u></p><p>目前关于化合物呼吸道致敏潜力的认知主要来自于人体数据。目前，尚未建立起适用于安全性风险评估的呼吸道致敏非临床模型。皮肤和呼吸道致敏剂的作用方式有共同点，但也存在差异，特别是在 T 细胞激活后。因此，皮肤致敏数据不应用于估计呼吸道致敏的风险，也不能提供呼吸道致敏的阈值。</p><p>呼吸道对具有致敏（和刺激）特性的化合物非常敏感 <sup><b>3</b></sup> 。因此，应评估任何可能具有致敏潜力或刺激性的结构元素的化合物（如异氰酸酯、腈、苯乙烯、短链醛）。如果认为一种化合物具有刺激性或致敏潜力，则应在评估该化合物的可用信息后，根据具体情况对患者风险进行评估。此外，应评估现有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E6%95%B0%E6%8D%AE">临床数据</a> ，以确定 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%8D%E8%89%AF%E5%8F%8D%E5%BA%94">不良反应</a> 的证据。如果未发现刺激性或致敏性问题，则可以采用注射给药对应的全身毒性 QT 值，如表 1 所示。</p><p><u>注射暴露</u></p><p>关于致敏的潜在风险，应区分皮下/皮内给药途径与静脉/肌肉/腹膜内给药途径。在皮下给药途径中，药物在触发皮肤致敏的关键组织和细胞（即朗格汉斯细胞）附近进行给药。特别是当浸出物未能快速分布，且在皮下组织中滞留时间较长时，可能激活相同的作用机制。因此，在评估皮下给药浸出物的致敏潜力时，可参考皮肤致敏潜力的现有数据。同样，对于皮内给药制剂，皮肤致敏数据可能具有相关性。相比之下，应用于皮肤表面的化合物需要首先穿透皮肤屏障。考虑到这一差异，将皮下和皮内给药制剂的阈值设为比经皮给药制剂低 10 倍是合理的，即 50 ppm，而非 500 ppm。</p><p>在目前已知的全身性超敏反应（I-IV 型）中，每种类型都具有不同的作用模式。IV 型依赖于半抗原的形成，因此与皮肤致敏具有一些共同的机制。然而，与经皮给药相反，肌内和静脉给药的物质会在全身迅速分布，因此需要大量药物才能激活免疫系统并诱发致敏。由于浸出物在制剂中的浓度较低，因此一般认为通过静脉或肌内注射给药的药物不太可能引起致敏性问题。</p><h2>6.5 ICH S9 产品的注意事项</h2><p>对于 <a href="https://lib.shilinx.com/wiki/index.php?title=S9_Nonclinical_Evaluation_for_Anticancer_Pharmaceuticals">ICH S9</a> 范围内的制剂，通常应根据上文第 3 节中概述的科学原则进行浸出物识别。可根据 ICH S9 中的“杂质评价”章节进行安全性风险评估。在此类情况下，TTC 将不再适用，SCT 将由 QT 定义。风险评估可重点关注目标患者人群的一般安全性，并参考 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_S9_Questions_and_Answers_Nonclinical_Evaluation_for_Anticancer_Pharmaceuticals_201804">ICH S9 Q&amp;A 2018</a> ，与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">API</a> 是否具有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9F%BA%E5%9B%A0%E6%AF%92%E6%80%A7">遗传毒性</a> 进行关联性考虑。</p><h2>6.6 安全性评估内容</h2><p>应对观察到的 1 类浸出物、检测水平高于相关 SCT 的 2 类浸出物以及浓度水平高于 1.0 mg/天的 3 类浸出物进行安全性评估。安全性评估应提供充分信息，以便就预期患者暴露水平的可接受性得出结论。附录 5 中详细描述了需要考虑的信息和得出可接受暴露水平的方法。</p><h2>7. 术语表</h2><p><b>分析评价阈值（AET）：</b></p><p>应对超过该阈值的可提取物或浸出物进行鉴别、定量，并报告进行安全性评估。</p><p><b>化学表征：</b></p><p>获取药品 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85">包装</a> 和药品生产组件等物品组成的化学信息的过程。</p><p><b>组件：</b></p><p>由一种或多种结构材料组成的单个物品，用于单一目的或执行单一和特定任务。</p><p><b>提取：</b></p><p>将受试物的成分转移到提取介质中的化学或物理过程。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E8%B4%A8%E9%87%8F%E5%B1%9E%E6%80%A7">关键质量属性</a> ：</b></p><p>一种物理、化学、生物或微生物属性或特性，应在适当的限度、范围或分布范围内以确保获得预期产品质量。</p><p><b>制剂：</b></p><p>在最终药品包装中用于上市的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%89%82%E5%9E%8B">剂型</a> 。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> ：</b></p><p>尚未进行配方的活性药品成分，随后可与辅料一起配方以生产剂型（或制剂）。</p><p><b>可提取物谱：</b></p><p>对提取研究中存在的可提取物的定性或半定量/定量说明。</p><p><b>浸出物谱：</b></p><p>对制剂中浸出物的定性和/或定量说明。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> ：</b></p><p>从最初的开发到上市，直到产品退市的产品生命中的所有阶段。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:NOAEL">未见（不良）反应剂量</a> （NO(A)EL）：</b></p><p>与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AF%B9%E7%85%A7%E7%BB%84%EF%BC%88%E4%B8%B4%E5%BA%8A%EF%BC%89">对照组</a> 相比，在暴露人群中不会引起任何统计学或生物学上显著的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%8D%E8%89%AF%E5%8F%8D%E5%BA%94">不良反应</a> 的浸出物或可提取物的最高浓度或数量。</p><p><b>交叉参照：</b></p><p>一种通过使用来自其他结构相关物质的相同终点数据来预测一种物质的终点信息的技术。</p><p><b>安全边际：</b></p><p>特定浸出物的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> 与基于日剂量的实际患者摄入量之间的相关性。</p><p><b>结构材料：</b></p><p>用于构建包装或生产组件或系统的单个材料。</p><p><b>新药制剂：</b></p><p>药物制剂类型，例如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%89%87%E5%89%82">片剂</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%83%B6%E5%9B%8A">胶囊</a> 、溶液、乳膏等，其尚未在某一区域或成员国注册，通常含有药物成分，并且可能含有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:NOAEL">未观察到（有害）作用水平</a> （NO(A)EL）：</b></p><p>与对照组相比，在暴露人群中不会引起任何统计学或生物学上显著的不良反应的浸出物或可提取物的最高浓度或数量。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">每日允许暴露量</a> （PDE）：</b></p><p>对于慢性暴露，药品中浸出物每天的最大 <a 计算的起点；可从人用剂量或适当的动物研究中得出。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%95%8C%E5%AE%9A%E9%99%90">界定限度</a> （QT）：</b></p><p>当浸出物含量高于该阈值时，将被考虑用于非致突变毒性作用的安全性评估，除非其被认定为高度关注的浸出物。</p><p><b>安全性关注阈值（SCT）：</b></p><p>当浸出物含量等于或低于该阈值时，其致突变和非致突变毒性引发的安全性问题可忽略不计，除非该浸出物被认定为高度关注的浸出物。</p><p><b>模拟制剂：</b></p><p>对浸出倾向和浸出物 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:TTC">毒理学关注阈值</a> （TTC）：</b></p><p>如 <a 所述，当浸出物含量低于该阈值时，不考虑对其进行致突变效应安全性评估。</p><h2>8. 参考文献</h2><p>国际人用药品注册技术协调会（2006）。 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">Q3A（R2）：新原料药中的杂质</a> SCT（μg/天）。在某些情况下，替代起始点可能相关（例如对于潜在的 1 类浸出物）。在所有计算过程中均应假设最差条件，例如制剂的最大批准剂量。提供了可提取物和浸出物研究的常见示例。AET 的计算应清晰标明单位及计算方法。无论用于表示 AET 的单位如何，给定研究的最终值须换算至与患者 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9A%B4%E9%9C%B2%E9%87%8F">暴露量</a> 相同的水平（即 SCT 乘以分析不确定因子[UF]）。</p><p><b>最大日剂量（MDD）和安全性关注阈值（SCT）</b></p><p>对于每种产品，AET 的计算应基于 MDD。MDD 是药物在一天内给药的最大批准剂量。为确定 SCT，应考虑 TTC 与 QT，如表 1 所示。SCT 基于其中的最低值确定。</p><p><b>间歇给药</b></p><p>如果不是每天给药，则应遵循 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">ICH M7</a> 来推导适用的 TTC（例如，当总给药天数≤30 天时，TTC = 120 μg）。</p><p>在 QT 推导中，当总给药天数≤30 天或给药频率为每月一次或更少时，可使用≤1 个月的QT。</p><p><b>多日产品</b></p><p>对于使用后可能在患处保持多天的产品（例如，多日贴剂、长效 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B3%A8%E5%B0%84%E5%89%82">注射剂</a> 、植入物），适用的 TTC 由治疗的总持续时间决定。对于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%87%B4%E7%AA%81%E5%8F%98%E6%9D%82%E8%B4%A8">致突变杂质</a> ，根据 ICH M7，应使用平均每日暴露量计算。对于非致突变浸出物，默认假设所有浸出物在一天内迁移。此时，适用的QT 由总给药次数定义。迁移速率较慢会减少非致突变浸出物的日剂量，但会增加给药天数。如采用较慢的迁移速率，则应使用数据证明其是合理的。</p><p><b>AET 计算示例</b></p><p><b>可提取物情形 1：液体制剂生产工艺中使用的过滤器</b></p><p>(1)AET（μg/过滤器）= SCT（μg/天） × <i>UF</i> × 每批制剂的剂量单位数* ÷ 每批过滤器数</p><p>(2)AET（μg/g 过滤器）= AET（μg/过滤器）÷ 重量（g）/过滤器</p><p>(3)AET（μg/mL 提取溶剂）=AET（μg/过滤器）÷ 提取溶剂（mL）/过滤器</p><p>(4)AET（μg/cm <sup>2</sup> ）=AET（μg/过滤器）÷ 接触表面积（cm <sup>2</sup> ）/过滤器</p><p>*应使用单日给药的MDD 和最小潜在批量来确定每批制剂的剂量单位数（即最差条件）。因此，如果单日内给予的最大批准剂量为 100 mg（= 0.1 g），最小潜在批量为 1 kg（=1000 g），则每批制剂的剂量单位数为 1000 g/批 ÷ 0.1 g/剂 = 10000 剂/批。</p><p><b>可提取物情形 2：液体制剂包装系统（CCS）中使用的橡胶瓶塞</b></p><p>(1)AET（μg/ <a 胶塞重量（g）</p><p>(3)AET（μg/mL 提取溶剂） = AET（μg/胶塞） ÷ 提取溶剂（mL）/胶塞</p><p>(4)AET（μg/mL 提取溶剂） = AET（μg/g 胶塞） ÷ 提取溶剂（mL）/胶塞克数</p><p>*应使用单日内给予的最大批准体积剂量（即最差条件）。如果剂量以质量（例如，mg/ 天）为基础进行描述，则应根据 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">活性成分</a> 的浓度将其转换为体积（mL）。因此，如果单日内给予的最大批准剂量为 100 mg（= 0.1 g），制剂浓度为 10 mg/mL，则计算的日最大剂量为 100 mg ÷ 10 mg/mL = 10 mL。</p><p><b>浸出物情形 1：液体制剂生产设备的浸出物</b></p><p>(1)AET（μg/批）= SCT（μg/天）× <i>UF</i> ×每批制剂的剂量单位数*</p><p>(2)AET（μg/mL 制剂）= SCT（μg/天）× <i>UF</i> ÷ 日最大剂量（mL）</p><p>*应使用单日给药的MDD 和最小潜在批量来确定每批制剂的剂量单位数（即最差条件）。因此，如果单日内给予的最大批准剂量为 5 mL，最小潜在批量为 10 L（= 10000 mL），则每批制剂的剂量单位数为 10000 mL/批 ÷ 5 mL/剂 = 2000 剂/批。</p><p><b>浸出物情形 2： <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%A2%84%E7%81%8C%E5%B0%81%E6%B3%A8%E5%B0%84%E5%99%A8">预灌封注射器</a> （PFS）的浸出物</b></p><p>(1)AET（μg/mL 制剂）= SCT（μg/天）× <i>UF</i> ÷ 日最大剂量（mL）*</p><p>(2)AET（μg/PFS）=AET（μg/mL 制剂）×每支 PFS 的体积（mL）</p><p>*应使用单日内给予的最大批准体积剂量（即最差条件）。如果剂量以质量（例如，mg/ 天）为基础进行描述，则应根据活性成分的浓度将其转换为体积（mL）。因此，如果单日内给予的最大批准剂量为 10 mg，制剂浓度为 10 mg/mL，则计算的日最大剂量为 10 mg ÷ 10 mg/mL = 1 mL。</p><h2>附录 4：浸出物的效力分类</h2><p>潜在浸出化合物的化学性质各不相同，其安全性数据库也是如此。为了在保护患者的同时能有效设定安全性阈值，除指导原则中应用的阈值外，还开发了浸出物分类办法。分类办法基于全身效应，广泛适用于所有 <a 类浸出物，需要进行进一步的安全性评估，以确定当患者接受特定制剂给药时与这些浸出物相关的潜在暴露风险。对于绝大多数潜在浸出物，目前尚未确定用于支持任何制剂中化合物安全暴露的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">每日允许暴露量（PDE）</a> 。此外，由于现有制剂的性质不同，以及可提取物和浸出物安全性风险评估的复杂性，通用方法（如既定的 PDE）并不总是最合适的。虽然本指导原则的重点不是设定单个化合物的可接受暴露水平，但通常可能需要在逐个产品的基础上确定特定化合物的限度。因此，本附录通过基于风险的方法，为在各种制剂类型和给药场景下适当地评估浸出物安全性提供了指导。</p><p>在判定浸出物潜在患者暴露水平可接受性的过程中，不同制剂和场景间对信息充分程度的要求可能变化甚大，且可能使用多种方法来确定其可接受性。最直接的方法是采用已建立的安全暴露水平，这些暴露水平已保守地假设了最差条件。因此，当现有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:ICH">ICH</a> 指导原则（例如 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3C_R9_Impurities_Guideline_for_Residual_Solvents_20240124">Q3C</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_M7_R2_Assessment_and_Control_of_DNA_Reactive_Mutagenic_Impurities_in_Pharmaceuticals_to_Limit_Potential_Carcinogenic_Risk_20230403">M7</a> ）中有既定的 PDE 时，在满足所有必要考量因素的情况下，引用该值即可。而另一种可能更为适用的方法是，使用类似方法和科学原则（如之前在指导原则中确定的）得出的可接受暴露水平。在其他情形下，定义明确、支持充分且论证合理的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:NOAEL">NOAEL</a> 与预期患者 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9A%B4%E9%9C%B2%E9%87%8F">暴露量</a> 之间的剂量比可能极大（例如&gt;10000），因此无需详细推导。</p><p>尽管在某些情况下，体外和/或体内研究（作为最后手段）可能被认为是确定可接受暴露水平所必需的，但鼓励通过可用计算机模拟分析，并对类似化合物（即替代化合物）进行交叉参照来提供科学论证（如适用），以确定可接受暴露水平。</p><p>尽管有多种计算机模拟毒理学工具可用，但在本指南的框架下，致突变性是唯一的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E7%90%86%E5%AD%A6">毒理学</a> 终点。恰当开展的致突变性试验目前已被确认可独立用于替代生物学数据（参见 ICHM7）。然而，在有适当的科学依据的情况下，应将使用计算机模拟、体外、或体内模型对其他毒理学终点的预测纳入安全性风险评估中，以基于证据权重的风险评估方法补充现有数据。在每个类别中，应优先使用那些考虑到相关暴露途径的已验证模型的数据。</p><p>由于大量潜在浸出物的毒理学数据集有限甚至缺乏，也可考虑采用交叉参照法。在交叉参照法中，替代化合物（或多种替代物）的相关毒理学数据将作为证据权重法的一部分， 或在无数据时作为目标浸出物数据的替代，用于支持目标浸出物的安全性评估。若安全性评估涉及替代化合物，应提供明确的替代物选择理由。在选择合适的替代物时，应考虑各种属性（如已知），包括作用方式、主要毒性基团和周围的化学环境（例如，存在可能影响 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%AD%A6%E6%B4%BB%E6%80%A7">生物活性</a> 的官能团）、总体结构相似性、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E4%BB%A3%E5%8A%A8%E5%8A%9B%E5%AD%A6">毒代动力学</a> 性质、理化性质（例如， 极性、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BA%B6%E8%A7%A3%E5%BA%A6">溶解度</a> 、电离性以及分子量）。在有充分理由的情况下，可使用新方法学（NAMs） 的计算机模拟工具和数据支持替代物的选择，并为交叉参照方法提供信息，但需要考虑上述标准。对于替代物如何纳入相关浸出物的安全性评估应进行科学论证。还应指出并适当说明与交叉参照方法相关的潜在不确定性，例如在用于确定可接受暴露水平时（见下文 F7）。</p><p><b>待评价并纳入安全性评估的数据</b></p><p>为了确立特定制剂中浸出物的安全性，需提供对化合物的全面安全性评估。下面列出了要包括的数据元素（如数据可用）。还应评估这些数据集的相关性和质量。如上所述， 替代化合物数据与计算机模拟分析的使用也应纳入安全性评估并证明其合理性。此外， 如将观察到的多个浸出物分组评价，则须包含该分组的详情与依据。</p><p><b><a 类不适用于浸出物）。</li></ul><ul><li>如适用，应提供遗传毒性和/或致癌性的机制，因为这与可接受暴露水平的确定高度相关。</li></ul><p>附加信息</p><ul><li>还应包括安全性评估的其他相关信息（如有）。</li></ul><ul><li>示例：现有基于健康的风险限度/评估、临床和流行病学数据、类似/相关化合物的毒理学数据</li></ul><p><b>可接受暴露水平的计算</b></p><p>除 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E6%8E%A5%E5%8F%97%E6%91%84%E5%85%A5%E9%87%8F">可接受摄入量（AI）</a> 等其他基于健康的限度外，PDE 概念在 ICH 指导原则中已被作为基于健康的暴露限度予以落实，其计算过程在各指导原则间基本一致。这种相同的基本方法已用于生成 PDE 值，以支持当前指导原则中确定的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%95%8C%E5%AE%9A%E9%99%90">界定限度</a> （包括生物利用度和使用交叉参照法时的额外校正因子）。下文简要描述并总结了该方法，其可用作特定制剂中浸出物可接受暴露水平的基础。</p><p>虽然此处描述的推导可接受暴露水平的方法基于其他 ICH 指导原则中的 PDE 方法，但需指出，可接受暴露水平不一定等同于 PDE。根据定义，PDE 为终生暴露水平，适用于多种产品，而特定产品的可接受暴露水平则需考虑暴露持续时间和最大日剂量。在对上述浸出物的可用数据与信息进行回顾和评价后，开始推导过程：首先选择合适的起始点（PoD），随后应用校正因子（F1-F7）。应使用最相关的研究来选择 PoD，同时考虑使用的物种、暴露途径、持续时间、监测的毒理学终点以及研究数据的质量；如有充分理由， 可不选择最低 NO(A)EL 作为 PoD。以往的指导原则使用了特定的校正因子，用于描述种间和种内变异性（分别为 F1 和 F2）、采用 PoD 的研究持续时间（F3）、毒性严重程度（F4）以及说明 NOAEL 缺失的因子（F5）。由于浸出物涵盖广泛的化学空间，不同给药途径的生物利用度可能有所差异。由于毒性数据通常仅适用于单一给药途径，故本指导原则建议纳入一个额外的校正因子（F6），以考虑在进行给药途径间外推时生物利用度的差异。此外，如前所述，有时可能需要使用替代化合物的 PoD（交叉参照法）。因此，建议使用另一个校正因子（F7）来解释与使用该替代化合物相关的不确定性。</p><p>由于 F1-F5 值的选择标准已在现有指导原则中详细说明，故在此不再赘述。新引入的与浸出物相关的校正因子（F6 和 F7）总结如下。</p><p><b>F6 = 用于考虑暴露途径外推的可变因子</b> （例如，口服给药至注射给药）。</p><p>在缺乏足够的通过制剂预期暴露途径获得的浸出物毒性数据的情况下，应使用 F6 来调整 PoD 研究给药途径与制剂暴露途径之间生物利用度的相关差异。理论上，F6 的取值应基于母体化合物的生物利用度数据。若采用放射性标记研究，应将其表述为吸收作用， 因无法明确放射性标记物是母体化合物、代谢物，或是两者的混合物。如果数据质量良好，相对生物利用度估计值可直接用于确定 F6 的值。当生物利用度估计值存在显著不确定性时，可替代应用默认因子。例如，当使用口服毒性数据推导注射给药的可接受暴露水平时：</p><p>口服生物利用度&lt;1%时，F6 = 100（除以校正因子 100）</p><p>口服生物利用度≥1%且&lt;50%时，F6 = 10（除以校正因子 10）</p><p>口服生物利用度≥50%且&lt;90%时，F6 = 2（除以校正因子 2）</p><p>口服生物利用度≥90%时，F6 = 1（除以校正因子 1）</p><p>在缺乏足够体内数据的情况下，应采用其他方法作为证据权重策略的一部分或代替体内 数据。例如，如果有适当的支持和科学依据，可以使用 NAM（结合估计吸收和内部 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B8%85%E9%99%A4%E7%8E%87(%E8%8D%AF%E4%BB%A3%E5%8A%A8%E5%8A%9B%E5%AD%A6)">清除率</a> 的体外数据以及计算机模拟 PBPK 模型）来生成数据以评估生物利用度。或者，建议 F6 的默认校正因子为 100。如需采用不足 100 的校正因子，需提供理由（例如，基于化合物的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%90%86%E5%8C%96%E7%89%B9%E6%80%A7">理化特性</a> 进行推理）。当可获得替代分子药物的适当生物利用度数据并允许采用交叉参照方法时，若有充分依据，可利用这些数据为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%A9%E7%94%A8%E5%BA%A6">生物利用度</a> 估计提供信息。</p><p>对于某些给药途径，如吸入给药，在确定合适的 F6 值时，需要额外考量。例如，对于吸入毒理学研究，呼吸道沉积、呼吸吸收率和肺部代谢的数据为 F6 的确定提供参考。对于经皮 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> ，若毒代动力学数据可用，则可用于估计全身剂量。在评估浸出物的估计每日总全身剂量时，可参考注射给药 QT。在缺乏 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AF%92%E4%BB%A3%E5%8A%A8%E5%8A%9B%E5%AD%A6">毒代动力学</a> 数据的情况下，当从皮肤剂量外推至全身剂量时，应采用较为保守的吸收率设定，即对多数有机 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BA%B6%E5%89%82">溶剂</a> 稀释液默认采用 70%吸收率，对水基或分散类稀释液默认采用 50%吸收率。如果分子量大于500 且 logPow
