三方协议指导原则</center></b><p>2002 年 2 月 7 日在ICH 指导委员会上进入ICH 进程第四阶段。推荐给ICH三方管理部门采纳。附件 2 于 2006 年 10 月 25 日进行过修订⁠⁠⁠⁠⁠⁠。</p><h2>1. 前言</h2><p>本文件旨在为那些尚未在任何地区或成员国注册的化学合成的新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 在注册申报时，对杂质研究的内容和限度确定提供指导。本指导原则不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 期间所用的新原料药。本文件不涵盖生物/ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%B6%E5%93%81">生物制品</a> 、肽、寡聚核苷酸、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%94%BE%E5%B0%84%E6%80%A7%E8%8D%AF%E5%93%81">放射性药物</a> 、发酵和半合成产品、草药以及来源于动、植物的粗制品。</p><p>新原料药中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> 应分两个方面阐述：</p><p>*化学方面：包括对杂质的分类和鉴定、检测报告的形成、在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 中列出杂质以及对 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 的简要讨论。</p><p>*安全性方面：对用于安全性研究和临床研究的新原料药批次中不存在或含量很低的那些杂质的界定提供专门的指导。</p><h2>2. 杂质的分类</h2><p>杂质可分为下列类型：</p><p>*有机杂质（与工艺和药物结构有关的）。</p><p>*无机杂质</p><p>* <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AE%8B%E7%95%99%E6%BA%B6%E5%89%82">残留溶剂</a></p><p>有机杂质可能会在新原料药的生产过程和/或储存期间产生。这些杂质可能是结构已鉴定的或者是未鉴定的、挥发性的或者非挥发性的。包括：</p><p>* <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%B7%E5%A7%8B%E7%89%A9%E6%96%99">起始物料</a></p><p>*副产物</p><p>*中间体</p><p>*降解产物</p><p>*试剂、配体、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%82%AC%E5%8C%96%E5%89%82">催化剂</a></p><p>无机杂质可能来源于生产过程，它们通常是已知的和结构已鉴定的，包括：</p><p>*试剂、配体、催化剂</p><p>*重金属或其他残留金属</p><p>*无机盐</p><p>*其他物质（例如：助滤剂、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B4%BB%E6%80%A7%E7%82%AD">活性炭</a> 等）</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BA%B6%E5%89%82">溶剂</a> 是在新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 合成过程中作为载体用于制备溶液或混悬液的有机或无机液体，由于他们一般具有已知毒性，故较易选择控制方法（见 ICH 指导原则 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3C_R9_Impurities_Guideline_for_Residual_Solvents_20240124">Q3C</a> 残留溶剂）。</p><p>不包括在本文件中的杂质为：⑴外源性污染物：不应该存在于新原料药中，可以用 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">GMP</a> 来控制的；⑵ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%9A%E6%99%B6%E5%9E%8B">多晶型</a> ；⑶ 对映异构体杂质。</p><h2>3. 杂质报告和控制的合理性</h2><h2>3.1 有机杂质</h2><p>申请人应对新原料药在合成、精制和储存过程中那些实际存在的和最可能产生的潜在杂质进行综述。该综述应建立在对合成所涉及的化学反应、由起始物料引入的杂质及可能的降解产物进行合理的、科学的评估基础之上。可以局限于根据化学反应以及相关条件下可能会产生的杂质进行讨论。</p><p>此外，申请人还应对新原料药中杂质检测的实验室研究工作进行综述。其内容包括对采用研制工艺的和拟上市工艺生产的所有批次产品的试验结果，以及为鉴定在储存期间可能产生的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BD%9C%E5%9C%A8%E6%9D%82%E8%B4%A8">潜在杂质</a> 而进行的强制降解试验结果（见 ICH 指导原则 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q1A(R2)_Stability_Testing_of_New_Drug_Substances_and_Products">Q1A</a> 稳定性项下）。同时应对那些拟上市的原料药批次和研制开发过程的原料药批次中的杂质谱进行比较，并就任何不同之处进行讨论。</p><p>申报资料中应对那些在新原料药中实际存在的、含量大于（&gt;）附件 1 中鉴定限度的杂质（例如：以 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的响应因子计算）的结构进行确证。应该注意的是，对于采用拟上市工艺生产的批次，所有出现的大于鉴定限度的杂质的结构都应确证；另外，对于在推荐的储存条件下进行稳定性研究中发现的大于（&gt;）鉴定限度的降解产物，也同样应对其结构进行确证；当某个杂质的结构无法确证时，申报资料中也应提供对该杂质所进行的不成功的研究工作的综述。如果已尝试确证过含量不大于（≤）鉴定限度的杂质的结构，那么在申报资料中提交这些研究结果也是很有用的。</p><p>通常没有必要对表观含量在鉴定限度以下（≤）的杂质进行鉴定。然而，对那些含量不大于（≤）鉴定限度但可能产生不寻常功效或毒性药理作用的潜在杂质，则应当研发出合适的分析方法。所有杂质均应按照本指导原则后续章节中的要求来界定。</p><h2>3.2 无机杂质</h2><p>无机杂质通常按药典或其他适当的方法来检测和定量。在新药的研制过程中应对遗留在新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%82%AC%E5%8C%96%E5%89%82">催化剂</a> 进行评估。对新原料药质量标准中是否收载无机杂质检查项目，应进行讨论。其限度应根据药典标准或已知的安全性数据来制定。</p><h2>3.3 溶剂</h2><p>应按 ICH <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3C_R9_Impurities_Guideline_for_Residual_Solvents_20240124">Q3C</a> “残留溶剂”指导原则的要求，对新原料药生产过程中所用溶剂的残留量的控制进行讨论和申报。</p><h2>4. 分析方法</h2><p>注册申请中应提供书面文件，证明分析方法是经过验证并适用于杂质的检测和定量（见 ICH Q2A 及 Q2B 分析方法论证指导原则项下）。技术因素（如生产能力与质控方法）可作为部分依据来论证可以基于拟上市生产工艺获得的生产经验选择其他的。 并不代表常规质量控制中分析方法所需达到的精度。因此，只需经过合适的验证和论证，可以使用较低精度的技术（如薄层 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%89%B2%E8%B0%B1">色谱法</a> ）。如果研发中所采用的分析方法和准备上市产品的分析方法不同，在申报资料中应予以讨论。</p><p>可用各种技术测定有机杂质的含量，这些技术包括把杂质的响应值与适当的对照品的响应值比较或与药物本身的响应值比较。应根据使用目的，对分析过程中用于控制 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> 的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AF%B9%E7%85%A7%E5%93%81">对照品</a> 进行定性和定量。可用原料药作为标准物质来估计杂质的量，如果原料药和杂质的响应因子不接近，只要应用了校正因子或测得的杂质量高于实际的杂质量，该方法仍是可行的。用于估计已鉴定或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%AA%E9%89%B4%E5%AE%9A%E6%9D%82%E8%B4%A8">未鉴定杂质</a> 的限度和分析方法可基于分析方法学上的假设（例如：相同的检测响应等）。但这些假设的合理性应在申报资料中加以讨论。</p><h2>5. 各批次产品杂质报告的内容</h2><p>注册申请应提供用于临床、安全性研究、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 试验的所有新原料药批次产品以及采用拟上市工艺生产的代表性批次产品的分析结果。测定结果应数字化，不应用“符合规定”，“符合限度”等一般性术语。在新原料药的所有批次中，应报告检测到的大于（&gt;）报告限度（见附件 1）的任何杂质和总杂质的含量，并附所用的分析方法。若 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> 含量低于 1.0%，结果应报告至小数点后两位（如 0.06%，0.13%），若大于或等于 1.0%，结果报告至小数点后一位（如 1.3%）。结果应按传统规则修约（见附件 2）。建议使用数据表格（如电子数据表），各杂质均应以编号或适当的描述表示（如：保留时间）。如果采用其它较高的报告限度，应充分论证其合理性。。</p><p>若在研制期间， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 发生了变化，报告的测试结果应标明所用的分析方法，并提供相应的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%96%B9%E6%B3%95%E9%AA%8C%E8%AF%81">方法学验证</a> 资料。应提供有代表性的色谱图。方法学验证中，显示杂质分离度和检测灵敏度的、具有代表性批次（例如：加样试验）的色谱图和常规杂质检测得到的色谱图，可以反映出有代表性的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8%E8%B0%B1">杂质谱</a> ，应在申报资料中予以提供。同时申请人应保证：如需要，可提供每个批次产品的完整的杂质谱（例如：色谱图）。</p><p>另外，申请人还应提供相应的表格，说明在每个安全性研究和临床研究中使用的新原料药的批次。</p><p>对每批新原料药，报告内容应包括：</p><p>*批号与批量</p><p>*生产日期</p><p>*生产地点</p><p>*生产工艺</p><p>*单个杂质和总杂质的含量</p><p>*批次的用途</p><p>*所涉及的分析方法的说明</p><h2>6. 质量标准中的杂质检查项目</h2><p>在新原料药的质量标准中应包括杂质检查项目。稳定性研究、化学方面的开发研究以及日常批次分析检验的结果有助于预测在上市产品中可能出现的杂质。在新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 中收载的杂质应根据在采用拟上市工艺生产的批次中所发现的杂质来加以取舍。在本指导原则中，对列入新原料药质量标准中、具有特定限度要求的各个杂质称为特定杂质。特定杂质可以是已鉴定杂质，也可以是未鉴定杂质。</p><p>申报资料应对质量标准中纳入哪些杂质提供合理的依据。该依据应包括对用于安全性和临床研究中批次中所发现的杂质谱进行讨论，同时也要考虑采用拟上市工艺生产的原料药中的杂质谱情况。对于那些具有特殊功效或产生毒性或未预料到的药理作用的杂质，其分析方法的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%9A%E9%87%8F%E9%99%90%E5%BA%A6">定量限</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A3%80%E6%B5%8B%E9%99%90%E5%BA%A6">检测限</a> 必须与该杂质应被控制的量相当。对于未鉴定的杂质，所使用的检测方法和确定杂质量时所采用的假设应予明确说明。特定的未鉴定杂质应采用适当应用适当的方法来标示（例如：“未鉴定杂质 A”，“相对保留时间为 0.9 的未鉴定杂质”）。</p><p>建立的杂质合理限度不能高于经安全性资料界定合理的水平，并且必须与生产工艺和分析能力所能达到的水平一致。如果没有安全性方面的问题，杂质限度应根据采用拟上市工艺生产的新原料药批次测定的数据来建立，并应为常规生产和分析上的正常变异及药物的稳定性特性留有足够的余地。尽管常规生产中的变化是可以预料的，然而批与批之间杂质水平的显著变化可能预示着新原料药的生产工艺尚未得到充分的控制和验证（见 ICH <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6A_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_New_Drug_Substances_and_New_Drug_Products:_Chemical_Substances_199910">Q6A</a> “质量标准”决策树 1，建立新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 中的特殊杂质的认可标准）。限度的两位小数（见附件 1）并不代表 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%89%B9%E5%AE%9A%E6%9D%82%E8%B4%A8">特定杂质</a> 和总杂质认可标准的精度。</p><p>总之，新原料药质量标准中应包括以下杂质检查项：</p><p>有机杂质：</p><p>*每种特定的已鉴定杂质</p><p>*每种特定的未鉴定杂质。</p><p>*任何不大于（≤）鉴定限度的非特定杂质。</p><p>*杂质总量。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AE%8B%E7%95%99%E6%BA%B6%E5%89%82">残留溶剂</a></p><p>无机杂质</p><h2>7. 杂质的界定</h2><p>杂质的界定是获得和评价某些数据的过程，这些数据可用于确保单个杂质或在特定的含量下的一系列杂质的生物安全性。申请人应对所确定的杂质限度提供包括安全性研究在内的理由。对于一个通过充分的安全性研究和临床研究的新原料药，其中任何一个杂质的水平应被认为是已经通过合理界定的。对于是动物和/或人体中的重要代谢物的杂质，通常也视为已通过界定。杂质的界定限量（水平）如果高于药物实际所含的杂质量，则同样可以根据对已完成的安全性研究中使用药物中的实际杂质量来判断其合理性。</p><p>如果可获得的数据不能界定某一杂质拟定的限度，而且当该限度超过了附件 1 所列的界定限度时，则必须做进一步研究，以获得必要的数据来支持该限度的合理性。</p><p>对于某些药物，可以根据科学原理和认知水平（如药物的类别和临床经验），对其杂质界定限度进行适当调整。例如，某些杂质在一些药物中或治疗类别中已证明与病人的不良反应有关，则该杂质的界定是非常重要的。在这种情况下，应制订更低的界定限度。反之，如果考虑相似情况（患者人群、药物类别、临床情况）后，对安全性的顾虑比通常情况小，那么这些药物的界定限度可以高一些。总之，对限度的确定应具体情况具体分析。</p><p>杂质鉴定和界定的决策树（附件 3）描述了当杂质含量超过限度时所应考虑到的事项。在一般情况下降低杂质含量使其低于限度要比提供杂质的安全性数据来得简单。或者当文献资料中对某一杂质的安全性数据比较充分时，也可用于界定该杂质的限度。如果两者均不可行，则应考虑进行额外的安全性试验。如何合理地界定一个杂质将取决于许多因素，包括患病人群、每日剂量、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 与疗程。这类试验虽然可用分离出来的杂质进行研究，但通常是用含有被控制的杂质的新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 来进行研究。</p><p>虽然本指导原则在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 阶段并不适用，但在研究后期，本原则的限度对于评价拟上市新原料药生产过程中的各批次中出现的新杂质是有用的。在研究后期，任何含量大于（&gt;）附件 1 中鉴定限度（见附件 3 杂质鉴定和界定的决策树）的新的杂质均需鉴定。同样，如果杂质量大于（&gt;）附件1 中的界定限度，都应进行界定。界定某新杂质的安全性评价研究，可将含有一定代表量新杂质的原料药与以前界定过的原料药进行比较。也可考虑采用已分离出来的杂质样品进行安全性评价研究。</p><h2>8. 术语</h2><p>化学开发研究（Chemical Development Studies）：对新原料药合成工艺进行放大、优化以及验证的研究。</p><p>对映异构体杂质（Enantiomeric Impurity）：与药物具有相同的分子式，但其分子中原子的空间排列不同并且为不能重合的镜像的化合物。</p><p>外源污染物（Extraneous Contaminant）：来源于生产工艺以外的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> 。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A4%8D%E7%89%A9%E8%8D%AF">草药</a> （Herbal Products）：以植物和/或植物药制品为活性组分制成的药物制剂。在一些传统药中，可能也含无机物的材料或动物组织。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%B2%E9%89%B4%E5%AE%9A%E7%9A%84%E6%9D%82%E8%B4%A8">已鉴定的杂质</a> （Identified Impurity）：已确证了其结构特征的杂质。</p><p>鉴定限度（Identified threshold）:为一限度，高于此限度的杂质需鉴定其结构。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> （Impurity）：存在于新原料药中，但其化学结构与新原料药不一样的任何一种成分。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8%E8%B0%B1">杂质谱</a> （Impurity Profile）：对存在于某一新原料药中的已鉴定或未鉴定杂质的数量及含量。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E4%BA%A7%E5%93%81">中间体</a> （Intermediate）：化学合成新原料药的过程中所产生的某一成分，必须进一步进行结构改变才能成为新原料药。</p><p>配体（Ligand）：对金属离子具有很强亲和力的试剂。</p><p>新原料药（New Drug Substance）：先前尚未在任何成员国或地区注册的具有治疗作用的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">活性成分</a> （也称为新分子或新化学实体）。它可以是某种已获批准的药物的一种复合物、简单的酯或盐。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%9A%E6%99%B6%E5%9E%8B">多晶型</a> （Polymorphic Forms）：某一药物的不同结晶形态。包括溶剂化或水合物（伪晶体及无定型）。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BD%9C%E5%9C%A8%E6%9D%82%E8%B4%A8">潜在杂质</a> （Potential Impurity）：按照理论推测在生产或储存过程中可能产生的杂质。其在新原料药中可能存在，也可能不存在。</p><p>界定（Qualification）：是获得和评价某些数据的过程，这些数据可用于确保单个杂质或在特定的含量下的一系列杂质的生物安全性。</p><p>试剂（Reagent）：一种与起始物、中间体或溶剂不同的物质，在新原料药的生产中使用。</p><p>报告限度（reporting threshold）：为一限度，高于此限度的杂质需报告其含量。即 Q2B 中的“报告水平”（reporting level）。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BA%B6%E5%89%82">溶剂</a> （Solvent）：在新原料药合成中用于制备溶液或混悬液的无机或有机液体。</p><p>起始物料（Starting Material）：在新原料药的合成中，作为一种成分结合到中间体或新原料药中的物质。起始物料通常市场上有供应，并具有确定的化学、物理性质和结构。</p><h2>附件 1：杂质限度</h2><div><table><tbody><tr><td>每日最大剂量
