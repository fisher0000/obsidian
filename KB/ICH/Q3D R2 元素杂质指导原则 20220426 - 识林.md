href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%83%E7%B4%A0%E6%9D%82%E8%B4%A8">元素杂质</a> 有多种来源；它们可能是在合成中有意添加的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%82%AC%E5%8C%96%E5%89%82">催化剂</a> 的残留，也可能是实际存在的杂质（如：与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E4%BA%A7%E8%AE%BE%E5%A4%87">生产设备</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85">包装</a> 系统相互作用产生的杂质或药品各个组分中存在的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> ）。因为元素杂质不能为患者提供任何治疗作用， 所以它们在药品中的含量需要被控制在可接受的限度范围内。本指导原则分为三个部分：评估潜在元素杂质的毒性数据；确定每一种有毒元素的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">每日允许暴露量</a> （PDE）；以及运用基于风险的方法控制药品中的元素杂质。只要药品中的元素杂质没有超过PDE， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B3%E8%AF%B7%E4%BA%BA">申请人</a> 就不必根据生产能力收紧其限度。本指导原则确定的 PDE 值可以保证所有患者人群的公众健康。有时元素杂质在低于毒性阈值的情况下可能会影响药品的其他质量属性（如：对原料药降解有催化作用的元素），在这种情况下需要保证将元素杂质控制在一个更低的水平。另外，对于具有较高 PDE 值的元素，可能需要从药品质量的角度出发考虑其他限度并参考其他指导原则（如： <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A</a> ）。</p><p>本指导原则给出一个采用 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q9_R1_Quality_Risk_Management_20230118">ICH Q9</a> 中所述的风险管理原则来评估和控制药品中元素杂质的方法。该方法为开发基于风险的药品中元素杂质的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8E%A7%E5%88%B6%E7%AD%96%E7%95%A5">控制策略</a> 提供了平台。</p><h2>2. 范围</h2><p>本指导原则适用于新制剂（定义见 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6A_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_New_Drug_Substances_and_New_Drug_Products:_Chemical_Substances_199910">ICH Q6A</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6B_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_Biotechnological/Biological_Products">Q6B</a> ）以及含已有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的新制剂。本指导原则的适用范围包括纯化蛋白质和多肽（含重组或非重组来源的蛋白质和多肽）及其衍生物，以及以它们作为组分的药品（如：偶联物），还包括含化学 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%90%88%E6%88%90%E5%A4%9A%E8%82%BD">合成多肽</a> 、多聚核苷酸、寡糖的药品。</p><p>本指导原则不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A4%8D%E7%89%A9%E8%8D%AF">植物药</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%94%BE%E5%B0%84%E6%80%A7%E8%8D%AF%E5%93%81">放射性药物</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%96%AB%E8%8B%97">疫苗</a> 、细胞代谢产物、DNA 产品、过敏原提取物、细胞、全血、血细胞成份或包括血浆及血浆衍生物在内的血液衍生物、非体循环透析液、以及为了治疗作用而特意添加到药品中的元素。本指导原则也不适用于基于基因（ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9F%BA%E5%9B%A0%E6%B2%BB%E7%96%97">基因治疗</a> ）、细胞（ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%86%E8%83%9E%E6%B2%BB%E7%96%97">细胞治疗</a> ）和组织（组织工程）的药品。在某些地区，这些药品被称为先进治疗药品。</p><p>本指导原则不适用于在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 阶段使用的药品。随着商业开发的进程，可以采用本指导原则的原理来评估新药中可能存在的元素杂质。</p><p>ICH 颁布本指导原则 36 个月内，已上市药品无需实施Q3D。</p><h2>3. 潜在元素杂质的安全性评估</h2><h2>3.1 口服、注射和吸入给药途径的元素杂质的安全性评估原则</h2><p>附录 1 详细讨论了每个元素杂质的 PDE 的确定方法。根据科学杂志、政府研究报告与课题、国际法规标准（适用于药品）和指导原则、监管机构的研究和评估报告等公开可获得的数据，对本指导原则中的元素进行评估。评估过程遵从“ <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3C_R9_Impurities_Guideline_for_Residual_Solvents_20240124">ICH Q3C：残留溶剂</a> ”中所述原则。综合所获得信息，确定了口服、注射和吸入 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 的PDE。为了便于应用，附录 2 表 A.2.1 中列出的适用于药品的PDE 值已修约至 1 或 2 位有效数字。</p><p>附录 3 包含了确定各元素 PDE 的关键研究的安全性评估总结。由于数据不充足，无法确定铱、锇、铑和钌的任何给药途径的 PDE。根据这些元素与钯的相似性，确定其PDE。</p><p>在确定PDE 值的安全性评估中所需考虑的因素，依大致相关性排序如下：</p><ul><li>药品中元素可能的氧化价态；</li></ul><ul><li>可获得的人体 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9A%B4%E9%9C%B2%E9%87%8F">暴露量</a> 和安全性数据；</li></ul><ul><li>最相关的动物研究；</li></ul><ul><li>给药途径；</li></ul><ul><li>相关终点。</li></ul><p>本指导原则中讨论的一些元素杂质已有食物、水、空气和职业暴露的日摄入量标准。在适当的时候，可以考虑采用这些标准来进行安全性评估和确定 PDE。</p><p>通常采用持续时间最长的动物研究结果来确定 PDE。如果采用较短期的动物研究作为最相关的结果，则在每个元素的安全性评估中说明其合理性。</p><p>对于吸入途径的安全性评估和吸入 PDE 值的推算，使用可溶性盐的吸入研究（如可获得）要优于使用颗粒的研究。依据可获得的数据，吸入途径的 PDE 值既可以局部毒性（呼吸系统）为基础，也可以全身毒性为基础。将剂量换算成 24 小时 7 天的暴露量，来确定吸入途径的PDE（口服或注射途径也适用）。</p><p>在注射和/或吸入给药途径的安全性评估数据缺乏和/或数据不充分的情况下， 可根据口服 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%A9%E7%94%A8%E5%BA%A6">生物利用度</a> 的校正因子由口服 PDE 推算注射和/或吸入给药途径的PDE：</p><ul><li>口服生物利用度&lt;1%：除以校正因子 100；</li></ul><ul><li>口服生物利用度≥1%且&lt;50%：除以校正因子 10；</li></ul><ul><li>口服生物利用度≥50%且&lt;90%：除以校正因子 2；</li></ul><ul><li>口服生物利用度≥90%：除以校正因子 1。</li></ul><p>在无口服生物利用度数据或职业吸入暴露限度的情况下，以口服 PDE 值除以校正因子 100 来计算吸入PDE <sup>[1]</sup> 。</p><h2>3.2 其他给药途径</h2><p>已确定口服、注射和吸入给药途径的 PDE。另外，附录 5 提供了皮肤和透皮给药途径的元素杂质的 PDE。当有必要确定其他给药途径的 PDE 值时，可采用本指导原则中所述概念进行推算。通过评估可增加或降低既定 PDE 值。其他给药途径的PDE 值的推算过程如下：</p><ul><li>以附录 3 中口服 PDE 值作为启动点，推算特定给药途径的 PDE 值。通过科学评估，注射和吸入途径的PDE 值更适合作为启动点。</li></ul><ul><li>评估以拟定给药途径给药时元素杂质是否会产生局部作用：</li></ul><dl><dd>o 如果产生局部作用，评估是否有必要对既定PDE 值进行修正。</dd></dl><dl><dd>o 关注那些预计会产生与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%8D%E8%89%AF%E5%8F%8D%E5%BA%94">不良反应</a> （用来确定既定PDE）相关作用的剂量/暴露量。</dd></dl><dl><dd>o 如果不产生局部作用，则无需对既定PDE 值进行修正。</dd></dl><ul><li>如可能，对拟定给药途径的元素的生物利用度进行评估，并与既定 PDE 值的给药途径的元素的生物利用度进行比较：</li></ul><dl><dd>o 如有差别，则需对既定 PDE 值引入一个校正因子。例如，当没有局部作用发生时，如果某一元素的口服生物利用度是 50%，而拟定给药途径的生物利用度是 10%，则校正因子为 5。</dd></dl><ul><li>如果新给药途径推荐的PDE 值高于既定PDE 值，则需要考虑质量属性。</li></ul><h2>3.3 元素杂质水平高于既定 PDE 值的论证</h2><p>在特定情况下， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%83%E7%B4%A0%E6%9D%82%E8%B4%A8">元素杂质</a> 水平可以高于既定 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> 值（见表 A.2.1）。这些情况包括但不限于：</p><ul><li>间隔给药；</li></ul><ul><li>短期给药（如：30 天或更短）；</li></ul><ul><li>特殊 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%80%82%E5%BA%94%E7%97%87">适应症</a> （如：危及生命的、未满足的医疗需求、罕见病）。</li></ul><p>以下提供了使用调整因子的次因子法论证提高元素杂质限度水平的实例 <sup>[2,3]</sup> 。也可以采用其他方法进行论证。任何高于既定 PDE 值的拟定水平都应视具体情况进行论证。</p><p>实例 1：某口服药品中存在元素 X。由附录 3 中元素X 的各论可知，其 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:NOAEL">未见不良反应剂量</a> （NOAEL）为 1.1 mg/kg/天，调整因子 F1-F5 分别为 5、10、5、1 和 1。采用附录 1 中所述调整因子标准方法计算PDE 值如下：</p><center>PDE=1.1 mg/kg/dx50 kg/（5x10x5x1x1）=220 µg/天</center><p>调整因子 F2（默认值=10）可细分为两个次因子，一个代表毒动学（TK），一个代表毒效学，每个值都在 1 至 3.16 的范围内。血浆半衰期以 5 天计，每周给药一次（~1 个 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8D%8A%E8%A1%B0%E6%9C%9F">半衰期</a> ）TK 次因子可降至 1.58，每月给药一次（~5 个半衰期） 可降至 1。使用 F2 次因子法，元素X 每周给药一次的推荐水平计算如下：</p><center>推荐水平=1.1 mg/kg/dx50 kg/（5x（1.6x3.16）x5x1x1）=440 µg/天</center><p>为了便于应用，将该数值修约至 400 µg/天。</p><p>实例 2：TK 次因子法也适用于不使用调整因子方法计算的元素杂质。使用最小风险水平（MRL）0.02 mg/kg/天推算元素 Z 的口服PDE 值。根据文献报道， 血浆半衰期是 4 天。该元素是某口服药品的杂质，该药品每 3 周给药一次（~5 个半衰期）。依据一级反应动力学，将既定 PDE 值 1000 µg/天修正如下：</p><center>推荐水平=0.02 mg/kg/dx50 kg/（1/3.16）=3.16 mg/天</center><p>为了便于应用，将该数值修约至 3000 µg/天。</p><h2>3.4 注射剂</h2><p>每日最大剂量不超过 2 升的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B3%A8%E5%B0%84%E5%89%82">注射剂</a> 可使用每日最大剂量由 PDE 值计算允许浓度。 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">说明书</a> 规定和/或临床实践确定日剂量超过 2 升的药品（如：生理盐水、葡萄糖、全胃肠外营养液、灌洗液），可使用 2 升体积由 PDE 值计算允许浓度 <sup>[4]</sup> 。</p><h2>4. 元素分类</h2><p>根据元素的毒性（PDE）及其在药品中出现的可能性，将本指导原则中元素分为 3 类。出现的可能性有多种因素，包括：在制药过程中使用的可能性，在制药过程所用物料中与其他元素杂质共生的可能性，以及元素的表观天然丰度和环境分布。本指导原则中低天然丰度元素指的是天然丰度报道值≤1 个原子/10 <sup>6</sup> 个硅原子的元素 <sup>[5]</sup> 。分类表旨在将风险评估聚焦于那些毒性最大并且在药品中极有可能出现的元素（见表 5.1）。元素杂质的分类如下：</p><p><b>1 类：</b> 元素砷（As）、镉（Cd）、汞（Hg）和铅（Pb）是人体毒素，在药品生产中应限制使用或禁用。在药品中出现的这类元素通常来自常用 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%89%A9%E6%96%99">物料</a> （如：矿物质 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> ）。由于它们的独特性质，这四种元素的所有潜在元素杂质来源以及给药途径都需要进行风险评估。根据风险评估的结果确定那些可能需要额外控制的组分，在某些情况下额外控制可能包括对 1 类元素进行检测。不是所有组分都需要检测 1 类元素杂质，仅当风险评估确定需要进行恰当控制以确保符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> 值时才实施检测。</p><p><b>2 类：</b> 这类元素通常被认为是给药途径依赖型的人体毒素。根据它们出现在药品中的相对可能性，进一步分成 2A 和 2B 亚类。</p><ul><li><b>2A 类</b> 元素出现在药品中的相对可能性高，因此对所有潜在元素杂质来源以及给药途径（见说明书）都需要进行风险评估。2A 类元素包括：钴（Co）、镍（Ni）和钒（V）。</li></ul><ul><li><b>2B 类</b> 元素丰度较低并且与其他物料共生的可能性较低，因此出现在药品中的概率较低。除非在原料药、辅料或其他药品组分生产中有意添加这些元素，否则无需进行风险评估。2B 类元素包括：银（Ag）、金（Au）、铱（Ir）、锇（Os）、钯（Pd）、铂（Pt）、铑（Rh）、钌（Ru）、硒（Se）和铊（Tl）。</li></ul><p><b>3 类：</b> 此类元素口服给药途径的毒性相对较低（高 PDE 值，通常&gt;500µg/天），但在吸入和注射 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 的风险评估中仍需考虑。除非有意添加这些元素，否则在口服给药途径的风险评估中不需考虑。在注射和吸入给药药品的风险评估中， 应对是否可能含有这些元素杂质进行评估，除非该给药途径特定的 PDE 值高于500 µg/天。此类元素包括：钡（Ba）、铬（Cr）、铜（Cu）、锂（Li）、钼（Mo）、 锑（Sb）和锡（Sn）。</p><p><b>其他元素：</b> 由于固有毒性低和/或区域监管的差异，有些元素杂质的PDE 值未被确定，本指导原则未涉及此类元素。如果药品中存在或包含这些元素，应遵从适用于特定元素的其他指导原则和/或地方法规和规范（如：铝导致肾功能损伤；锰和锌导致肝功能损伤）或药品的质量考虑（如：治疗性蛋白质中存在的杂质钨）。需考虑的一些元素包括： 铝（Al）、硼（B）、钙（Ca）、铁（Fe）、钾（K）、镁（Mg）、锰（Mn）、钠（Na）、钨（W）和锌（Zn）。</p><h2>5. 元素杂质的风险评估和控制</h2><p>可以采用 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q9_R1_Quality_Risk_Management_20230118">ICH Q9</a> 中所述的质量风险管理原则，来开发药品中元素杂质的控制措施。根据科学的认知和原则进行风险评估。将对药品和生产工艺的理解与患者的安全性考虑联系起来（ <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q8_Pharmaceutical_Development">ICH Q8</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q11_Development_and_Manufacture_of_Drug_Substance">Q11</a> ）。因此，对于元素杂质，药品风险评估的重点是结合本指导原则中的 PDE 值评估药品中元素杂质的水平。用于这种风险评估的信息包括但不限于： <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B3%E8%AF%B7%E4%BA%BA">申请人</a> 的申报数据，原料药和/或辅料生产商提供的信息和/或公开可获得的文献数据。</p><p>申请人应以适当的方式记录风险评估和控制方法。风险评估的程度和形式应与风险等级成正比。并非所有情况都适合或有必要使用正式的风险管理流程（使用公认的工具和/或正式的程序，例如： <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E5%87%86%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">标准操作规程</a> ）。也可以使用非正式的风险管理流程（使用经验工具和/或内部程序）。ICH Q8 和 Q9 中介绍了风险评估的辅助工具，本指导原则中不再介绍。</p><h2>5.1 总则</h2><p>基于本指导原则建立的目的，风险评估可分为三个步骤：</p><ul><li>确定药品中已知和潜在的元素杂质来源。</li></ul><ul><li>通过确定实测或预测的杂质水平并与既定PDE 值比较，来评估药品中存在的特定元素杂质。</li></ul><ul><li>总结和记录风险评估。确定已建立的过程中控制是否充分或者是否需要额外的控制来限定药品中的元素杂质。</li></ul><p>很多情况下，需要同时考虑这些步骤。风险评估的结果叠加，开发一种最终方法，确保潜在元素杂质不超过PDE 值。</p><h2>5.2 元素杂质的潜在来源</h2><p>在药品生产中，元素杂质的潜在来源广泛，包括：</p><ul><li>在原料药、辅料或其他药品组分生产中有意添加元素（如催化剂）的残留。原料药的风险评估应注意可能包含在药品中的元素杂质。</li></ul><ul><li>非有意添加但在药品生产所用原料药、水或辅料中可能存在的元素杂质。</li></ul><ul><li>生产设备可能引入到原料药和/或制剂中的元素杂质。</li></ul><ul><li>包装系统可能浸出至原料药和制剂中的元素杂质。</li></ul><p>下图是用于药品生产的典型物料、设备和组分的实例。通过上述每一种潜在的来源，元素杂质可以任何单独或组合的形式被引入到药品中。风险评估过程中， 元素杂质的每一种来源的潜在贡献都需考虑，以确定元素杂质对药品的整体贡献情况。</p><div><a 潜在元素杂质的识别</h2><p><b>由有意添加的催化剂和无机试剂引入的潜在元素杂质：</b> 如果有意添加任何表 5.1 中列出的元素，则应在风险评估中予以考虑。此类 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BD%9C%E5%9C%A8%E6%9D%82%E8%B4%A8">潜在杂质</a> 为已知杂质，其控制技术易于表征和确定。</p><p><b>可能存在于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 和/或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 中的潜在元素杂质：</b> 尽管不是有意添加，有些 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%83%E7%B4%A0%E6%9D%82%E8%B4%A8">元素杂质</a> 也可能出现在某些原料药和/或辅料中。这些元素被引入到药品中的可能性也应在风险评估中反映。</p><p>对口服给药途径，风险评估应评估药品中是否可能存在 1 类和 2A 类元素杂质。对注射和吸入给药途径，风险评估应评估是否可能存在表 5.1 所示的 1 类、2A 类和 3 类元素杂质。</p><p><b>由生产设备引入的潜在元素杂质：</b> 此来源元素杂质的贡献有限，应根据药品生产所用的生产设备确定需考虑的元素杂质的种类。工艺知识的应用、设备选择、设备认证以及GMP 控制都能保证来自生产设备的元素杂质的贡献较低。根据对与药品组分接触的生产设备组件组成的认知，对关注的特定元素杂质进行评估。此来源元素杂质的风险评估可以被用于有着相似生产工艺的多种药品。</p><p>通常，当评估从生产设备中可能浸出或迁移的元素杂质时，一个具体原料药的制备过程比制剂过程更具代表性。来自制剂生产设备的元素杂质的贡献预计会低于原料药观察到的贡献。然而，根据对工艺的认知或理解，如果不是上述这种情况，申请人还应在风险评估中考虑从制剂生产设备引入元素杂质的可能性（如： <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%83%AD%E7%86%94%E6%8C%A4%E5%87%BA">热熔挤出</a> ）。</p><p><b>包装系统中浸出的元素杂质：</b> 根据对特定药品类型与包装系统之间潜在相互作用的科学理解，识别包装系统引入的潜在元素杂质。当包装系统组成的综述资料表明包装系统不含元素杂质时，无需进行额外的风险评估。元素被浸出进入固体制剂的可能性极小，因此无需在风险评估中进一步考虑。液体和半固体制剂在有效期内元素杂质从包装系统中浸出的可能性高，应对包装系统的潜在浸出物进行研究（洗涤、灭菌、辐射等操作后）。通常在制剂的包装系统研究中对此来源的元素杂质进行评估。</p><p>需考虑的因素（对于液体和半固体制剂）包括但不限于：</p><ul><li>亲水性/疏水性；</li></ul><ul><li>离子强度；</li></ul><ul><li>pH；</li></ul><ul><li>温度（冷链与室温的对比和工艺条件）；</li></ul><ul><li>接触表面积；</li></ul><ul><li>容器/组件组成；</li></ul><ul><li>终端灭菌；</li></ul><ul><li>包装过程；</li></ul><ul><li>组件灭菌；</li></ul><ul><li>贮存期限。</li></ul><h2>5.4 风险评估过程总结</h2><p>风险评估的总结是通过综合相关药品或组分的特定数据，并结合从药品或制备工艺中获得的信息和认知，用以识别在药品中很可能观察到的元素杂质。</p><p>总结中应考虑实测的或预测的元素杂质水平相对于其 PDE 值的显著性。将药品既定 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> 值的 30%定义为控制阈值，作为实测元素杂质水平显著性的衡量指标。控制阈值可用于判断是否需要额外的控制。</p><p>如果药品中所有来源的总元素杂质水平始终低于 PDE 的 30%，只要申请人对数据进行了适当的评估并证明已对元素杂质进行了足够的控制，则不再需要额外的控制。</p><p>如果风险评估无法证明某个元素杂质水平始终低于控制阈值，就需要建立控制方法以保证药品中元素杂质水平不超过PDE 值（参见第 6 章）。</p><p>在使用药品控制阈值时，应考虑元素杂质水平的变异性。变异的来源可能包括：</p><ul><li><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 的变异；</li></ul><ul><li>特定来源的元素杂质水平的变异；</li></ul><ul><li>药品中元素杂质水平的变异。</li></ul><p>递交申报资料时，在缺乏其他证明的情况下，可通过提供 3 批代表性生产规模或 6 批代表性中试规模的组分或药品的数据，来建立元素杂质水平和变异性。对于一些具有固有变异性的组分（如：矿物来源辅料），使用控制阈值还需要额外的数据。</p><p>可采用多种方法对风险评估进行总结和记录，包括：以表格、文字形式对评估考虑和结论进行总结。总结应明确元素杂质、来源及必要的控制方法和验收标准。</p><h2>5.7 生物制品的特殊考虑</h2><p>对于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%B6%E5%93%81">生物制品</a> ， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 阶段 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%83%E7%B4%A0%E6%9D%82%E8%B4%A8">元素杂质</a> 达到引发安全担忧水平的风险很低。这主要是因为：a）生物制品生产中通常不采用元素作为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%82%AC%E5%8C%96%E5%89%82">催化剂</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AF%95%E5%89%82">试剂</a> ；b） <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%86%E8%83%9E%E5%9F%B9%E5%85%BB">细胞培养</a> 过程中在培养基中加入痕量元素，这些元素无蓄积且在进一步的处理过程中可被显著稀释/清除；c）用于生物制品生产过程的典型纯化方法，例如萃取、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%89%B2%E8%B0%B1">色谱</a> 分离和透析或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B6%85%E6%BB%A4">超滤</a> -渗滤（UF/DF）都具有将细胞培养/发酵步骤或与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E4%BA%A7%E8%AE%BE%E5%A4%87">生产设备</a> 接触引入的元素杂质清除至可忽略水平的能力。因此，通常不需要对生物制品原料药的元素杂质进行特殊控制。当生物制品原料药含有合成结构（如：抗体-药物偶联物）时，需要对小分子部分的元素杂质的控制方法进行评估。</p><p>然而，对于生物制品制剂，则需考虑在药品生产过程中元素杂质的潜在来源（如：辅料）和其他环境性来源。需要评估这些元素杂质来源对制剂的贡献，因为元素杂质通常可在药品生产的某步工艺过程中引入，而在后续步骤中没有去除元素杂质的操作。在评估中应考虑的风险因素包括所用辅料类型、工艺条件及其对环境因素（如： <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%97%A0%E8%8F%8C">无菌</a> 生产的受控区域和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BA%AF%E5%8C%96%E6%B0%B4">纯化水</a> 的使用）的易污染性和整体给药频率。</p><h2>6. 元素杂质的控制</h2><p>元素杂质的控制是药品整体控制策略的一部分，用以确保元素杂质不超过 PDE 值。当元素杂质水平超过控制阈值时，需采取额外的措施来确保元素杂质水平不超过PDE 值。 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B3%E8%AF%B7%E4%BA%BA">申请人</a> 能够遵循的方法包括但不限于：</p><ul><li>调整生产工艺步骤，通过特定或非特定的纯化步骤将元素杂质降低至控制阈值之下；</li></ul><ul><li>实施工艺过程中或上游控制，旨在将药品中元素杂质的浓度限制在控制阈值以下；</li></ul><ul><li>建立辅料或物料（如：合成中间体）的标准限度；</li></ul><ul><li>建立原料药的标准限度；</li></ul><ul><li>建立制剂的标准限度；</li></ul><ul><li>选择合适的包装系统。</li></ul><p>根据 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6A_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_New_Drug_Substances_and_New_Drug_Products:_Chemical_Substances_199910">ICH Q6A</a> 的原则对元素杂质进行定期检测。</p><p>申报资料中提供的有关元素杂质控制的信息包括但不限于：风险评估总结、适当的数据（如需要），以及确定元素杂质限度控制的描述。</p><h2>7. PDE 值与浓度限度的相互转换</h2><p>本文中 PDE 值代表每日最大给药剂量的药品中每个元素的最大日允许摄入量，以 µg/天表示。由于 PDE 值仅反映了来自药品的总暴露量，因此将 PDE 值转换为浓度，作为一种评估药品或其组分中元素杂质含量的工具更为实用。本章列出了一些确定药品或其组分中元素杂质的可接受浓度的方法，以确保药品中元素杂质不超过PDE 值。只要所获得的允许浓度确保药品不超过 各论中使用的形态具有更高或更低毒性时，可使用这些信息证明更低或更高水平的合理性。</p><p>当基于组分中总元素杂质水平进行风险评估时， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B3%E8%AF%B7%E4%BA%BA">申请人</a> 不必提供某元素杂质从其存在组分中释放的相关信息。但是，此信息能够被用来证明比药品总元素杂质含量具有更高水平的合理性。</p><h2>9. 分析方法</h2><p>应采用适当的符合预期目的的方法检测元素杂质。除非另有说明，方法应对在风险评估中识别出的需要控制的元素杂质具有专属性。可采用药典方法或其他适合的方法测定元素杂质水平。</p><h2>10. 生命周期管理</h2><p><a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q10_Pharmaceutical_Quality_System">ICH Q10</a> 中所述的质量体系和管理职责旨在鼓励在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 的每个阶段都使用基于科学和风险的方法，从而促使药品整个生命周期的持续改进。药品和工艺知识的管理应从研发开始并贯穿于药品商业生命直至药品撤市。</p><p>从开发中获得的认知，结合商业生产的经验和数据可用来进一步提升对工艺的理解和工艺性能。这种提升能够增强对元素杂质的控制。本指导原则发布时， 一些可用于指导申请人进行特定元素杂质控制的数据有限。随着开发的深入，可根据额外的数据改进控制方法。</p><p>如果药品或组分的变更对药品中元素杂质的量有潜在影响，应重新进行风险评估，包括既定元素杂质的控制方法。这些变更包括但不限于：合成路线、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 供应商、原材料、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA">工艺</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AE%BE%E5%A4%87">设备</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85">包装</a> 系统或 <a 。人体的癌症和非癌症效应都与砷暴露量有关。口服暴露与皮肤、肝、肺、肾和膀胱的癌症有关。有证据表明吸入暴露可增加患肺癌的风险 <sup>[1,4~7]</sup> 。</p><p>皮肤（色素沉着、掌跖角化病）和胃肠道（如：恶心）似乎是口服摄入后非癌症不良反应的最敏感的靶器官，而血管疾病、对生殖和神经的影响也被报道为非癌症终点 <sup>[1,3,8]</sup> 。口服研究显示高于 0.02 mg As/kg/天的暴露水平可出现皮肤病变；从 0.0004 到 0.01 mg As/kg/天的水平通常观察不到不良反应 <sup>[4]</sup> 。尚无充分的流行病学数据用以设置其他终点的 LOEL 或 NOEL 值。局部过度角化症可进展为皮肤癌 <sup>[4]</sup> ，且被认为可能预测皮肤与内部癌症以及非癌症长期不良健康反应 <sup>[9~11]</sup> 。</p><p>以中国台北西南部大量人群（约 40000 人）饮用砷浓度为 1000 μg/L 甚至更高的井水患皮肤癌以及最近报道的膀胱癌和肺癌的研究作为风险评估的基础 <sup>[7]</sup> 。最近癌症风险的 meta 分析表明，低浓度暴露（&lt;100~200 μg/L）不会发生额外的膀胱癌风险 <sup>[12~14]</sup> 。这与Schuhmacher-Wolz 等人 <sup>[3]</sup> 的研究结果一致。</p><p>美国环境保护署（US EPA）基于来自两个冶炼厂的数据确定了吸入途径的癌症风险单位值为 0.0043 μg/m <sup>3</sup> <sup>[8]</sup> 。德克萨斯环境质量委员会结合后续多年随访的US EPA 的数据以及来自英国和瑞典工人的附加数据，向 US EPA 提供了更新后的单位风险系数（URF）。委员会计算的 URF 元素杂质指导原则。该指导原则给出了 24 种元素杂质（EI）口服、注射和吸入给药途径的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">每日允许暴露量</a> （PDE）。在指导原则的第 3.2 章，给出了建立其他 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> PDE 的原则。在 Q3D 开发进程中，开发皮肤和透皮给药途径 PDE 引起关注，因为这些药品是尚未正式建立元素杂质 PDE 的药品中最大的一类。附录 5 旨在扩展 Q3D 指南正文中提供的信息，并提供皮肤和透皮给药途径更具体的信息。</p><p>建立皮肤和透皮给药途径限度时，皮肤的作用至关重要。皮肤是环境屏障， 也是具有许多功能的复杂器官，包括限制异物渗透、代谢、防止水分流失、温度调节以及作为免疫器官 <sup>[1]</sup> 。皮肤由外部表皮和内部真皮组成，都由许多细胞层构成。真皮（或透皮）吸收，即化合物从皮肤外表面进入全身循环，取决于皮肤的特性、解剖部位、化合物的特性以及给药的特性。</p><p>吸收的主要屏障是表皮的最外层（即角质层），通常由 10-20 层无活性细胞组成。角质层是一种非常有效的屏障，尤其是对带电荷物质（如金属离子）。因此，包括任何 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> （API）在内的物质透皮进入全身循环通常需要理化试剂（如促渗剂）以帮助 API 的透皮吸收。</p><p>关于这些“促渗剂”，值得注意的是，由于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%90%86%E5%8C%96%E7%89%B9%E6%80%A7">理化性质</a> 的本质差异，增强 API渗透的促渗剂通常不适用于元素杂质。评估皮肤给药途径元素杂质全身吸收的研究有限。皮肤对暴露的反应不同。例如，大约一半被皮肤吸收的汞蒸气（剂量的1-4%）在暴露后几周随着表皮细胞脱落而除去，皮肤中剩余部分缓慢释放进入全身循环 <sup>[2]</sup> 。Hostýnek 等人提出银（Ag）优先蓄积在皮肤上并且不会被释放 <sup>[3]</sup> 。现有数据表明金（Au）呈惰性并且在体液中不被离子化，不易通过皮肤吸收 <sup>[4]</sup> 。金盐易于与表皮角蛋白的巯基结合并留在皮肤上 <sup>[4]</sup> 。在某些婴儿和成人皮肤上存在金属结合蛋白（例如表皮和外发根鞘的基底角质形成细胞），但在其他类型细胞（例如外分泌腺外分泌部分）中没有，表明皮肤具有金属结合和代谢的功能 <sup>[5]</sup> 。</p><p>正如 Hostýnek 等人对定量吸收数据的综述所示 <sup>[3]</sup> ，皮肤层的这些特性表明皮肤是全身暴露的重要屏障。据报道，本指导原则评估的大多数元素杂质的全身暴露吸收均＜1%。在第 3 章对元素杂质的透皮吸收进行详细讨论。</p><p>通过对科学杂志、政府研究报告和研究、监管机构研究和评估报告等公开数据进行回顾，来对本指导原则中的元素进行评估。通常科学文献研究仅报道元素杂质从皮肤层消失而不是透皮吸收。大多数元素杂质及其配对离子的定量数据缺失 <sup>[6]</sup> 。此外，没有合适的职业暴露标准可用于进行皮肤给药途径的风险评估。因此，采用一种通用方法来建立限度，而不是逐个元素地建立限度。</p><h2>2 范围1</h2><p>本 Q3D 指导原则附录适用于起局部作用或全身作用的皮肤和透皮给药途径药品（在本附录中简写为皮肤给药途径药品）。本附录不适用于粘膜给药（口腔、鼻腔、阴道）、局部眼科用药、直肠给药、表皮下和真皮下给药的药品。本附录未涵盖的药品应根据 Q3D 指南正文第 3.2 章中讨论的方法进行评估。</p><h2>3 皮肤给药途径药品的安全性评估原则</h2><p>文献综述着重于药品中可能存在的形式（见指导原则正文），因此对无机形式元素杂质的可用文献进行评估，数据相关性顺序如下：人体内数据、动物体内数据、体外数据。</p><p>考虑局部和全身毒性。通常，除致敏作用外，没有迹象表明对皮肤有局部毒性。对皮肤给药途径的全身毒性进行综述，结果显示铊具有显著的全身毒性。由于本附录中元素的透皮吸收的信息有限，不能得到每个元素的吸收百分数并转化成皮肤给药途径的PDE，所以不支持采用逐个元素的方法。因此，开发出一种通用方法，假设注射给药途径生物利用度为 100%，根据皮肤给药途径全身暴露， 用皮肤给药途径调整因子（CMF），对注射给药途径 PDE 进行校正，得到皮肤给药途径PDE（见第 4 章）。皮肤给药途径 PDE 适用于长期每天皮肤给药。</p><h3>3.1 元素杂质（EI）的透皮吸收</h3><p>吸收进入全身循环（全身吸收）的程度是元素的安全性评估的重要组成部分。对元素的皮肤渗透、吸收、全身生物利用度和毒性研究进行回顾，结果显示许多元素数据缺失。对于已有透皮吸收和/或毒性研究的元素，很少数据适用于正确的定量分析，并且多样化的试验设计不能进行各项研究之间或各元素之间的比较 <sup>[6]</sup> 。现有数据表明，即使存在促渗剂，元素杂质通常也很难通过完整皮肤吸收。例如， 暴露后12 天测定尿中Pb，在闭合状态下大鼠从氧化铅中吸收的Pb 小于0.005%。在人皮肤的体外系统中没有检测到氧化铅渗透 <sup>[7]</sup> 。</p><p>皮肤途径给药后，很多因素可能影响透皮吸收和全身 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%A9%E7%94%A8%E5%BA%A6">生物利用度</a> 。这些因素分类如下：</p><ul><li>化合物相关因素（例如元素杂质的物理状态、离子化、溶解度、结合特性、反应性和配对离子），和/或</li></ul><ul><li>给药相关因素（例如给药浓度和总剂量、给药/暴露的持续时间、给药之间的清除、表面积、共同给药的物质/辅料和闭合状态）</li></ul><ul><li>受试者相关因素（例如人种差异的可比性、身体部位、皮肤水合作用/年龄、温度）</li></ul><p>通过皮肤的透皮渗透与元素和化合物种类有关，需要在不同条件下对每种元素进行试验评估，以开发有效的模型。由于这种复杂性，得到每种药品中每种元素杂质的每种可能情况是不可行的。</p><p>鉴于在设计良好的研究中通过皮肤给药途径获得的有关透皮吸收和毒性研究的数据有限，因此利用可用数据开发一种通用的保守的方法。用先前根据适当的毒性数据建立的每种元素注射给药途径 PDE 得到皮肤给药途径 PDE。为了得到这种假设低但未量化的透皮吸收，并考虑所有可能影响吸收的潜在因素，对大多数元素杂质，在注射给药途径 PDE 的基础上使用 10 倍系数。10 倍系数的推导和应用详见第 4 章。</p><h3>3.2 直接作用于真皮的药品的 PDE</h3><p>受损的基底细胞层能够促进元素杂质直接进入真皮及其周围血管（可能会增加全身吸收）。因此，本附录中皮肤给药途径的通用 PDE 不适用于治疗表皮的基底细胞层受到实质性破坏的皮肤给药药品。对于药物需要与真皮接触的适应症（例如，皮肤溃疡、二度和三度烧伤、大疱疮、大疱性表皮松懈症），建议根据 ICH Q3D 第 3.3 章原则建立针对具体案例的合理依据。通常适合采用注射给药途径PDE 作为这些药品的启动点。</p><p>小伤口、针刺、皮肤擦伤和其他快速愈合的日常皮肤损伤与上述表皮的基底细胞层实质性破坏无关。因此，可能通过接触而进入真皮的药品总量可以忽略不计。所以，皮肤给药途径 PDE 适用于治疗这些皮肤擦伤或者其他快速愈合急性损伤的药品。</p><h2>4 皮肤给药途径每日允许日暴露量（PDE）的建立</h2><p>用每种元素杂质注射给药途径的PDE 和皮肤给药途径调整因子（CMF），计算得到各元素杂质皮肤给药途径的PDE。</p><h3>4.1 皮肤给药途径调整因子（CMF）的建立</h3><p>如上所述（第 1 章和第 3 章），有限的可用的完整皮肤研究数据表明，大多数元素杂质的透皮吸收小于 1%。如第 3.1 章所述，很多因素可能影响这种吸收。为避免逐一考虑这些因素，同时考虑到相对缺乏可靠的定量透皮吸收数据，采用一种可以防止潜在全身毒性的方法得到皮肤给药途径 PDE。为了解决这些不确定性，采用以下方法得到CMF。</p><dl><dd>1. 除砷（As）和铊（Tl）以外的元素杂质，采用最大皮肤给药途径生物利用度（CBA）为 1%。</dd></dl><dl><dd>2. 考虑到可能提高 CBA 的各种因素，采用 10 倍系数来增加 CBA（调整后CBA）。</dd></dl><dl><dd>3. 用注射给药途径 BA（100%）除以调整后 CBA，计算CMF。</dd></dl><h3>4.2 皮肤给药途径 PDE</h3><p>皮肤给药途径PDE 计算如下：</p><dl><dd>皮肤给药途径PDE=注射给药途径PDE×CMF</dd></dl><p>注射给药途径 PDE 的计算已经包括安全系数 F1-F5 或者从口服给药途径 PDE 计算得出，口服给药途径 PDE 也包括安全系数以说明变异性和外推（见 ICH Q3D 附录 1）。因此，皮肤给药途径 PDE 不需要进一步调整。</p><p>表 A.5.1 列出了皮肤给药途径PDE。</p><h4>4.2.1 除砷（As）和铊（Tl）以外的元素杂质的 PDE</h4><p>对于低CBA（＜1%）的元素杂质，采用 CMF 为 10。</p><p>对于CBA＜1%的元素杂质，调整后 CBA=1%×10=10%。</p><p>注射给药途径BA 除以调整后CBA 得到CMF。</p><p>100%/10%=10。</p><p>皮肤给药途径PDE 计算：</p><dl><dd>皮肤给药途径PDE=注射给药途径PDE×CMF。</dd></dl><dl><dd>皮肤给药途径PDE=注射给药途径PDE×10。</dd></dl><p>表 A.5.1 列出了各元素杂质皮肤给药途径 PDE。</p><h4>4.2.2 砷的 PDE</h4><p>对于无机砷，现有数据表明其透皮吸收大于大多数其他元素杂质（大约 5%） <sup>[8]</sup> 。因此，砷的CMF 为 2，计算如下：</p><dl><dd>调整后CBA 计算：5%×10=50%。</dd></dl><p>将注射给药途径BA 除以调整后CBA 得到CMF</p><p>100%/50%=2。</p><p>皮肤给药途径PDE 计算：</p><dl><dd>皮肤给药途径PDE=注射给药途径PDE×CMF。</dd></dl><dl><dd>皮肤给药途径PDE=15μg/day×2=30μg/day。</dd></dl><h4>4.2.3 铊的 PDE</h4><p>铊通过皮肤被大量吸收。由于无法获得定量数据，假定其等效于注射给药途径水平。调整后PDE 等于注射给药途径PDE，CMF 为 1。</p><p>皮肤 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> 外还应该限制浓度，以减少已过敏个体引起皮肤反应的可能性。该浓度限度被称为皮肤和透皮给药途径浓度限度（CTCL）。对于其他元素杂质（例如 Cr），引起过敏反应的阈值大约等于皮肤给药途径PDE（Cr）或者远大于皮肤给药途径 PDE，因此不需要额外控制 <sup>[9]</sup> 。</p><p>Menné 等人最初建立了 Ni 的皮肤浓度限度为 0.5μg/cm <sup>2</sup> /week，作为二甲基乙二肟（DMG）检查的检测限 <sup>[10]</sup> 。根据欧盟国家 Ni 法规和欧盟镍指令（REACH, Entry 27, Annex XVII）的规定，该 Ni 限度适用于直接和长期皮肤接触消费品（例如珠宝）中。该指令实施后，镍过敏的患病率显著下降 <sup>[11,12]</sup> 。采用该限度来建立药品中 Ni 的皮肤给药途径的浓度。用于患病皮肤部位的最小剂量单位称为 1 个指尖单位（FTU），约为 0.5g。皮肤给药途径药品开发时 1FTU 的给药面积通常约为 250cm <sup>2</sup> （相当于一只手的表面积） <sup>[13]</sup> 。由于每个皮肤区域的用药量不随皮肤部位而变化，因此 CTCL 不受单次用药剂量和用药部位的影响。根据每天 0.5g 剂量药品给药皮肤表面积为 250 cm <sup>2</sup> ，如下所示得出药品的CTCL 为 35μg/g。最近得出的将对 Co 过敏降至最低的限度显示出与 Ni 类似的限度 31-259ppm <sup>[14]</sup> ， 相同的CTCL 也适用于 Co。</p><dl><dd>0.5 μg/cm <sup>2</sup> /week = 0.07 μg/cm <sup>2</sup> /day</dd></dl><dl><dd>0.07 μg/cm <sup>2</sup> /day x 250 cm <sup>2</sup> = 17.5 μg/day</dd></dl><dl><dd>17.5μg/day/0.5 g/day = 35 µg/g</dd></dl><h2>6 药品的风险评估</h2><p>按照 ICH Q3D 指导原则第 5 章进行皮肤给药药品的风险评估。皮肤给药途径药品的潜在元素杂质来源的识别、计算方法和额外控制的考虑与口服、注射和吸入给药途径药品相同。</p><p>对于 Ni 和 Co，除了考虑相对于 PDE 的药品中元素杂质的水平外，还需要评估相对于表 A.5.1 中CTCL 的药品中该元素杂质的浓度（µg/g）。因此，药品的风险评估应确认总的Ni 和 Co 的水平（µg/day）小于等于 PDE，同时药品中其各自的浓度不超过表 A.5.1 中CTCL。</p><p>如 ICH Q3D 第 5.2 章所述，药品风险评估的总结是通过综合相关药品或组分的特定数据，并结合从药品或制备工艺中获得的信息和认知，用以识别在药品中很可能观察到的元素杂质。</p><p>总结中应考虑实测的或预测的元素杂质水平相对于其 PDE 的显著性，对于 Ni 和 Co，还要考虑相对于 Ni 和 Co 的 CTCL 的显著性。将既定 PDE（以及 Ni 和 Co 的 CTCL）的 30%定义为控制阈值，作为实测元素杂质水平显著性的衡量指标。控制阈值可用于判断是否需要额外的控制。如果药品中所有来源的总的实测或预测的元素杂质水平（µg/day）或者皮肤给药途径浓度（µg/g），始终低于 PDE或 CTCL 的 30%，只要申请人对数据进行了适当的评估并证明已对元素杂质进行了足够的控制，则不再需要额外的控制。</p><p>由于皮肤给药药品的最大总日剂量不总是明确的，因此药品风险评估的前提是对最差情况暴露的合理评估可以作为评估的基础 <sup>[13,15,16]</sup> 。另外，每日用药次数也可能不明确。由于 CTCL 是按照每日用药一次计算得出，因此，可接受的浓度可根据每日最大使用次数以及对各种因素（如药物在皮肤上的停留时间）的评估进行校正。虽然过敏性风险不受使用面积大小的影响，但是风险会随着同一部位多次用药而增加。</p><p>皮肤给药药品与口服、注射或吸入给药药品的不同之处在于其可以从给药部位除去或冲掉。在评估患者可能接触的潜在元素杂质时，评估药品在典型使用条件下的保留时间可能很重要。例如，某些产品（如洗发水）的使用持续时间很短。因此，建议风险评估使用保留因子进行调整（关于保留时间的更多信息，请参阅ICH Q3D 培训模块 1 ； <a href="https://www.ich.org/products/guidelines/quality/article/quality-guidelines.html">url</a> ）。如果以这种方式调整 PDE，则新拟定的水平可以作为可接受水平，并提交给相关监管机构根据具体情况进行考虑。</p><h2>7 皮肤给药途径的 PDE 值</h2><p>表
