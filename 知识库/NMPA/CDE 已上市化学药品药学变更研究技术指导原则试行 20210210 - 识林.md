2021.05</a></li></ul><dl><dd><ul><li>模块二 《已上市化学药品药学变更研究技术指导原则（试行）》解读</li></ul></dd></dl></div><h2>一、概述</h2><p>本指导原则适用于已上市化学 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 和化学制剂的变更研究，供药品上市许可持有人/原料药登记企业（以下简称持有人/登记企业）开展变更研究时参考。</p><p>根据变更对药品安全性、有效性和质量可控性产生影响的风险，本指导原则对所述及的变更分为三类：重大变更、中等变更、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%AE%E5%B0%8F%E5%8F%98%E6%9B%B4">微小变更</a> 。对药品的安全性、有效性或质量可控性产生影响的可能性为重大的变更属于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%87%8D%E5%A4%A7%E5%8F%98%E6%9B%B4">重大变更</a> ；对药品安全性、有效性或质量可控性产生影响的可能性为中等的变更属于中等变更；对药品的安全性、有效性或质量可控性产生影响的可能性为微小的变更属于微小变更。</p><p>药品上市后变更管理属于药品全 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 管理的一部分。。注册阶段的研究工作越系统、深入，生产过程中积累的数据越充分，对上市后的变更研究越有帮助。</p><p>持有人/登记企业可以参考本指导原则对变更进行研究和分类，也可以在对药品及其工艺、质量控制等不断深入理解的基础上，采用 ICH 指导原则（如 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q12_Technical_and_Regulatory_Considerations_for_Pharmaceutical_Product_Lifecycle_Management_201911">ICH Q12</a> 等）中的各种变更管理工具，对变更进行研究和分类，这将更有利于主动对已上市药品进行持续改进和创新。</p><p>本指导原则涵盖的变更情形包括：制剂处方中辅料的变更、原料药和制剂生产工艺变更、生产场地变更、生产批量变更、制剂所用原料药的供应商变更、注册标准变更、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85%E6%9D%90%E6%96%99">包装材料</a> 和容器变更、有效期和贮藏条件变更、增加规格，并列举了每种变更情形下的重大变更、中等变更、微小变更，以及需进行的研究验证工作。本指导原则列出的上述内容为一般性技术要求。持有人/登记企业在进行变更研究时，应结合品种特点和变更情况开展研究，不能仅局限于本指导原则列举的内容。同时，本指导原则不能涵盖已上市化学药品的所有变更情形，对于未列举的变更情形，持有人/登记企业可参考本指导原则、根据变更具体情形开展研究。持有人/登记企业完成研究工作后，按照 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%85%A8%E5%9B%BD%E4%BA%BA%E6%B0%91%E4%BB%A3%E8%A1%A8%E5%A4%A7%E4%BC%9A_%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E8%8D%AF%E5%93%81%E7%AE%A1%E7%90%86%E6%B3%9520190826">《药品管理法》</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=SAMR_%E8%8D%AF%E5%93%81%E6%B3%A8%E5%86%8C%E7%AE%A1%E7%90%86%E5%8A%9E%E6%B3%95_20200122">《药品注册管理办法》</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=SAMR_%E8%8D%AF%E5%93%81%E7%94%9F%E4%BA%A7%E7%9B%91%E7%9D%A3%E7%AE%A1%E7%90%86%E5%8A%9E%E6%B3%95_20200122">《药品生产监督管理办法》</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=NMPA_%E8%8D%AF%E5%93%81%E4%B8%8A%E5%B8%82%E5%90%8E%E5%8F%98%E6%9B%B4%E7%AE%A1%E7%90%86%E5%8A%9E%E6%B3%95%E8%AF%95%E8%A1%8C_20210112">《药品上市后变更管理办法（试行）》</a> 的规定，通过补充申请、备案或者 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B9%B4%E6%8A%A5">年度报告</a> 实施各项变更。</p><p>本指导原则反映了目前对变更分类和变更研究的认知，随着认知的不断深入，本指导原则也将不断完善。</p><h2>二、已上市化学药品药学变更研究工作的基本原则</h2><h2>（一）持有人/登记企业是变更研究的主体</h2><h3>1、设计并开展变更研究工作</h3><p>持有人/登记企业应对药品的研发和生产、质量控制、产品的性质等有着全面和准确的了解。当发生变更时，持有人/登记企业应当清楚变更的原因、变更的情况及对药品的影响，针对变更设计并开展相应的研究工作。</p><p>本指导原则按照重大变更、中等变更、微小变更，分别列出每种变更类别研究验证工作要求。考虑到每种变更类别包括多种具体的变更情形，每种变更情形需要进行的研究工作可能有所不同，所以本指导原则中列出的研究验证工作有时可能不能涵盖某些变更情形所需要的全部研究工作，也有时部分研究工作可能不适用于某些变更情形，所以持有人/登记企业需要根据变更的类别和变更的具体情形、原料药及/或制剂的性质、变更对药品影响程度等综合考虑，设计并开展相关研究工作。当体外研究结果无法准确判定变更对药品安全性、有效性和质量可控性的影响时，需考虑进行体内等效性研究。</p><h3>2、全面评估变更对药品的影响</h3><p>药品处方、生产工艺、批量、质量标准等某一个方面的变更可能对药品安全性、有效性和质量可控性带来全面的影响。</p><p>持有人/登记企业需在变更研究的基础上，对研究结果进行充分分析，全面评估、验证变更事项。</p><h2>（二）关联变更</h2><p>药品某一项变更往往不是独立发生的。例如，批量变更往往同时伴随生产设备及生产工艺的变更，处方变更可能伴随或引发 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%8D%AF%E5%93%81%E6%B3%A8%E5%86%8C%E6%A0%87%E5%87%86">药品注册标准</a> 变更，增加规格可能会调整处方等。本指导原则将一项变更伴随或引发其他变更称之为关联变更。</p><p>对于关联变更，研究工作可按照本指导原则中各项变更研究工作的基本思路分别进行，也可综合考虑各项变更研究工作的要求而一并进行。由于这些变更对药品安全性、有效性和质量可控性影响程度可能不同，即这些变更可能归属于本指导原则中各项变更的不同类别，在按照不同类别变更相应技术要求开展研究工作时，研究工作总体上应按照技术要求较高的变更类别进行，同时建议关注多项关联变更对药品安全性、有效性和质量可控性产生的叠加影响。</p><h2>（三）关于研究用样品的考虑</h2><p>已上市化学药品药学变更发生在药品获准上市后的生产阶段，，如采用中试规模样品，应提供充分的依据。</p><p>在对变更后药品进行研究时，各项研究（如处方研究、工艺研究与验证、相容性研究、密封性研究、溶出度比较研究、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8%E8%B0%B1">杂质谱</a> 比较研究（包括致突变杂质、元素杂质等）、检测方法学验证、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 研究等）应符合相关指导原则要求（包括已在国内实施的ICH指导原则）。在进行药学对比研究时，如果变更前的药品是按照质量和疗效与参比制剂一致的技术要求批准上市的，可以考虑与变更前药品进行药学对比研究。对于需要进行体内等效性研究的，通常建议选择与参比制剂进行对比研究。参比制剂应符合国家药品监督管理局的相关要求。</p><h2>（四）稳定性研究的考虑</h2><p>在按照本指导原则中提出的相关要求进行稳定性研究时，应充分考虑研究工作和研究结果能否充分反映变更后药品的稳定性变化情况，必要时需要增加研究批次或者延长研究时间。而对于部分变更，在充分评估的基础上，可能不需要针对变更进行稳定性研究。</p><p>在按照本指导原则提供稳定性研究资料的同时，应承诺按照稳定性研究方案考察长期稳定性并在年报中进行报告。</p><h2>（五）关注原辅包变更对制剂的影响</h2><p>原辅包的各项变更，如生产工艺的变更、批量的变更、质量标准的变更等，可能对原辅包的质量存在影响，进而对制剂产生影响。鼓励药品上市许可持有人与各物料供应商和/或生产商签订质量协议，及时掌握相关信息，当原辅包发生各项变更时，根据原辅包变更情况对制剂进行必要的研究。</p><h2>三、变更原料药生产工艺</h2><p>本指导原则中涵盖的变更原料药生产工艺主要指化学合成原料药生产工艺或半合成原料药的化学合成及之后生产工艺的变更，一般包括变更合成路线（如延长/缩短合成路线，变更起始原料等）、变更生产条件、变更物料控制/过程控制及其它可能的变更。生产工艺变更可能只涉及上述某一种情况的变更，也可能涉及上述多种情况的变更。对于变更合成路线的，变更后合成路线中起始原料的选择应符合 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q11_Development_and_Manufacture_of_Drug_Substance">ICH Q11</a> 的相关要求。</p><p>在本指导原则中，当对比研究结果符合以下条件时，则可认为杂质谱一致：①新增杂质未高于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%8C%96%E5%AD%A6%E8%8D%AF%E7%89%A9%E6%9D%82%E8%B4%A8%E7%A0%94%E7%A9%B6%E6%8A%80%E6%9C%AF%E6%8C%87%E5%AF%BC%E5%8E%9F%E5%88%99">《化学药物杂质研究的技术指导原则》</a> 及 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A</a> 等规定的鉴定限度；②已有杂质（包含立体异构体）及杂质总量均在质量标准规定的限度内，如标准中无规定，应在原工艺生产的多批产品测定范围内；③新使用的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AE%8B%E7%95%99%E6%BA%B6%E5%89%82">溶剂残留量</a> 符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%8C%96%E5%AD%A6%E8%8D%AF%E7%89%A9%E6%AE%8B%E7%95%99%E6%BA%B6%E5%89%82%E7%A0%94%E7%A9%B6%E6%8A%80%E6%9C%AF%E6%8C%87%E5%AF%BC%E5%8E%9F%E5%88%99">《化学药物有机溶剂残留量研究的技术指导原则》</a> 及 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3C_Impurities">ICH Q3C</a> 等的有关规定；④新的无机杂质符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%8C%96%E5%AD%A6%E8%8D%AF%E7%89%A9%E6%9D%82%E8%B4%A8%E7%A0%94%E7%A9%B6%E6%8A%80%E6%9C%AF%E6%8C%87%E5%AF%BC%E5%8E%9F%E5%88%99">《化学药物杂质研究的技术指导原则》</a> 3～6 个月的稳定性研究资料，并与变更前产品的稳定性情况进行比较，变更后产品的稳定性不低于变更前。</p><p>（6）对于治疗窗窄的药物或水难溶性药物的普通 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%A3%E6%9C%8D%E5%9B%BA%E4%BD%93%E5%88%B6%E5%89%82">口服固体制剂</a> 和缓控释制剂，此类变更对药品安全性、有效性和质量可控性均可能产生较显著的影响，一般需考虑进行生物等效性研究。其他制剂，应结合工艺的复杂程度、药物特点以及变更情况等方面综合考虑是否需要进行生物等效性试验。如申请免除生物等效性研究，需进行充分的研究和分析。</p><h2>六、变更制剂所用原料药的供应商</h2><p>变更制剂所用原料药的供应商不应对药品安全性、有效性和质量可控性产生负面影响。</p><p>减少原料药供应商或变更原料药供应商名称（主体不变），按微小变更管理，不需研究验证。</p><p>变更后的原料药为已获得批准的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> ，一般按照中等变更管理。变更后的原料药如尚未获得批准，按照重大变更管理。均需要进行以下研究验证工作：</p><dl><dd>1、说明变更的具体情况和原因。</dd></dl><dl><dd>2、提供变更前后原料药的质量标准。</dd></dl><dl><dd>3、对变更前后的原料药进行全面的质量对比研究，与制剂质量相关的关键理化性质（如晶型、粒度等）应保持一致，变更后原料药的杂质控制应符合相关指导原则的要求。</dd></dl><dl><dd>4、对采用变更前和变更后原料药制备的制剂进行质量对比研究，变更前后样品的溶出曲线、关键理化性质应保持一致，杂质控制应符合相关指导原则的要求，制剂质量应保持一致。</dd></dl><dl><dd>7、如变更前后制剂的溶出曲线、关键理化性质等存在差异，一般需考虑进行生物等效性研究，如申请免除 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E7%AD%89%E6%95%88%E6%80%A7">生物等效性</a> 研究，需进行充分的研究和分析。此种情形应按照重大变更管理。</dd></dl><h2>七、变更生产批量</h2><p>生产批量变更是指在原批准批量（如关键临床试验批、BE 批等）基础上扩大或缩小生产批量。生产批量缩小至相关指导原则或技术要求。</p><h2>（一）原料药批量的变更</h2><h3>1、微小变更</h3><h4>1.1、变更情况</h4><p>原料药的生产批量变更在原批准批量的10倍以内（包括10倍）。</p><h4>1.2、研究验证工作</h4><p>（1）说明批量变更的具体情况和原因，对变更前后的生产工艺及生产设备的设计及工作原理进行对比分析，对变更后的批量进行研究和/或验证。对于无菌原料药，还需进行无菌/灭菌工艺验证。</p><p>（2）提供变更后一批样品的批生产记录。</p><p>（3）对变更前后的样品进行质量对比研究，变更前后样品的杂质谱、关键理化性质等应保持一致，应符合相关指导原则的要求。</p><p>（4）对变更后 1-3 批样品进行检验，应符合质量标准的规定。</p><p>（5）对变更后首批样品进行长期稳定性考察，并在年报中报告该批样品的长期稳定性试验数据。</p><h3>2、中等变更</h3><h4>2.1、变更情况</h4><p>原料药的生产批量变更在原批准批量的10倍以上。</p><h4>2.2、研究验证工作</h4><p>（1）说明批量变更的具体情况和原因，对变更前后的生产工艺及 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E4%BA%A7%E8%AE%BE%E5%A4%87">生产设备</a> 的设计及工作原理进行对比分析，对变更后的批量进行研究和/或验证。对于无菌原料药，还需进行无菌/灭菌工艺验证。</p><p>（2）提供变更后一批样品的批生产记录。</p><p>（3）对变更前后的样品进行质量对比研究，变更前后样品的杂质谱、关键理化性质等应保持一致，并符合相关指导原则的要求。</p><p>（4）对变更后三批样品进行检验，应符合质量标准的规定。</p><p>（5）对变更后一批样品进行加速及长期稳定性考察，申请时提供不少于3个月的稳定性研究资料，并与变更前产品的稳定性情况进行比较，变更后样品的稳定性不低于变更前。</p><h2>（二）制剂批量的变更</h2><h3>1、微小变更</h3><h4>1.1、变更情况</h4><p>此类变更包括但不限于以下情形：</p><p>（1）普通口服固体制剂和非无菌半固体制剂的生产批量变更在关键临床试验批或BE批批量的10倍以内（包括10倍）。</p><p>（2）非无菌液体制剂的生产批量变更。</p><p>（3）采用终端灭菌工艺的制剂，微生物负荷水平不变的前提下，溶液储存时间的增加不超过原批准时限的 50%。</p><h4>1.2、研究验证工作</h4><p>（1）说明批量变更的具体情况和变更的原因，对变更前后生产工艺及生产设备的设计和工作原理进行对比分析，对变更后的批量进行研究和/或验证。</p><p>（2）提供变更后一批样品的批生产记录。</p><p>（3）进行变更前后样品进行对比研究，变更前后样品的溶出曲线、杂质谱、关键理化性质等应保持一致，并符合相关指导原则的要求。</p><p>（4）对变更后 1-3 批样品进行检验，应符合质量标准的规定。</p><p>（5）对变更后首批样品进行长期稳定性考察，并在年报中报告该批样品的长期稳定性试验数据。</p><h3>2、中等变更</h3><h4>1.1、变更情况</h4><p>此类变更包括但不限于以下情形：</p><p>（1）普通口服固体制剂和非无菌半固体制剂的生产批量变更在关键临床试验批或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E7%AD%89%E6%95%88%E6%80%A7">BE</a> 批批量的10倍以上。</p><p>（2）采用终端灭菌工艺的制剂，微生物负荷水平不变的前提下，溶液储存时间的增加超过原批准时限的50%。</p><p>（3）采用无菌生产工艺的无菌制剂的批量变更，同时与无菌保障水平相关的步骤的生产时间（包括配液、药液存放、过滤、灌装等）增加。</p><h4>1.2、研究验证工作</h4><p>（1）说明批量变更的具体情况和变更的原因，对变更前后的生产工艺及生产设备的设计及工作原理进行对比分析，对变更后的批量进行研究和/或验证。对于无菌制剂，必要时还需进行无菌/灭菌工艺验证。</p><p>（2）提供变更后一批样品的批生产记录。</p><p>（3）对变更前后的样品进行质量对比研究，变更前后样品的溶出曲线、杂质谱、关键理化性质等应保持一致，并符合相关指导原则的要求。</p><p>（4）对变更后三批样品进行检验，应符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 的规定。</p><p>（5）对变更后一批样的稳定性研究资料，并与变更前产品的稳定性情况进行比较，变更后样品的稳定性不低于变更前。</p><h3>3、重大变更</h3><h4>3.1、变更情况</h4><p>此类变更包括但不限于以下情形：</p><p>特殊剂型制剂（如复杂工艺的缓控释制剂及肠溶制剂、透皮给药制剂、脂质体、长效制剂等）的生产批量变更。</p><h4>3.2、研究验证工作</h4><p>。</p><p>（2）提供变更后一批样品的批生产记录。</p><p>（3）对变更前后的样品进行质量对比研究，重点比较变更前后样品的溶出曲线、杂质谱、关键理化性质等，应符合相关指导原则的要求。</p><p>（4）对变更后连续生产的三批样品进行检验，应符合质量标准的规定。</p><p>（5）对变更后三批样品进行加速及长期稳定性考察，申请时提供 3-6 个月的稳定性研究资料，并与变更前产品的稳定性情况进行比较，变更后产品的稳定性不低于变更前。</p><p>（6）根据变更情况，综合评估是否需要进行生物等效性研究，如申请免除生物等效性研究，需结合工艺的复杂程度、药物特点、批量变更情况、生产设备情况等方面综合考虑，提供充分的依据。</p><h2>八、变更注册标准</h2><p>变更中的检验项目、检验方法、限度等。药品注册标准变更可能只涉及上述某一种情况的变更，也可能涉及上述多种情况的变更。</p><p>一般而言，变更原料药和制剂注册标准不应引起药品质量控制水平的降低。变更药品注册标准尚需考虑是否会影响到药品的有效期，如对标准进行了提高（例如收紧限度、增加检验项目、优化检测方法等），需考察在原定的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%89%E6%95%88%E6%9C%9F">有效期</a> 内，药品是否符合修订后质量标准的要求。</p><p>对于随国家药品标准变更而引起的注册标准变更，可参照相关公告要求执行（如，国家药品监督管理局 <a href="https://lib.shilinx.com/wiki/index.php?title=NMPA_%E5%85%B3%E4%BA%8E%E5%AE%9E%E6%96%BD2020%E5%B9%B4%E7%89%88%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E8%8D%AF%E5%85%B8%E6%9C%89%E5%85%B3%E4%BA%8B%E5%AE%9C%E7%9A%84%E5%85%AC%E5%91%8A_2020%E5%B9%B4%E7%AC%AC80%E5%8F%B7_20200703">关于实施《中华人民共和国药典》有关事宜的公告</a> ）。</p><h2>（一）中等变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）新增检验项目。</p><p>新增检验项目应可以更有效地控制产品质量，新增检测项目的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%96%B9%E6%B3%95%E9%AA%8C%E8%AF%81">方法学验证</a> 和拟定的控制限度，均应符合相关指导原则的要求。</p><p>该变更不包括因安全性或质量可控性原因导致的增加检验项目。另外，因生产工艺改变导致药学方面特性发生变化，而在标准中增加检验项目也不属于此类变更范畴。例如，原料药改用微粉化处理后，在标准中增加粒度分布检查。</p><p>（2）在原标准规定范围内收紧限度</p><p>这类变更是指在原标准规定范围内收紧控制限度。例如，原料药或制剂经过多批次生产验证，水分、有关物质等指标可以达到更优的控制水平，进而收紧控制限度。</p><p>由于药品的生产工艺、处方等方面的重大变更而引起限度范围缩小不属于此类变更范畴。例如，原料药生产工艺中增加微粉化处理而引起粒度变小的变更。</p><p>（3）注册标准中文字描述的变更，此类变更不应涉及检验方法、限度等的变更。</p><h3>2、研究验证工作</h3><p>（1）说明具体变更情况和原因，提供变更后的质量标准。</p><p>（2）对质量标准变更合理性进行研究。</p><p>若涉及限度修订，需对一定批次样品（建议含近效期样品）批分析结果进行汇总，为限度修订提供依据。另外，需考察在原定的有效期内，药品是否符合修订后质量标准的要求。</p><p>如涉及增加检验项目，需对检验方法进行方法学研究（包括方法的选择、验证）、提供限度拟定依据。需对一定批次样品（建议含近效期样品）批分析结果进行汇总，以考察在原定的有效期内，药品是否符合修订后质量标准的要求。</p><p>（3）按变更后的质量标准对三批样品进行检验，应符合规定。</p><h2>（二）重大变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）变更检验方法。</p><p>（2）放宽控制限度。</p><p>（3）删除注册标准中的任何项目。</p><h3>2、研究验证工作</h3><p>（1）说明具体变更情况和原因，提供变更后的质量标准。</p><p>（2）对质量标准变更合理性进行研究。</p><p>如涉及检验方法改变，需对新方法进行方法学研究验证并应与变更前方法进行比较，确保方法变更不引起药品质量控制水平的降低。另外，需对一定批次样品（建议含近效期样品）批分析结果进行汇总，以考察在原定的有效期内，药品是否符合修订后质量标准的要求。</p><p>如涉及删除标准中任何内容，需结合药品生产过程控制、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%8D%AF%E5%93%81%E7%A0%94%E5%8F%91">药品研发</a> 过程及药品性质等综合分析和证明该项变更不会引起药品质量控制水平的降低。</p><p>如涉及放宽控制限度，需进行详实的研究，必要时需要有关安全性和/或有效性试验资料或文献资料的支持。限度变更还需基于一定批次样品的检测数据并符合相关的技术指导原则。</p><p>（3）按变更后的质量标准对三批样品进行检验，应符合规定。</p><h2>九、变更包装材料和容器</h2><p>包装材料和容器是药品的组成部分，本指导原则涉及的包装材料和容器主要指直接接触药品的包装。包装材料和容器的变更可能对药品的理化性质、杂质谱、含量、稳定性等产生影响，其风险取决于制剂的给药途径、包装材料和容器的性能以及包装和制剂的相容性等。总体上，变更药品的包装材料和容器应能对保证药品的质量和稳定性起到有益的作用，不得对药品的保护性、功能性、安全性和质量产生不良影响。</p><p>本指导原则涵盖的药品的包装材料和容器的变更包括改变、增加或去除的情形。</p><h2>（一）微小变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）变更原料药及单剂量包装制剂的包装装量，如，每袋的克数，每板胶囊的粒数，每盒注射剂的支数，等。</p><p>（2）本指导原则中未规定的原料药、非无菌固体制剂包装材料和容器的材质和/或类型的变更。变更后的包装材料和容器已在具有相同 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 的已上市药品中使用，并且具有相同或更好适用性能。</p><p>（3）</p><h3>2、研究验证工作：</h3><p>（1）说明包装材料和容器变更的原因，并详细描述变更后的包装材料和容器情况。列出变更后包装材料和容器的质量标准。</p><p>（2）变更前后包装材料和容器相关特性的对比研究。</p><p>（。</p><p>（4）对变更后首批样品进行长期稳定性考察，并在年报中报告该批样品的长期稳定性试验数据。</p><h2>（二）中等变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）变更多剂量包装制剂的包装装量，如，每瓶的片数，每支的克数，每瓶的毫升数，等。</p><p>（2）变更液体/半固体制剂（吸入制剂、注射剂、眼用制剂除外）、无菌和/或液体原料药的包装材料和容器的材质和/或类型。如，口服液体药用聚丙烯瓶变更为口服液体药用聚酯瓶，等。</p><p>（3）</p><p>（4）变更注射剂的包装材料和容器的供应商、尺寸和/或形状。</p><h3>2、研究验证工作</h3><p>（1）说明包装材料和容器变更的原因，并详细描述变更后的包装材料和容器情况。列出变更后包装材料和容器的质量标准。</p><p>（2）变更前后包装材料和容器相关特性的对比研究，进行包材的等同性/可替代性研究。</p><p>（3）酌情进行包材 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%9B%B8%E5%AE%B9%E6%80%A7">相容性</a> 研究。对于密封件的变更还应开展包装密封性研究。</p><p>（4）进行包装工艺验证。对于无菌产品，必要时进行无菌/灭菌工艺验证。</p><p>（5）对变更后 1-3 批样品进行检验，应符合质量标准的规定。</p><p>（6）对变更后一批样品进行加速及长期稳定性考察，申请时提供不少于 3 个月的稳定性研究资料，并与变更前产品的稳定性情况进行比较，变更后产品的稳定性不低于变更前。根据变更情况，酌情进行使用中稳定性研究。</p><h2>（三）重大变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）变更吸入制剂、注射剂、眼用制剂的包装材料和容器的材质和/或类型。如，三层共挤输液袋变更为五层共挤输液袋，聚丙烯输液瓶变更为直立式聚丙烯输液袋，钠钙玻璃输液瓶变更为五层共挤输液袋。</p><p>（2）变更吸入制剂定量给药装置的供应商、尺寸和/或形状。</p><p>（3）去除对药品提供额外保护的次级包装（如，高阻隔性外袋）。</p><p>（4）变更为全新材料、全新结构、风险度提高的新用途的包装材料和容器。</p><p>（5）变更纳入登记管理的包装材料和容器，变更后的包装材料和容器尚未登记或登记状态为 I。</p><h3>2、研究验证工作</h3><p>（1）说明包装材料和容器变更的原因，并详细描述变更后的包装材料和容器情况。列出变更后包装材料和容器的质量标准。</p><p>（2）变更前后包装材料和容器相关特性的对比研究，进行包材的等同性/可替代性研究。</p><p>（3）酌情进行包材相容性研究。对于密封件的变更还应开展包装密封性研究。对于定量给药装置发生变更，需根据给药装置的特点进行相应的研究，证明变更后给药剂量准确性不低于变更前。</p><p>（4）进行包装工艺验证。对于无菌制剂，必要时进行无菌/灭菌工艺验证。</p><p>（5）对变更后连续生产的三批样品进行检验，应符合质量标准的规定。</p><p>（6）对变更后三批样品进行加速及长期稳定性研究，申请时提供 3～6 个月的稳定性研究资料，并与变更前产品的稳定性情况进行比较，变更后产品的稳定性不低于变更前。根据变更情况，酌情进行使用中稳定性研究。</p><h2>十、变更有效期和贮藏条件</h2><p>药品有效期和贮藏条件变更可能包含以下几种情况：①延长有效期；②缩短有效期；③严格贮藏条件；④放宽贮藏条件。变更可能只涉及上述某一种情况的变更，也可能涉及上述多种情况的变更。</p><h2>（一）中等变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）延长药品有效期</p><p>有效期变更主要依据长期稳定性试验结果。延长药品有效期不应超过长期稳定性试验已完成的时间。</p><p>（2）缩短药品有效期</p><p>此类变更不包括因药品的生产或稳定性出现问题而要求缩短药品有效期。</p><h3>2、研究验证工作</h3><p>（1）说明变更后的有效期及变更原因。</p><p>（2）提供三批样品的长期稳定性考察数据。</p><h2>（二）重大变更</h2><h3>1、变更情况</h3><p>此类变更包括但不限于以下情形：</p><p>（1）变更药品贮藏条件。</p><p>（2）由于药品的生产工艺、处方、质量标准、直接接触药品的包装材料和容器等方面的变更导致的有效期变更。</p><h3>2、研究验证工作</h3><p>（1）说明变更后的贮藏条件和/或有效期，研究说明变更的原因。</p><p>（2）按照确定的稳定性试验方案，对三批样品进行稳定性考察，提供 3～6 个月的稳定性研究资料。</p><h2>十一、增加规格</h2><p>本指导原则涵盖的增加规格，是指片剂/ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%83%B6%E5%9B%8A">胶囊剂</a>
