10日</td><td>Q7</td></tr></tbody></table></div><p>ICH 指导委员会在2000年11月10日完成了ICH进程的第四阶段，该指南推荐给ICH三方的监管机构采用。</p><h2>1 总则</h2><h2>1.1 目的</h2><p>本指南旨在适宜的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E4%BD%93%E7%B3%BB">质量管理体系</a> 下为原料药(英文简称 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">APIs</a> ）的生产提供 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">药品生产质量管理规范</a> (GMP）指导⁠⁠⁠⁠⁠⁠。本指南也有助于确保原料药符合其预定的质量与纯度要求。</p><p>本指南中的“生产”指包括原料药的物料接收、生产过程、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85">包装</a> 、重新包装、贴签、重新贴签、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%8E%A7%E5%88%B6">质量控制</a> 、放行、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%AE%E5%AD%98">贮存</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%91%E8%BF%90">发运</a> 在内的所有操作及相关控制。本指南中的“应当”,指期望采用的建议，除非其不适用，或者使用经证明具有至少同等质量保证水平的方法。本指南中的“现行药品生产质量管理规范(cGMP)”等同于“ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">药品生产质量管理规范</a> (GMP)”。</p><p>本指南不涉及人员安全及环境保护的要求，这些是生产企业的固有责任，具体要求由国家相关法律规定。</p><p>本指南中如有涉及注册/申报或药典修订的，应按照监管机构制定的关于注册/申报的特定要求执行，并切实履行申报文件中的承诺。</p><h2>1.2 法规适用性</h2><p>各国对 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的法律界定不尽相同。当某种物料被界定为原料药后，其生产或用于药品生产时则应遵循本指南。</p><h2>1.3 范围</h2><p>本指南适用于人用药中非无菌原料药生产及 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%97%A0%E8%8F%8C%E5%8E%9F%E6%96%99%E8%8D%AF">无菌原料药</a> 生产中非无菌生产工序的操作。无菌原料药的灭菌和无菌生产过程应参照GMP 中有关无菌药品的要求。</p><p>本指南适用于通过化学合成、提取、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%86%E8%83%9E%E5%9F%B9%E5%85%BB">细胞培养</a> /发酵生产，或从天然资源中获取的原料药，也包括采用上述多种方法组合生产的原料药。第18章规定了采用细胞培养/ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%91%E9%85%B5">发酵</a> 工艺生产原料药的特定要求。</p><p>本指南不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%96%AB%E8%8B%97">疫苗</a> 、全细胞、全血和血浆、血液和血浆制品(血浆提取物)、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9F%BA%E5%9B%A0%E6%B2%BB%E7%96%97">基因治疗</a> 的原料药。但本指南涵盖由血液或血浆作为原材料生产的原料药。此外，细胞基质(哺乳动物、植物、昆虫和微生物的细胞、组织或动物来源包括转基因动物)和前期的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA">工艺</a> 步骤可能需要遵守GMP规范，但不包含在本指南之内。本指南同样不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%BB%E7%94%A8%E6%B0%94%E4%BD%93">医用气体</a> 、待分装制剂和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%94%BE%E5%B0%84%E6%80%A7%E8%8D%AF%E5%93%81">放射性药品</a> 特定的生产及控制。</p><p>第19章仅适用于为生产临床 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AF%95%E9%AA%8C%E7%94%A8%E8%8D%AF%E5%93%81">试验用药品</a> (研究用药品)中所用原料药的生产。</p><p>“原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%B7%E5%A7%8B%E7%89%A9%E6%96%99">起始物料</a> ”是指原料药生产所用的、且以主要结构单元的形式被结合进原料药结构中的原料、中间体或原料药。原料药的起始物料可以是市售商品，也可以通过合同或商业协议从一个或多个供应商处购得，或者企业自行生产。原料药的起始物料一般有其特定的化学性质和结构。</p><p>生产企业应指定 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 生产的起点，并以文件形式说明理由。对于合成工艺，以“原料药的起始物料”引入工艺为起点。对于其他工艺(例如发酵、提取、纯化等)起点的确定，应当具体问题具体分析。表1给出了确定原料药生产起点的指导原则，通常以起始物料引入工艺为起点。</p><p>从起点开始，中间体和/或原料药生产的各步操作，均应当符合本指南的相关要求，包括对原料药质量有影响的关键工艺步骤的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AA%8C%E8%AF%81">验证</a> 。但企业选择某一工艺步骤进行验证，并不意味着该步骤一定是关键工艺步骤。</p><p>本指南通常适用于表1中灰色区域的步骤，但这并不意味着灰色区域的步骤应当完全遵循本指南。原料药生产过程中，随着工艺的进行，从原料药早期的步骤到最后的步骤、精制和包装， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">GMP</a> 要求应当越来越严格。原料药的物理加工，如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%B6%E7%B2%92">制粒</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A1%A3">包衣</a> 、粒径的物理控制(如研磨、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%AE%E7%B2%89%E5%8C%96">微粉化</a>)均应当至少符合本指南的要求。</p><p>本指南不适用于“原料药的起始物料”引入前的步骤。</p><div><table><caption>表1 width="300" height="72"></center><h2>2 质量管理</h2><h2>2.1 原则</h2><p>2.10 质量应当是所有参与生产活动人员的职责。</p><p>2.11 原料药生产企业应当建立 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E4%BF%9D%E8%AF%81">质量保证</a> 系统，同时建立完整的文件体系，并配备足够的、符合要求的生产和质量管理人员，以保证系统有效运行。</p><p>2.12 质量管理体系应包含组织机构、程序、流程和资源，以及确保原料药符合预定的质量和纯度要求所需的活动。所有与质量相关的活动均应当有文件规定并记录。</p><p>2.13 应当设立独立于生产部门的质量管理部门，履行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E4%BF%9D%E8%AF%81">质量保证</a> (QA）和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%8E%A7%E5%88%B6">质量控制</a> (QC）的职责。根据企业的规模和结构，质量管理部门可以分别设立质量保证部门和质量控制部门，也可以由一个人或一个小组组成。</p><p>2.14 应当指定授权放行中间体和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的人员。</p><p>2.15 所有与质量有关的活动均应当及时记录。</p><p>2.16 任何 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a> 均应当记录并说明。重大偏差应当进行调查，并有调查报告记录偏差的调查和结论。</p><p>2.17 在质量管理部门未完成质量评价之前，任何物料不得放行或使用，除非有合理的程序规定允许使用(例如，10.20节所述 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%85%E9%AA%8C">待验</a> 状态下的放行，或是处于评估过程中的原料或中间体的使用)。</p><p>2.18 应当建立相关规程，确保将药品监管部门检查、严重GMP <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BC%BA%E9%99%B7">缺陷</a> 、产品缺陷及相关活动(如质量 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%95%E8%AF%89">投诉</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AC%E5%9B%9E">召回</a> 、监管活动等)等信息及时通知管理层。</p><h2>2.2 质量管理部门的职责</h2><p>2.20 质量管理部门应当参与所有与质量相关的事务。</p><p>2.21 质量管理部门应当审核和批准所有与质量相关的文件。</p><p>2.22 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 不得将主要职责委托给其它部门。应当书面描述质量管理部门的职责，包括但不限于以下内容：</p><dl><dd>1 所有原料药的放行或拒绝放行，包括本企业控制范围之外使用的中间体。</dd></dl><dl><dd>2 建立放行或拒绝放行原料、中间体、包装和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">标签</a> 材料的系统。</dd></dl><dl><dd>3 原料药出厂放行前应当完成关键步骤 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E7%94%9F%E4%BA%A7%E8%AE%B0%E5%BD%95">批生产记录</a> 和批检验记录的审核。</dd></dl><dl><dd>4 确保已对重大 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a> 进行调查并得到处理。</dd></dl><dl><dd>5 批准所有的质量标准和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E8%A7%84%E7%A8%8B">工艺规程</a> 。</dd></dl><dl><dd>6 批准所有影响中间体或原料药质量的规程。</dd></dl><dl><dd>7 确保内部审计(自检)的执行。</dd></dl><dl><dd>8 批准中间体和原料药的受托生产商。</dd></dl><dl><dd>9 批准对中间体或原料药质量有潜在影响的变更。</dd></dl><dl><dd>10 审核和批准 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AA%8C%E8%AF%81%E6%96%B9%E6%A1%88">验证方案</a> 和报告。</dd></dl><dl><dd>11 确保调查和解决与质量相关的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%95%E8%AF%89">投诉</a> 。</dd></dl><dl><dd>12 确保使用有效的系统维护和校准关键设备。</dd></dl><dl><dd>13 确保物料已进行了适当的检验并报告了结果。</dd></dl><dl><dd>14 确保具有支持 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 和/或中间体(如适用)的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 或有效期及储存条件的稳定性数据。</dd></dl><dl><dd>15 开展产品 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E5%9B%9E%E9%A1%BE">质量回顾</a> (详见2.5节)</dd></dl><h2>2.3 生产部门的职责</h2><dl><dd>应当书面描述生产部门的职责，包括但不限于以下内容：</dd></dl><dl><dd>1 根据书面规程起草、审核、批准和分发用于中间体或原料药生产的指导文件。</dd></dl><dl><dd>2 按照已批准的指导文件生产原料药、中间体(如适用)。</dd></dl><dl><dd>3 审核所有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E7%94%9F%E4%BA%A7%E8%AE%B0%E5%BD%95">批生产记录</a> ，确保已全部完成并签名。</dd></dl><dl><dd>4 确保所有生产 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a> 均已报告、评估，所有重大偏差已经过调查并有结论。</dd></dl><dl><dd>5 确保生产设施 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B8%85%E6%B4%81">清洁</a> ，必要时 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B6%88%E6%AF%92">消毒</a> 。</dd></dl><dl><dd>6 确保进行必要的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%A1%E5%87%86">校准</a> 并记录。</dd></dl><dl><dd>7 确保厂房和设备的维护并记录。</dd></dl><dl><dd>8 确保 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AA%8C%E8%AF%81%E6%96%B9%E6%A1%88">验证方案</a> 和报告经过审核、批准。</dd></dl><dl><dd>9 评估产品、工艺及设备的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%98%E6%9B%B4">变更</a> 。</dd></dl><dl><dd>10 确保新建和改造后的(如适用)厂房和设备经过确认。</dd></dl><h2>2.4 内部审计(自检)</h2><p>2.40 应当根据已批准的计划定期进行内部审计，以评估是否符合本指南要求。</p><p>2.41 审计结果及整改措施应当形成文件并向企业高层管理人员报告。获得批准后的整改措施应当及时、有效地完成。</p><h2>2.5 产品质量回顾</h2><p>2.50 应当对原料药定期进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E5%9B%9E%E9%A1%BE">质量回顾</a> ，以确认工艺稳定可靠。通常，应当每年进行一次回顾并形成文件，至少应当包括：</p><dl><dd>- 回顾关键中间控制点及原料药关键检验结果；</dd></dl><dl><dd>- 回顾所有不符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 的批次；</dd></dl><dl><dd>- 回顾所有重大偏差或不符合情形及其相关调查；</dd></dl><dl><dd>- 回顾所有生产工艺或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A3%80%E9%AA%8C">检验</a> 方法等的变更；</dd></dl><dl><dd>- 回顾 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 考察的结果；</dd></dl><dl><dd>- 回顾所有质量相关的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%80%80%E8%B4%A7">退货</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%95%E8%AF%89">投诉</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AC%E5%9B%9E">召回</a> ；</dd></dl><dl><dd>- 回顾纠正措施的充分性。</dd></dl><p>2.51 应当评估回顾分析的结果，提出是否需要采取整改措施或进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%86%8D%E9%AA%8C%E8%AF%81">再验证</a> 的评估意见及理由，经记录并批准后应当及时、有效地完成整改。</p><h2>3 人员</h2><h2>3.1 人员资质</h2><p>3.10 应当配备足够数量、具备一定教育背景、培训和/或实践经验的人员执行和监督中间体和原料药的生产。</p><p>3.11 应当书面明确规定所有从事中间体和原料药生产的人员职责。</p><p>3.12 应当由有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%84%E8%B4%A8">资质</a> 的人员定期对员工进行培训，内容至少包括员工所从事的特定操作和与其岗位有关的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">GMP</a> 要求。培训记录应当保存，并定期评估培训效果。</p><h2>3.2 人员卫生</h2><p>3.20 员工应当养成良好的卫生和健康习惯。</p><p>3.21 员工应当穿着适合其从事生产操作的洁净服装，并适时更换。必要时应当佩戴劳保用品，如头、脸、手和臂等的遮护用品，避免 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 和中间体受到污染。</p><p>3.22 员工应当避免直接接触中间体或原料药。</p><p>3.23 抽烟、进食、饮水、咀嚼和食品存放等应当严格限制在与生产区域隔开的指定区域。</p><p>3.24 患有传染病或体表有伤口的人员，不应当从事危及原料药质量的工作。任何时候(经体检或日常医学观察),任何患有危及到原料药质量的疾病或体表有伤口的人员，都不应当参与可能危及原料药质量的活动，直到恢复健康或者由有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%84%E8%B4%A8">资质</a> 的医学人员确认其不会危及到原料药的安全和质量时为止。</p><h2>3.3 顾问</h2><p>3.30 为中间体或原料药生产和控制提供咨询服务的顾问，应当具备足够的教育背景、培训经历和实践经验，或同时满足上述三项条件中的两项，能在受聘领域提供咨询。</p><p>3.31 顾问的姓名、地址、资质和提供的服务类型都应当记录。</p><h2>4 厂房和设施</h2><h2>4.1 设计和建造</h2><p>4.10 用于生产中间体和原料药的厂房和设施，应当基于生产类型和工序进行选址，其设计、建造应当便于清洁、维护和操作。设施的设计应当尽可能减少潜在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B1%A1%E6%9F%93">污染</a> 。中间体或原料药 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 中有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%AE%E7%94%9F%E7%89%A9%E6%A3%80%E9%AA%8C">微生物检验</a> 项目的，设施的设计应当特别注意防止有害微生物的污染。</p><p>4.11 厂房和设施应当有足够空间，以便设备和物料有序放置，防止 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B7%B7%E6%B7%86">混淆</a> 和污染。</p><p>4.12 设备本身能有效保护物料时(如密闭系统或者限制系统),可安置在室外。</p><p>4.13 厂房和设施的物流和人流设计应当能防止混淆和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B1%A1%E6%9F%93">污染</a> 。</p><p>4.14 以下活动应当有指定区域或其它控制系统：</p><dl><dd>- 来料的接收、鉴别、取样和待验，等待放行或拒绝放行；</dd></dl><dl><dd>- 中间体和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 放行或拒绝放行前的待验；</dd></dl><dl><dd>- 中间体和原料药的取样；</dd></dl><dl><dd>- 不合格物料处理前(如退货、返工或销毁)的贮存；</dd></dl><dl><dd>- 已放行物料的贮存；</dd></dl><dl><dd>- 生产操作；</dd></dl><dl><dd>- 包装和贴标签操作；</dd></dl><dl><dd>- 实验室操作。</dd></dl><p>4.15 应当为员工提供足够的清洁盥洗和如厕设施，盥洗设施应配备冷热水(视情况而定)、肥皂或洗涤剂、干手机或一次性手巾。盥洗设施应和生产区域分开且便于到达 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E4%BA%A7%E5%8C%BA">生产区</a> 。应当根据情况提供足够的淋浴和(或)更衣设施。</p><p>4.16 实验区域/操作通常应当与生产区域分开。当生产操作不影响检验结果的准确性，且检验操作对生产过程、中间体或者 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 也无不利影响时，有些实验区域，尤其是中间控制实验室可设在生产区内。</p><h2>4.2 公用设施</h2><p>4.20 应当对所有可能影响产品质量的公用设施(如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%92%B8%E6%B1%BD">蒸汽</a> 、气体、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%8B%E7%BC%A9%E7%A9%BA%E6%B0%94">压缩空气</a> ，加热、通风和空调系统)进行确认，并适当监测，超出限度时应当采取相应措施。应当保存这些公用设施的图纸。</p><p>4.21 应当根据情况提供足够的通风、空气过滤和排气系统。这些系统在设计和建造时应当考虑最大限度地降低 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B1%A1%E6%9F%93">污染</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 的风险，并配备设备来控制与生产阶段相适宜的空气压力、微生物(如适用)、粉尘、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B9%BF%E5%BA%A6">湿度</a> 和温度。应当特别关注原料药生产的暴露工序区域。</p><p>4.22 如果空气以回风形式进入生产区域，应当采取适宜的措施控制污染和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 的风险。</p><p>4.23 应当对固定管道适当标识，包括采用管道标示、文件记录、计算机控制系统或其他替代方法。管道的安装应当避免中间体或原料药受到污染。</p><p>4.24 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8E%92%E6%B0%B4%E8%AE%BE%E6%96%BD">排水设施</a> 应当大小适宜，应当视情况安装空气隔断或适宜的防止 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%80%92%E7%81%8C">倒灌</a> 的装置。</p><h2>4.3 水</h2><p>4.30 应当证明原料药的生产用水符合其预定用途。</p><p>4.31 除非有其他理由，工艺用水应当至少符合世界卫生组织(WHO）饮用水 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 。</p><p>4.32 如果饮用水不足以确保原料药的质量，需要提高其化学和/或微生物标准，应当制定与物理/化学特性、微生物总数、控制菌和/或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%86%85%E6%AF%92%E7%B4%A0">内毒素</a> 相关的质量标准。</p><p>4.33 生产商对工艺用水进行处理使其达到规定质量时，应当对处理工艺进行验证，并设置适当的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BA%A0%E5%81%8F%E9%99%90%E5%BA%A6">纠偏限度</a> 进行监测。</p><p>4.34 非无菌原料药生产商生产的原料药可进一步用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%97%A0%E8%8F%8C">无菌</a> 药品生产时，最终分离和精制阶段的用水应当监控微生物总数、控制菌和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%86%85%E6%AF%92%E7%B4%A0">内毒素</a> 。</p><h2>4.4 控制</h2><p>4.40 生产 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AB%98%E8%87%B4%E6%95%8F%E6%80%A7">高致敏性</a> 药品，如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%9D%92%E9%9C%89%E7%B4%A0%E7%B1%BB">青霉素类</a> 或头孢类药物等，应采用专用的生产区域，包括生产设施、空气净化系统和/或工艺设备。</p><p>4.41 生产具有感染性或高活性或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AB%98%E6%AF%92%E6%80%A7">高毒性</a> 物质(如某些激素类物质或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%86%E8%83%9E%E6%AF%92%E6%80%A7">细胞毒性</a> 抗肿瘤物质)应当使用专用生产区域，除非制定并应用经 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AA%8C%E8%AF%81">验证</a> 的灭活和/或清洁程序。</p><p>4.42 应当建立并执行适宜的措施，防止由于人员、物料等在各专用区域间流动而造成的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 。</p><p>4.43 不应当使用生产 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的厂房和/或设备来开展高毒性的非药用物质(如除草剂、杀虫剂)的任何生产活动(包括称重、粉碎或包装)。这些高毒性非药用物质的处理和贮存应与原料药分开。</p><h2>4.5 照明</h2><p>4.50 所有区域都应当有充足的照明，便于清洁、维护和正确操作。</p><h2>4.6 污水和污物</h2><p>4.60 厂房及其邻近区域内产生或排出的污水、污物和其它废弃物(如生产过程产生的废渣、废液和废气)应当安全、及时和卫生地处置。废弃物使用的容器和/或管道应当有明确标识。</p><h2>4.7 卫生和维护</h2><p>4.70 用于生产原料药和中间体的厂房应当适当维护、维修并保持清洁。</p><p>4.71 应当建立书面规程规定卫生工作职责、厂房和设施清洁计划、方法、清洁用设备和物料。</p><p>4.72 必要时，应当在书面规程中规定使用适当的灭鼠药、杀虫剂、抗真菌剂、熏蒸剂和清洁消毒剂，避免其对设备、原料、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85%E6%9D%90%E6%96%99">包装材料</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">标签</a> 、中间体和原料药造成污染。</p><h2>5 工艺设备</h2><h2>5.1 设计和构造</h2><p>5.10 中间体和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 生产用设备应当有合理的设计和足够的尺寸，并放置在适宜的位置，便于其使用、清洁、消毒(如有必要)和维护。</p><p>5.11 设备的构造应确保其与原料、中间体或原料药接触的表面不影响中间体或原料药的质量，使其不符合法定或其他预定标准。</p><p>5.12 生产设备应当在确认的参数范围内使用。</p><p>5.13 中间体或原料药生产过程中使用的主要设备(如反应釜、贮存容器)和固定安装的工艺管道应当有合适的标识。</p><p>5.14 设备所需的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B6%A6%E6%BB%91%E5%89%82">润滑剂</a> 、加热或冷却介质等，不应当与中间产品或原料药直接接触，以免影响中间体或原料药的质量。当任何偏离上述要求的情况发生时，应当进行评估，确保不会对物料的预期用途造成不利影响。应当尽可能使用食用级润滑剂和润滑油。</p><p>5.15 应尽量使用密闭或者限制系统的设备。使用开放式设备或设备打开操作时，应当采取适宜的防护措施将污染风险降至最低。</p><p>5.16 应当保存设备和关键装置(如仪器和公用系统)的现行版图纸。</p><h2>5.2 设备的维护和清洁</h2><p>5.20 应当制定设备的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%A2%84%E9%98%B2%E6%80%A7%E7%BB%B4%E6%8A%A4">预防性维护</a> 计划和操作规程(包括职责分配)。</p><p>5.21 应当制定设备清洁及后续放行用于中间体和原料药生产的操作规程，清洁 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">操作规程</a> 应当尽量详细，确保操作者能以可重现的有效的方式对各类设备进行清洁。清洁操作规程应当包括以下内容：</p><dl><dd>- 设备清洁工作职责分配；</dd></dl><dl><dd>- 清洁计划，必要时包括消毒计划；</dd></dl><dl><dd>- 对清洁方法和物料的完整描述，包括清洁设备所用清洁剂的稀释；</dd></dl><dl><dd>- 为确保正确清洁，如需拆装设备，还应当规定设备拆装方法；</dd></dl><dl><dd>- 移除或涂去前一批次标识的方法；</dd></dl><dl><dd>- 保护已清洁设备在使用前免受污染的方法；</dd></dl><dl><dd>- 如果可行，使用前检查设备清洁状况的方法；</dd></dl><dl><dd>- 必要时，还应当规定设备生产结束至清洁前所允许的最长间隔 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%97%B6%E9%99%90">时限</a> 。</dd></dl><p>5.22 设备和器具应进行清洁、存放，必要时进行消毒或灭菌，防止污染或残留物对中间产品或原料药的质量产生影响，使其不符合法定或其他规定的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 。</p><p>5.23 同一设备连续生产同一原料药或阶段性生产连续数个批次时，宜间隔适当的时间对设备进行清洁，防止污染物(如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%99%8D%E8%A7%A3%E4%BA%A7%E7%89%A9">降解产物</a> 、微生物)的累积和残留。</p><p>5.24 非专用设备更换品种生产前，应当对设备进行清洁，防止 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 。</p><p>5.25 对残留物的可接受标准、所选清洁方法和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B8%85%E6%B4%81%E5%89%82">清洁剂</a> ，应当有明确规定并说明理由。</p><p>5.26 设备应当用适当方法标明其内容物和清洁状态。</p><h2>5.3 校准</h2><p>5.30 应当按照 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">操作规程</a> 和校准计划对中间体或原料药生产和检验用关键控制设备、衡器、量具、仪表以及测试仪器进行校准。</p><p>5.31 应当使用标准器具校准设备，该标准器具可溯源至经认定的标准器具(如有)。</p><p>5.32 应保存校准记录。</p><p>5.33 关键设备当前的校准状态应可知、可核实。</p><p>5.34 不得使用校准不合格的仪器。</p><p>5.35 关键仪器校准若出现 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a> ，应对其进行调查，需明确自最近一次合格校准以来，对采用该设备生产的中间体或原料药的质量是否有影响。</p><h2>5.4 计算机化系统</h2><p>5.40 GMP相关的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AE%A1%E7%AE%97%E6%9C%BA%E5%8C%96%E7%B3%BB%E7%BB%9F">计算机化系统</a> 应当进行验证。验证的深度和广度取决于计算机化系统应用的多样性、复杂性和关键性。</p><p>5.41 应当进行适当的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%89%E8%A3%85%E7%A1%AE%E8%AE%A4">安装确认</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BF%90%E8%A1%8C%E7%A1%AE%E8%AE%A4">运行确认</a> ，证明计算机软硬件适用于执行所指定的任务。</p><p>5.42 对已确认的商业化计算机软件不需要开展同等程度的测试。若现有系统在安装时未经验证，如有足够的证明文件，可对其进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9B%9E%E9%A1%BE%E6%80%A7%E9%AA%8C%E8%AF%81">回顾性验证</a> 。</p><p>5.43 计算机化系统应有足够的控制以防止未经许可的人员访问或更改数据。应有控制措施防止数据遗失(如系统关机导致数据未采集)。应当记录任何数据更改、原始值、更改人和更改时间。</p><p>5.44 企业应当建立计算机化系统的操作和维护规程。</p><p>5.45 当人工输入关键数据时，应当复核输入记录以确保其准确性。这个 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E6%A0%B8">复核</a> 可以由另外的操作人员或由系统本身完成。</p><p>5.46 对可能影响中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 质量、记录或检验结果可靠性的计算机化系统事件应当记录并加以调查。</p><p>5.47 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AE%A1%E7%AE%97%E6%9C%BA%E5%8C%96%E7%B3%BB%E7%BB%9F">计算机化系统</a> 的变更应当根据变更规程执行，并经过正式批准、记录和测试，所有变更应当被记录，包括对系统的软硬件和其它任何关键部件的修改和升级。记录应证明系统处于验证状态。</p><p>5.48 若系统故障或损坏可导致记录的永久性丢失，企业应建立备份系统。所有计算机化系统均应有数据保护措施。</p><p>5.49 除采用计算机系统记录数据外，可采用其他方式进行补充记录。</p><h2>6 文件和记录</h2><h2>6.1 文件系统和质量标准原则</h2><p>6.10 应按照书面规程起草、审核、批准和分发与中间体和原料药生产有关的所有文件，文件可以是纸质或电子形式。</p><p>6.11 所有文件的发放、修订、替换和撤销均应受控，并有修订历史。</p><p>6.12 应当制定保存所有相关文件(如研发历史报告、放大报告、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%80%E6%9C%AF%E8%BD%AC%E7%A7%BB">技术转移</a> 报告、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E9%AA%8C%E8%AF%81">工艺验证</a> 报告、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9F%B9%E8%AE%AD%E8%AE%B0%E5%BD%95">培训记录</a> 、生产记录、检验记录和发运记录)的规程，并规定这些文件的保存期限。</p><p>6.13 所有生产、检验和发运记录应至少保存至该批产品 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%89%E6%95%88%E6%9C%9F">有效期</a> 后一年。对于有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 的原料药，记录应至少保存至该批全部发运后三年。</p><p>6.14 操作完成后，应当及时以不易擦除的方式在相应的空格内记录，并能识别填写人。记录填写的任何更改都应当签注姓名和日期，并使原有信息仍清晰可辨。</p><p>6.15 在记录原件或副本的保存期内，应当能在相应工作现场方便的查阅。可以接受通过电子或其他方式从其他场地快速调阅。</p><p>6.16 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 、指导文件、规程和记录的保存方式既可以是原件，也可以是真实副本，如纸质副本、缩微胶卷、缩微胶片或其它一些准确复制方式。当采用缩微胶卷等缩小技术或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B5%E5%AD%90%E8%AE%B0%E5%BD%95">电子记录</a> 方式时，应准备适当的读取设备，并能随时打印纸质版副本。</p><p>6.17 应制订原料、必要的中间体、原料药、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">标签</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85%E6%9D%90%E6%96%99">包装材料</a> 的书面质量标准。此外，某些其它物料也适宜制定质量标准，例如工艺助剂、垫圈，其他在原料药或中间体生产中使用的可能对质量产生重大影响的物料。应建立 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E8%BF%87%E7%A8%8B%E6%8E%A7%E5%88%B6">中间过程控制</a> 的书面可接受标准。</p><p>6.18 如果在文件中使用 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B5%E5%AD%90%E7%AD%BE%E5%90%8D">电子签名</a> ，应确保其真实性和安全性。</p><h2>6.2 设备的清洁和使用记录</h2><p>6.20 主要设备的使用、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B8%85%E6%B4%81">清洁</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B6%88%E6%AF%92">消毒</a> 和/或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%81%AD%E8%8F%8C">灭菌</a> 以及维护应当有记录，记录内容包括日期、时间(如有必要)、所生产产品的名称和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> ，以及执行清洁和维护的操作人员。</p><p>6.21 如设备专用于某一种中间体或原料药的生产，且生产批次 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E8%BF%BD%E6%BA%AF">可追溯</a> ，则不必建立单独的设备记录；采用专用设备时，相应清洁、维护和使用记录既可以作为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E8%AE%B0%E5%BD%95">批记录</a> 的一部分，也可以单独保存。</p><h2>6.3 原料、中间体、原料药标签和包装材料的记录</h2><p>6.30 记录内容应当包括：</p><dl><dd>- 每次到货的原料、中间体、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 标签和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85%E6%9D%90%E6%96%99">包装材料</a> 的识别信息、数量和生产商名称；供应商的名称；供应商标识的批号或其他编号；接收后企业指定的编号；接收日期；</dd></dl><dl><dd>- 所有检验或检查的结果及相应结论；</dd></dl><dl><dd>- 物料使用过程的记录；</dd></dl><dl><dd>- 确认原料药的标签和包装材料是否符合既定标准的检查和审核记录；</dd></dl><dl><dd>- 不合格原料、中间体、原料药标签和包装材料的最终处理决定。</dd></dl><p>6.31 应当保存经批准的标签样张，用于核对发放的标签。</p><h2>6.4 工艺规程(主生产和检验记录)</h2><p>6.40 为了保证持续稳定的生产，应当制定中间体和原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E8%A7%84%E7%A8%8B">工艺规程</a> 。工艺规程应当由制定人起草、并签注姓名日期， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 的人员应当独立审核、并签注姓名日期。</p><p>6.41 工艺规程内容应当包括</p><dl><dd>- 所生产的中间体或原料药的名称和代码(如适用);</dd></dl><dl><dd>- 所用原料和中间体的完整清单，包括能够表明其质量属性的名称或代码；</dd></dl><dl><dd>- 每种原料和中间体的准确投料量或投料比，包括计量单位； 如用量不固定，应当说明计算方法。如投料量有经论证的波动范围，应当注明；</dd></dl><dl><dd>- 生产场所和主要 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E4%BA%A7%E8%AE%BE%E5%A4%87">生产设备</a> ；</dd></dl><dl><dd>- 生产操作的详细说明，包括：</dd></dl><dl><dd><dl><dd>- 生产步骤；</dd></dl></dd></dl><dl><dd><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%8F%82%E6%95%B0">工艺参数</a> 范围；</dd></dl></dd></dl><dl><dd><dl><dd>- 取样方法， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E8%BF%87%E7%A8%8B%E6%8E%A7%E5%88%B6">中间过程控制</a> 及其可接受标准(如适用);</dd></dl></dd></dl><dl><dd><dl><dd>- 完成单个步骤和/或整个工艺过程的时限(如适用);</dd></dl></dd></dl><dl><dd><dl><dd>- 必要的工艺阶段或时间点的预期收率范围。</dd></dl></dd></dl><dl><dd>- 应当遵守的特殊注意事项和预防措施，或相关要求(如适用）;</dd></dl><dl><dd>- 中间体或原料药的贮存要求，包括标签、包装材料、特殊存放条件及存放 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%97%B6%E9%99%90">时限</a> (如适用)。</dd></dl><h2>6.5 批生产记录(批生产和检验记录)</h2><p>6.50 每批中间体和原料药都应当有相应的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E7%94%9F%E4%BA%A7%E8%AE%B0%E5%BD%95">批生产记录</a> ，其内容应当包括与生产和过程控制有关的完整信息。批生产记录在发放前应核查其版本是否正确，是否清晰准确地采用了相关 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E8%A7%84%E7%A8%8B">工艺规程</a> 。如果批记录依据产品管理文件的某一单独部分制定，该文件应当将现行生产工艺规程列为参考文件。</p><p>6.51 批生产记录在发放时应当有唯一的批号或编号，应记录发放日期并签名。连续生产情况下，在最终批号确定之前，产品代码、日期和时间可以共同作为唯一的识别编号。</p><p>6.52 批生产记录(批生产和检验记录)中每个关键步骤的内容应当包括：</p><dl><dd>- 日期、时间(如适用);</dd></dl><dl><dd>- 所用的主要设备(如反应釜、干燥设备、粉碎设备等)标识；</dd></dl><dl><dd>- 每一批次的具体标识信息，包括生产中所用原料、中间体或所有返工物料的重量、体积等计量值及批号；</dd></dl><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E5%B7%A5%E8%89%BA%E5%8F%82%E6%95%B0">关键工艺参数</a> 的控制结果；</dd></dl><dl><dd>- 取样；</dd></dl><dl><dd>- 生产操作中每个关键步骤的操作人员及负责监督或复核的相关人员的签名；</dd></dl><dl><dd>- 中间过程和实验室检验的结果；</dd></dl><dl><dd>- 必要的工艺阶段或时间点的实际产量/收率；</dd></dl><dl><dd>- 中间体、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 包装材料和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">标签</a> 描述；</dd></dl><dl><dd>- 商业销售用中间体和原料药的标签样张；</dd></dl><dl><dd>- 所有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a> 及相关评估、调查(如适用);如调查记录单独存放应将其列为参考文件；</dd></dl><dl><dd>- 放行检验的结果。</dd></dl><p>6.53 应当制定重大偏差、不合格中间体或原料药调查的书面规程并遵照执行。调查应当包括所有可能与该不合格或偏差相关的批次。</p><h2>6.6 检验记录</h2><p>6.60 检验记录应当包括为确保符合规定标准所进行的全部检验的完整数据，具体包括如下内容：</p><dl><dd>- 待检样品的信息，包括物料名称、来源、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> 或其它的编号、取样日期、接收样品的数量和日期(如适用);</dd></dl><dl><dd>- 所用检验方法的说明或依据；</dd></dl><dl><dd>- 每个检验项目的样品用量；标准品/对照品、试剂和标准溶液的制备与检验信息；</dd></dl><dl><dd>- 每项检验得到的所有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E5%A7%8B%E6%95%B0%E6%8D%AE">原始数据</a> 的完整记录，以及实验室检验仪器产生的图形、图谱和光谱，并准确标明所检验物料的名称和批次等信息；</dd></dl><dl><dd>- 与检验相关的所有计算过程，如计量单位、换算因子和等价因子等；</dd></dl><dl><dd>- 检验结果判定；</dd></dl><dl><dd>- 每项检验人员的签名和日期；</dd></dl><dl><dd>- 对原始记录的准确性、完整性及标准的符合性进行审核的人员(复核人)的签名和日期。</dd></dl><p>6.61 以下情况也应当有完整记录：</p><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 的修订；</dd></dl><dl><dd>- 实验室仪器、设备、仪表和记录装置的定期校准；</dd></dl><dl><dd>- 原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 考察；</dd></dl><dl><dd>- 检验结果超标(<a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B6%85%E6%A0%87">OOS</a> ）的调查。</dd></dl><h2>6.7 批生产记录审核</h2><p>6.70 应当建立 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E7%94%9F%E4%BA%A7%E8%AE%B0%E5%BD%95">批生产记录</a> (包括包装和贴签)和检验记录的审核、批准的书面规程，并遵照执行，以确定中间体或原料药放行或发运之前符合规定标准。</p><p>6.71 原料药放行或发运之前，应当由 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 对其关键步骤的批生产记录和检验记录进行审核、批准。非关键步骤的批生产和检验记录可由有资质的生产人员或其它部门人员按照质量管理部门批准的规程进行审核。</p><p>6.72 所有相关的偏差、调查和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B6%85%E6%A0%87">OOS</a> 报告均应当作为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E8%AE%B0%E5%BD%95">批记录</a> 审核的一部分在放行前进行审核。</p><p>6.73 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 可以授权生产部门放行中间体，但发运至本企业控制范围之外的中间体除外。</p><h2>7 物料管理</h2><h2>7.1 原则</h2><p>7.10 物料的接收、标识、待验、贮存、处理、取样、检验、批准或拒绝放行均应当有书面规程。</p><p>7.11 中间体和/或原料药的生产企业应当建立关键物料供应商的评估系统。</p><p>7.12 物料应当按协议标准从经质量管理部门批准的一个或多个供应商处采购。</p><p>7.13 如果某 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E7%89%A9%E6%96%99">关键物料</a> 的供应商和生产商不同，则中间体和/或原料药的生产企业应当知道该物料生产商的名称和地址。</p><p>7.14 关键原料来源的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%98%E6%9B%B4">变更</a> 应当遵照第13节变更控制执行。</p><h2>7.2 接收和待验</h2><p>7.20 物料接收时，应当检查包装容器的标签是否正确(包括供应商所用名称与企业内部所用名称不同时，核对两者的相互关系)、容器是否受损，封口有无破损和开启痕迹、有无污染。物料应当待验贮存，直至其被取样、检查或检验(如适用),并放行使用。</p><p>7.21 进厂物料与现有库存(如溶剂或料仓库存)混合之前，应当确认物料标识是否正确，是否已进行检验，如需放行，还应确认是否已被放行。应当制定防止将物料错放到现有库存中的规程。</p><p>7.22 采用非专用槽车运送的大宗物料，应当采取措施确保避免来自槽车所致的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> ，所采取的适当措施应当包括以下一种或几种：</p><dl><dd>- 清洁证明；</dd></dl><dl><dd>- 痕量杂质的检验；</dd></dl><dl><dd>- 供应商审计。</dd></dl><p>7.23 大的贮存容器及其所附配件、进料管路和出料管路都应当有适当的标识。</p><p>7.24 用于盛装物料的每个或每组容器都应当指定并标明其识别码、批号或接收号，该号码应当用于记录每批物料的处置情况。应当建立标识每批物料状态的系统。</p><h2>7.3 进厂物料的取样和检验</h2><p>7.30 除7.32条款中所述物料外，其它每批物料都应当至少进行一项鉴别检验。如生产企业建立了供应商评估体系，可采用供应商的检验报告代替其他检验项目。</p><p>7.31 供应商批准前，应当进行评估，获取足够的证据(如历史质量情况),证明生产商能够持续提供符合标准的物料。在减少内部检测项目之前应当至少进行三个批次的全检。同时至少应当在适当的时间间隔进行一次全检，并将企业检验结果与供应商的检验报告进行对比。检验报告的可靠性应当定期审核。</p><p>7.32 对于工艺助剂、有害或有剧毒的原料、其它特殊物料或转移到本企业另一生产场地的物料，如上述物料生产企业提供的检验报告显示其符合标准，则可以免检。应当对其 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%B9%E5%99%A8">容器</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">标签</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> 进行目检予以确认。这些物料的免检情形应当说明理由并有文件记录。</p><p>7.33 样品应当具有代表性。取样方法应当规定取样件数、每个容器的取样部位以及其取样量。应当根据物料的关键程度、批内或批间的差异性、供应商以往的质量情况和检验用量来制定取样计划，确定取样件数和取样量。</p><p>7.34 取样应在指定地点按照规定的规程进行，以防止污染所取样物料和其它物料。。</p><p>7.35 应当小心开启被抽取样品的容器，随后恢复包装。已取样的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%96%E5%8C%85%E8%A3%85">外包装</a> 应当有标识。</p><h2>7.4 贮存</h2><p>7.40 物料的处理和贮存应当能够防止降解、污染和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 。</p><p>7.41 贮存在纸桶、袋和箱内的物料不应当直接放在地上，必要时留出适当的空间以便于清洁和检查。</p><p>7.42 物料的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%AE%E5%AD%98">贮存</a> 条件和周期不应当对其质量产生不利影响，通常应当先进先出。</p><p>7.43 在适当的容器内保存的物料可以存放在室外，确保标签清晰，在开启或使用前对容器进行清洁。</p><p>7.44 不合格物料应当有标识，处理之前应当有效隔离，以防止其未经许可用于生产。</p><h2>7.5 再评估</h2><p>7.50 必要时应当对物料进行再评估，以确保其适用性(如存放时间过长或暴露在高温潮湿环境中)。</p><h2>8 生产和过程控制</h2><h2>8.1 生产操作</h2><p>8.10 用于生产中间体和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的原料应当在适宜的、不影响其适用性的条件下称量或计量，称量和计量的装置应当具有与其使用目的相适应的精度。</p><p>8.11 如物料取出一部分留待以后生产使用，应当存放在适当的容器中并有标识，标识内容应当包括：</p><dl><dd>- 物料名称和/或代码；</dd></dl><dl><dd>- 接收或控制编码；</dd></dl><dl><dd>- 新容器中物料的重量或数量；</dd></dl><dl><dd>- 再评估或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> (如适用)。</dd></dl><p>8.12 关键的称量、计量或分装操作应当有复核或有类似的控制手段。使用前，生产人员应当核实所用物料正确无误。</p><p>8.13 其它关键操作也应当有复核或有类似的控制手段。</p><p>8.14 应当将生产过程中指定步骤的实际收率与预期收率比较。预期收率的范围应当根据以前的实验室、中试或商业化规模生产的数据来确定。应当对关键工艺步骤收率的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a> 进行调查，确定偏差对相关批次产品质量的影响或潜在影响。</p><p>8.15 任何偏差都应记录并作解释。任何重大偏差均应进行调查。</p><p>8.16 应当在设备主要部件上标明工作状态，或以文件形式、计算机控制形式或其它方式标明主要设备单元的工作状态。</p><p>8.17 应当对返工或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%87%8D%E6%96%B0%E5%8A%A0%E5%B7%A5">重新加工</a> 的物料进行适当的控制，以防止其未经许可用于生产。</p><h2>8.2 时限</h2><p>8.20 如果生产 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E8%A7%84%E7%A8%8B">工艺规程</a> (见6.41节)中规定了操作 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%97%B6%E9%99%90">时限</a> ，应当遵照执行，以确保中间体和原料药的质量。如有偏差，则应记录并评估。在有目标值的工艺条件下(如调 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PH">pH 值</a> 、氢化、干燥至符合既定标准)，则不适用时限控制，因其反应终点或加工步骤的完成是根据中间控制的取样和检验来确定的。</p><p>8.21 需进一步加工的中间产品应当在适宜的条件下存放，确保其适用性。</p><h2>8.3 中间控制和取样</h2><p>8.30 应当建立书面规程，用于监测和控制那些影响中间体和原料药质量的工艺步骤。应当根据研发阶段的信息或历史数据确定中间控制及其可接受标准。</p><p>8.31 应当综合考虑所生产中间体或原料药的特性、反应类型、工艺步骤对产品质量影响的大小等因素来确定控制标准、检验类型和范围。前期生产的中间控制严格程度可较低，越接近最终工序(如分离和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BA%AF%E5%8C%96">纯化</a>)中间控制越严格。</p><p>8.32 应当有经 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 批准的书面文件，规定关键中间控制(和关键工艺监测),包括工艺控制点和控制方法。</p><p>8.33 有资质的生产部门人员可进行中间控制，并可在质量管理部门事先批准的范围内对生产操作进行必要的调整，不再经质量管理部门批准。所有检验及其结果均应当完整记录，并作为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E8%AE%B0%E5%BD%95">批记录</a> 的一部分。</p><p>8.34 应当制定书面规程说明生产过程中的物料、中间体和原料药的取样方法。取样计划和程序应当基于科学合理的取样实践。</p><p>8.35 应当建立防止污染和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 的中间控制取样规程，并遵照执行。应当制定规程，以确保取样后样品密封完好</p><p>8.36 以工艺监测和/或调节为目的的中间控制检验结果超标(<a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B6%85%E6%A0%87">OOS</a> ）通常不需要进行调查。</p><h2>8.4 中间体或原料药的混合</h2><p>8.40 本文中“混合”指将符合同一 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 的原料药或中间产品合并，以得到均一产品的工艺过程。将来自同一批次的各部分产品(如同一结晶批号的中间产品分数次离心)在生产中进行合并，或将几个批次的中间产品合并在一起作进一步加工，可作为生产工艺的组成部分，不视为混合。</p><p>8.41 不得将不合格批次与其他批次混合以使其达到符合质量要求的目的。拟混合的每批产品均应当按照规定的工艺生产、单独检验，并符合相应质量标准。</p><p>8.42 可接受的混合操作包括但不限于以下情形：</p><dl><dd>- 将数个小批次混合以增加批量；</dd></dl><dl><dd>- 将同一中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的多批零头产品(如某个单独批次物料的一小部分)混合成为一个批次。</dd></dl><p>8.43 混合过程应当加以控制并有完整记录，混合后的批次应当适当进行检验，确认其符合质量标准。</p><p>8.44 混合的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E8%AE%B0%E5%BD%95">批记录</a> 应当能够追溯到参与混合的每个单独批次。</p><p>8.45 物理性质至关重要的原料药(如用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%A3%E6%9C%8D%E5%9B%BA%E4%BD%93%E5%88%B6%E5%89%82">口服固体制剂</a> 或混悬剂的原料药),其混合工艺应当进行验证，验证应当包括证明混合批次的质量均一性及对关键特性(如粒径分布、松密度和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A0%86%E5%AF%86%E5%BA%A6">堆密度</a>)的检测。</p><p>8.46 混合可能对产品的稳定性产生不利影响的，应当对最终混合的批次进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 考察。</p><p>8.47 混合批次的有效期或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 应当根据参与混合的最早批次产品的生产日期确定。</p><h2>8.5 污染的控制</h2><p>8.50 同一中间产品或原料药的残留物(如黏附在微粉机壁上的残留物、离心出料后筒体内残留的潮湿结晶、物料转至下步工序时未排尽的液体或结晶)带入后续数个批次中的，应当严格控制。残留物的带入不应当引入可能对原料药的杂质分布产生不利影响的降解物或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%AE%E7%94%9F%E7%89%A9%E6%B1%A1%E6%9F%93">微生物污染</a> 。</p><p>8.51 生产操作应当能够防止中间产品或原料药被其它物料污染。</p><p>8.52 原料药精制后的操作应当特别注意防止污染。</p><h2>9 原料药和中间体的包装和贴签</h2><h2>9.1 原则</h2><p>9.10 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E7%AD%BE">标签</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85%E6%9D%90%E6%96%99">包装材料</a> 的接收、标识、待验、取样、检查和/或检验、放行和处理应当有书面规程。</p><p>9.11 标签和包装材料应当符合标准。不符合标准的应当拒绝放行，以防止其不当使用。</p><p>9.12 无论标签和包装材料是否接收，其进货记录都应当有购货凭证、检查或检验记录。</p><h2>9.2 包装材料</h2><p>9.20 容器应当能够保护中间产品或原料药，使其在运输和规定的贮存条件下不变质、不受污染。</p><p>9.21 应当对容器进行清洁，如中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的性质有要求时，还应当进行消毒，确保其适用性。容器不得因与产品发生反应、释放物质或吸附作用而影响中间体或原料药的质量。</p><p>9.22 应当按照 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">操作规程</a> 对可以重复使用的容器进行清洁，并去除或涂毁容器上原有的标签。</p><h2>9.3 标签的发放和控制</h2><p>9.30 未经授权的人员不得进入标签存放区域。</p><p>9.31 应当按照操作规程检查标签发放、使用和退回的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%89%A9%E6%96%99%E5%B9%B3%E8%A1%A1">物料平衡</a> ，对发放标签与已贴标签容器的数量进行评估。评估发现物料平衡有异常时，应当进行调查，并经 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 批准。</p><p>9.32 已打印批号或其他批信息的剩余标签均应当销毁。应当采取专门措施保存退回的标签，防止混淆并正确标识。</p><p>9.33 废弃和过期的标签应当予以销毁。</p><p>9.34 应当对标签打印设备进行控制，以确保所有标签的打印内容符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E7%94%9F%E4%BA%A7%E8%AE%B0%E5%BD%95">批生产记录</a> 中规定的要求。</p><p>9.35 发放的标签打印后应当仔细检查确保正确无误，并确认其符合批生产记录中规定的标准。检查结果应当有记录。</p><p>9.36 所用打印标签实样应当放入批生产记录。</p><h2>9.4 包装和贴签操作</h2><p>9.40 应当建立 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">操作规程</a> 以确保所用包装材料和标签正确无误。</p><p>9.41 贴签操作应当合理设计以避免混淆。不同中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的贴签操作应当采取物理隔离或空间隔离措施。</p><p>9.42 中间体或原料药容器上的标签应当注明与质量有关的关键信息，如产品名称、代码、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> 及贮存条件。</p><p>9.43 中间体或原料药一旦脱离生产商物料管理系统的控制，其标签上还应当注明生产商名称、地址、数量、特殊运输条件和特殊法定要求。有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%89%E6%95%88%E6%9C%9F">有效期</a> / <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 的中间体或原料药，应当在标签和检验报告中注明有效期/复验期。</p><p>9.44 包装开始前应当对包装和贴签的设施设备进行检查，确保已移除与本批产品包装无关的物料。该检查应当记录于批生产记录、设备使用日志或其他文件中。</p><p>9.45 应当检查已包装贴签的中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> ，确保容器和包装上标签正确无误。该检查应当纳入包装操作，检查结果应当记录在批生产或检验记录中。</p><p>9.46 中间产品或原料药一旦脱离生产商控制，其容器应当采取适当的方式进行密封，以便在密封被破坏或缺失时，可警示接收者其内容物可能发生改变。</p><h2>10 贮存和发运</h2><h2>10.1 入库规程</h2><p>10.10 仓储区应当能够提供满足所有物料贮存的条件(如，必要的温湿度)。对影响物料质量的贮存条件应当有记录。</p><p>10.11 为避免误用或未经批准使用， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%85%E9%AA%8C">待验</a> 、不合格、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%80%80%E8%B4%A7">退货</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AC%E5%9B%9E">召回</a> 的物料在明确后续用途前，应当存放于单独的隔离区域。</p><h2>10.2 发运规程</h2><p>10.20 仅当质量部门放行后，原料药和中间体方可发运给第三方。待验中的原料药和中间体经质量控制部门授权，或者进行必要的控制措施并记录后，可移交给同一企业的另一部门。</p><p>10.21 原料药和中间体的运输方式不得对其质量有不利影响。</p><p>10.22 有特殊运输或贮存条件要求的原料药或中间体应当在其标签中注明。</p><p>10.23 生产企业应当确保 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 或中间体的运输方了解运输和贮存条件要求，并遵照执行。</p><p>10.24 应当建立召回系统，确保能够快速有效地确定并召回已发运的每一批中间体和/或原料药。</p><h2>11 实验室管理</h2><h2>11.1 基本原则</h2><p>11.10 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%8E%A7%E5%88%B6%E5%AE%9E%E9%AA%8C%E5%AE%A4">质量控制实验室</a> 应当配备足够的实验设施。</p><p>11.11 应当制定物料取样、检测、批准或拒绝放行、实验数据记录和保存的程序。实验记录的保存应符合第6.6节的有关规定。</p><p>11.12 质量标准、取样计划和检验规程应当科学合理，以确保原料、中间体、原料药、标签和包装材料符合预定的质量与纯度要求。质量标准和检验规程应当与注册申报一致。此外，可建立附加的质量标准。质量标准、取样计划、检验规程及其变更应当由适宜的部门起草并经质量管理部门审核和批准。</p><p>11.13 原料药质量标准应当根据可接受标准及生产工艺制定，应当包括杂质检测项(如有机杂质、无机杂质和残留溶剂)。对于有微生物限度、细菌内毒素标准要求的原料药，应当建立并执行合理的微生物总数和控制菌、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%86%85%E6%AF%92%E7%B4%A0">内毒素</a> 行动限度。</p><p>11.14 应当严格遵守检验规程，并及时记录实验操作。任何偏离检验规程的偏差情况都应当记录并说明。</p><p>11.15 应当建立检验结果超标调查的操作规程。任何检验结果超标都必须按照操作规程进行调查并记录。调查内容应当包括对数据的分析，重大问题评估，整改措施的责任人和结论。其中重新取样和/或复检均应当按规程开展。</p><p>11.16 试剂和标准溶液应当按 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">操作规程</a> 进行配制和标识。适用时分析试剂或标准溶液应标明使用截止日期。</p><p>11.17 原料药生产应有标准品/对照品。标准品/对照品的来源应当有记录，其贮存和使用应当遵循供应商的建议，并进行记录。法定来源的标准品/对照品，如始终按照供应商建议的条件进行保存，可直接使用而不必检测。</p><p>11.18 当无法获得法定来源标准品/对照品时，应当制备“内部标准品/对照品”,并进行全面的鉴别和纯度分析。相关检测应当记录并存档。</p><p>11.19 工作标准品/对照品应当进行合理的制备、鉴别、检验、批准和贮存。每批工作 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%87%E5%87%86%E5%93%81">标准品</a> / <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AF%B9%E7%85%A7%E5%93%81">对照品</a> 在首次使用前应与标准品/对照品进行比对，以确定其适用性。应当按照书面文件对每批工作标准品/对照品定期进行确认。</p><h2>11.2 中间体和原料药的检测</h2><p>11.20 应当对每一批中间体和原料药进行适当的检测，确认其符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 。</p><p>11.21 通常每种原料药均应当建立杂质档案，对按受控的生产工艺生产的典型批次中的已知和未知杂质进行描述。杂质档案应当包括所观察到的每一种杂质的范围、鉴别或某些定性分析特征(如保留时间)、每一种已知杂质的类别(如有机杂质、无机杂质、溶剂)。杂质分布一般与原料药的生产工艺和所用起始原料有关。从植物或动物组织制得的原料药通常不要求建立杂质档案。关于生物技术方面的考虑见 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6B_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_Biotechnological/Biological_Products">ICH Q6B</a> 。</p><p>11.22 应当定期将产品的杂质分析资料与注册申报资料中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> 档案，或与以往的杂质数据相比较，查明原料、设备运行参数和生产工艺的变更所致原料药质量的变化。</p><p>11.23 对于有微生物控制要求的中间体或原料药，每批均应当进行适当的微生物检测。</p><h2>11.3 分析方法的验证一参见第12节</h2><h2>11.4 检验报告</h2><p>11.40 每批中间体或原料药应当按要求出具真实可靠的检验报告。</p><p>11.41 检验报告应当包括中间体或原料药的名称、等级(必要时)、批号和放行日期，对于规定了有效期或复验期的中间体或原料药，其 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%89%E6%95%88%E6%9C%9F">有效期</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 应当在其标签和检验报告中注明。</p><p>11.42 检验报告应当列明按药典或客户要求所进行的每一项检测，包括可接受限度和数值结果(对于检验结果是数值的)。</p><p>11.43 检验报告应当由经授权的质量管理部门人员签名并写明日期，其中应标明原生产商的名称，地址和电话号码。如果由分包装商或重新加工方进行检测的，检验报告中应当标明分包装商或重新加工方的名称、地址、电话号码，以及原生产商的名称。</p><p>11.44 分包装商、重新加工方、代理商、中间商及其代表出具新的检验报告时，检验报告中应当标明进行检测的实验室的名称、地址、电话号码，同时应提供原生产企业的名称和地址，并附原批检验报告复印件。</p><h2>11.5 原料药稳定性考察</h2><p>11.50 应当建立持续进行的稳定性考察程序，监测原料药的稳定性特性，考察结果应当用于确认适当的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%AE%E5%AD%98">贮存</a> 条件、复验期或有效期。</p><p>11.51 稳定性检测的方法应当能够显示质量变化趋势，并经验证。</p><p>11.52 用于稳定性试验的样品应当模拟市售包装。例如采用包装袋装入纤维桶进行上市包装的原料药，其稳定性试验的样品可以装在相同材料的包装袋并置于与上市包装桶的材料相近或相同的小规格的包装桶中。</p><p>11.53 通常应对商业化生产批次的前三批进行稳定性考察以确认复验期或有效期。如历史研究数据表明原料药至少在两年内质量稳定，稳定性试验的批次可少于3批。</p><p>11.54 此后，每年至少抽取一批原料药进行稳定性考察(除非当年没有生产),并至少每年进行一次检测，以确认产品的稳定性。</p><p>11.55 对于保存期限较短的原料药，其检测的频率应更高。例如，对于保存期限在一年或更短的生物技术产品/ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%B6%E5%93%81">生物制品</a> 以及其它原料药，其稳定性样品在前三个月内应每月检测一次，之后每三个月检测一次。如果已有的稳定性试验数据表明该原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 不受影响，则可考虑不做某些特定时间间隔的检测(如9个月时的检测)。</p><p>11.56 原则上，稳定性试验的存放条件应与ICH 稳定性指南一致。</p><h2>11.6 有效期和复验期</h2><p>11.60 当一个中间体要运送到生产商的物料管理系统控制范围以外，并规定了有效期或复验期时，应当有支持性的稳定性信息(如公布的数据、检测结果)。</p><p>11.61 原料药的有效期或复验期应基于对其稳定性试验数据的评估。通常采用复验期，而非有效期。</p><p>11.62 如果满足以下条件，可以基于中试规模的批次来确定原料药初步的有效期或复验期：(1)中试批次采用的生产方法和程序是模拟用于商业生产规模的最终工艺；(2) <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E8%AF%95">中试</a> 原料药的质量可以代表商业化规模产品的质量。</p><p>11.63 供复验的样品应有代表性。</p><h2>11.7 留样</h2><p>11.70 留样的包装和存放是供将来对原料药批次进行可能的质量评估之用，而不是供以后的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 试验之用。</p><p>11.71 经适当标识的每一批原料药的留样应留存至生产商所规定的有效期后一年，或该批发运后三年，以较长时间为准。对于有 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 的原料药，类似留样应留存至该批次被生产商全部发运完后三年。</p><p>11.72 留样应保存在与原料药保存相同的包装系统中，或保存在与上市包装等同或保护性更好的包装系统中。留样量应至少可供两次按药典标准的全检之用；若无药典标准，则应至少可供两次按质量标准的全检之用。</p><h2>12 验证</h2><h2>12.1 验证方针</h2><p>12.10 企业的总体验证方针、目的和方法，包括针对生产工艺验证、清洁操作规程、分析方法、生产过程控制的检验规程的验证和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AE%A1%E7%AE%97%E6%9C%BA%E5%8C%96%E7%B3%BB%E7%BB%9F">计算机化系统</a> 验证，以及负责设计、审核、批准和记录每个验证阶段的人员，都应当有文件规定。</p><p>12.11 关键质量属性和工艺参数通常在研发阶段或根据历史资料和数据确定，且应当规定工艺操作重现性所必需的范围，包括：</p><dl><dd>- 定义原料药的关键产品属性；</dd></dl><dl><dd>- 确认可能对原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E8%B4%A8%E9%87%8F%E5%B1%9E%E6%80%A7">关键质量属性</a> 产生影响的工艺参数；</dd></dl><dl><dd>- 确定日常生产和工艺控制中所用各 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E5%B7%A5%E8%89%BA%E5%8F%82%E6%95%B0">关键工艺参数</a> 的预期范围；</dd></dl><p>12.12 验证的范围应当延伸至对原料药质量和纯度至关重要的操作。</p><h2>12.2 验证文件</h2><p>12.20 应当建立书面 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%AA%8C%E8%AF%81%E6%96%B9%E6%A1%88">验证方案</a> ，阐明如何进行具体工艺的验证。验证方案应当由质量管理部门和其它指定部门审核并批准。</p><p>12.21 验证方案应当明确规定关键工艺步骤和可接受标准、验证类型(如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9B%9E%E9%A1%BE%E6%80%A7%E9%AA%8C%E8%AF%81">回顾性验证</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%89%8D%E9%AA%8C%E8%AF%81">前验证</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%90%8C%E6%AD%A5%E9%AA%8C%E8%AF%81">同步验证</a>)以及工艺运行次数。</p><p>12.22 应当拟定一份交叉引用验证方案的验证报告，总结所得到的结果，对观察到的任何偏差进行评估，并得出合理的结论，包括为缺陷整改而提出变更。</p><p>12.23 任何与验证方案的偏离都应当进行记录，并有合理解释。</p><h2>12.3 确认</h2><p>12.30 工艺验证开始前，关键设备及辅助系统应当完成适当的确认。通常通过下列单独或联合活动来进行确认：</p><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AE%BE%E8%AE%A1%E7%A1%AE%E8%AE%A4">设计确认</a> (DQ): 确认设施、设备或系统的设计适合其预定用途，并有相应文件。</dd></dl><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%89%E8%A3%85%E7%A1%AE%E8%AE%A4">安装确认</a> (IQ): 确认安装或调整后的设备或系统符合经批准的设计标准、制造商建议和(或)用户要求，并有相应文件。</dd></dl><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BF%90%E8%A1%8C%E7%A1%AE%E8%AE%A4">运行确认</a> (OQ): 确认安装或调整后的设备或系统在整个预期操作范围内能按意图正常运行，并有相应文件。</dd></dl><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%80%A7%E8%83%BD%E7%A1%AE%E8%AE%A4">性能确认</a> (PQ): 确认设备及辅助系统连接后，按经批准的工艺方法和标准，能重复、有效地发挥其性能，并有相应文件。</dd></dl><h2>12.4 工艺验证的方式</h2><p>12.40 工艺验证(PV）是证明生产工艺在其规定参数范围内运行时，能重复有效地生产出符合既定 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 和质量属性的中间体或原料药的书面证据。</p><p>12.41 工艺验证有三种方式。前验证是首选方式，但在某些情况下亦可采用其它方式。这些验证方式及其适用性在下文阐述。</p><p>12.42 第12.12节中所述的所有原料药工艺一般都应进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%89%8D%E9%AA%8C%E8%AF%81">前验证</a> 。原料药工艺的前验证应当在用其生产的制剂产品上市销售前完成。</p><p>12.43 由于原料药生产批次有限、原料药不经常生产或原料药生产所采用的已验证的生产工艺发生 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%98%E6%9B%B4">变更</a> ，无法从连续生产中获得数据，此时可采用同步验证方式。在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%90%8C%E6%AD%A5%E9%AA%8C%E8%AF%81">同步验证</a> 完成之前，基于对原料药批次的充分监控和检测，这些批次可以放行并用于商业化制剂产品的生产。</p><p>12.44 对于已投入使用的生产工艺，如没有发生因原料、设备、系统、设施或生产工艺改变而对原料药质量有影响的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%87%8D%E5%A4%A7%E5%8F%98%E6%9B%B4">重大变更</a> 时，可以作为例外采用 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9B%9E%E9%A1%BE%E6%80%A7%E9%AA%8C%E8%AF%81">回顾性验证</a> 。此类验证方式可在以下情况下采用：</p><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E8%B4%A8%E9%87%8F%E5%B1%9E%E6%80%A7">关键质量属性</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%85%B3%E9%94%AE%E5%B7%A5%E8%89%BA%E5%8F%82%E6%95%B0">关键工艺参数</a> 均已确定；</dd></dl><dl><dd>- 已建立了合适的中控可接受标准和过程控制；</dd></dl><dl><dd>- 除操作人员失误或与设备适用性无关的设备故障外，无其他原因导致的重大工艺问题/产品不合格；</dd></dl><dl><dd>- 已经建立了现有原料药的杂质档案。</dd></dl><p>12.45 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9B%9E%E9%A1%BE%E6%80%A7%E9%AA%8C%E8%AF%81">回顾性验证</a> 所选批次应当能够代表审核时段内的所有批次，包括所有不符合质量标准的批次，并且应有足够的批数来证明工艺的一致性。可通过检测留样来获取工艺回顾性验证所需的数据。</p><h2>12.5 工艺验证程序</h2><p>12.50 工艺验证时的工艺运行次数取决于工艺的复杂性或所考虑的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%8F%98%E6%9B%B4">工艺变更</a> 的大小。对于前验证和同步验证，原则上应当采用连续三个成功批次，但有些情况下(如复杂或耗时很长的原料药工艺),需要增加验证批数以证明工艺的一致性。对于回顾性验证，一般应审核连续 10-30批的数据，以评估工艺的一致性，但有合理理由时，也可减少审核的批次。</p><p>12.51 工艺验证研究期间，应对关键工艺参数进行控制和监测。与产品质量无关的工艺参数，比如某些为减少能耗或设备使用而进行调控的变量，可不列入验证对象。</p><p>12.52 工艺验证应当确认各 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的杂质档案均在规定限度内。其杂质档案应与历史数据相似或更好；如有可能，与 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%BC%80%E5%8F%91">工艺开发</a> 阶段确定的杂质档案或供关键临床和毒理研究的批次比较，应相似或更好。</p><h2>12.6 已验证系统的定期审核</h2><p>12.60 应当对系统或工艺进行定期评估，以确认它们仍能有效地运作。当系统或工艺未发生显著性变化，且质量回顾表明系统或工艺在持续稳定地生产符合质量标准的物料，通常不需要进行再验证。</p><h2>12.7 清洁验证</h2><p>12.70 清洁规程通常应当经过验证。一般而言， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B8%85%E6%B4%81%E9%AA%8C%E8%AF%81">清洁验证</a> 应当针对那些物料污染或残留对原料药质量有最大风险的情况或工艺步骤。例如，生产的前期阶段，可能不必对设备的清洁规程进行验证，因其残留物可在后续纯化步骤中除去。</p><p>12.71 清洁规程的验证应当反映设备实际使用情况。如果同一设备用于多个中间体或原料药的生产，且采用相同的方法清洗，则可选择某个有代表性的中间体或原料药进行清洁验证。此时，应当根据其溶解性、清洁难度以及基于效价、毒性或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 计算的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AE%8B%E7%95%99%E9%99%90%E5%BA%A6">残留限度</a> 来做出选择。</p><p>12.72 清洁验证方案应当写明需要清洁的设备、程序、物料、可接受的清洁水平、需要监测和控制的参数以及分析方法。同时应当指明所需取样的样品类型以及如何取样和标识。</p><p>12.73 取样的方法应当包括擦拭、淋洗或其它适用方法(如直接萃取)，同时检测可溶性和不溶性的残留物。所用的取样方法应当能对清洁后设备表面残留物水平进行定量检测。由于设备设计和/或工艺限制(如软管、输送管路、开口小或处理有毒物质的反应釜、微粉机和微型 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B5%81%E5%8C%96%E5%BA%8A">流化床</a> 等小型复杂设备的内表面)使得产品接触表面不易触及，此时擦拭取样的方法可能不可行。</p><p>12.74 应当采用经验证的有足够灵敏度的分析方法来检测残留物或污染物。各分析方法的检测限应当足以检测到可接受水平的残留物或污染物。应当确定分析方法可达到的回收率。 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%AE%8B%E7%95%99%E9%99%90%E5%BA%A6">残留限度</a> 应当切实可行、可达到、可证实，并基于危害程度最高的残留物来制定。可以根据原料药或其最有害组分的已知最低药理、毒理或生理活性浓度来制定限度。</p><p>12.75 对于那些需降低原料药中微生物总数和内毒素的工艺，或需关注此类污染的工艺(如用于生产无菌产品的非无菌原料药),其设备清洁/消毒研究应当考虑解决微生物和内毒素污染问题。</p><p>12.76 验证后的清洁规程应当适时监控，以确保这些规程在日常生产中的有效性。尽可能通过分析测试和目视检查来监控设备的清洁程度。目视检查能够发现那些用取样和/或分析检测发现不了的、集中在小面积上的严重污染。</p><h2>12.8 分析方法验证</h2><p>12.80 分析方法应当进行验证，除非采用的方法在相关药典或其他公认的参照标准中有记载。所有分析方法的适用性都应当在实际使用条件下核实，并记录。</p><p>12.81 方法验证应当考虑ICH分析方法验证指南中的要素。分析方法验证的程度应当反映分析的目的和原料药生产工艺的阶段。</p><p>12.82 开始分析方法验证前，应当考虑对分析设备进行适当的确认。</p><p>12.83 对已验证的分析方法的任何修改均须保留完整的记录。记录中应当包括修改的原因和合适的数据，以证实采用修改后的方法得到的结果与已有的方法同样准确、可靠。</p><h2>13 变更控制</h2><p>13.10 应当建立一套正式的变更控制系统，以评估可能影响原料药或中间体的生产和控制的所有变更。</p><p>13.11 应当建立书面规程，对原料、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 、分析方法、设施、支持系统、设备(包括计算机硬件)、工艺步骤、标签、包装材料以及计算机软件的变更进行识别、记录、适当的审核和批准。</p><p>13.12 任何与GMP相关的变更申请都应当由合适的组织部门进行起草、审核和批准，并由质量管理部门审核和批准。</p><p>13.13 应当评估所提变更对中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 质量的潜在影响。采用某种分类规程可有助于确定对已验证的工艺进行变更所需的检测、验证和文件记录的程度。变更可以根据其性质、变更程度及其可能对工艺产生的影响来分类(如次要变更或主要变更)。为确认对一个已验证工艺进行变更的合理性，应当采用科学的判断来决定进行什么样的附加测试和验证研究才是适当的。</p><p>13.14 实施已经批准的变更时，应当采取措施确保变更影响到的所有文件都进行修订。</p><p>13.15 变更实施后，应当对变更后生产或检验的前几批产品进行评价。</p><p>13.16 应当评估关键变更影响既定复验期或有效期的可能性。如有必要，变更后生产的中间体或原料药样品可进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8A%A0%E9%80%9F%E7%A8%B3%E5%AE%9A%E6%80%A7%E8%AF%95%E9%AA%8C">加速稳定性试验</a> 和/或列入稳定性考察计划。</p><p>13.17 可能影响原料药质量的生产和工艺控制 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%98%E6%9B%B4">变更</a> 应当告知现有的制剂生产商。</p><h2>14 物料的拒绝放行和再使用</h2><h2>14.1 拒绝放行</h2><p>14.10 不符合既定质量标准的中间体和原料药应进行标识并隔离。这些中间体或原料药可按以下要求进行返工或重新加工，不合格物料的最终处置应有记录。</p><h2>14.2 返工</h2><p>14.20 通常可以将中间体或原料药(包括不符合质量标准的)重复既定生产工艺中的步骤进行返工，例如重结晶或其它物理、化学处理，如蒸馏、过滤、层析、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%B2%89%E7%A2%8E">粉碎</a> 。但是，多数批次都要进行的返工，应当作为一个工艺步骤列入常规的生产工艺中。</p><p>14.21 经中间控制检测表明某一工艺步骤尚未完成，仍可按正常工艺继续操作的，不属于返工。</p><p>14.22 将未反应的物料重新引入生产工艺并重新进行化学反应的操作属于返工，除非该操作是既定工艺的一部分。进行此类返工前应仔细评估，以确保不会由于可能形成的副产物和过度反应产物而对中间体或原料药的质量产生不良影响。</p><h2>14.3 重新加工</h2><p>14.30 在决定对不符合既定质量标准的批次进行重新加工之前，应当对其不合格的原因进行调查。</p><p>14.31 经过重新加工的批次应当进行适当的评估、检测，必要时还需进行稳定性试验，并予以记录，以表明 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%87%8D%E6%96%B0%E5%8A%A0%E5%B7%A5">重新加工</a> 产品的质量与原工艺生产产品的质量相同。对于重新加工工艺，通常采用同步验证的方式进行验证。这样可在方案中规定重新加工的程序、工艺如何实施以及预期结果。如果仅有一个批次进行重新加工，则可形成书面报告，如确认可接受，可放行该批产品。</p><p>14.32 应当有操作规程规定对每批重新加工的产品与常规工艺生产的产品进行杂质档案比较。当常规 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 不足以评估重新加工批次的特性时，应当采用其他方法。</p><h2>14.4 物料和溶剂的回收</h2><p>14.40 如有经批准的回收规程，并且回收物料符合与其预定用途相适应的质量标准时，反应物、中间体或原料药可以回收(如从母液或滤液中回收)。</p><p>14.41 溶剂可以回收，并在相同或不同的工艺中重新使用，但应当控制和监测回收过程，以确保溶剂在重新使用前或与其它合格物料混合前符合适当的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 。</p><p>14.42 如果有足够的检测数据表明适用于其可能被使用的所有生产工艺，则可以将新鲜及回收溶剂和试剂混合。</p><p>14.43 回收溶剂、母液和其它回收物料的使用应当充分记录。</p><h2>14.5 退货</h2><p>14.50 应当对退回的中间体或原料药进行标识并隔离。</p><p>14.51 退回的中间体或原料药在其退回之前、或者运输途中的贮存条件或其包装状况使其质量值得怀疑时，应当视情形对退回的中间体或原料药进行返工、重新加工或销毁。</p><p>14.52 应当保存中间体或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 退货记录。每次的退货记录应当包括：</p><dl><dd>- 收货方的名称和地址；</dd></dl><dl><dd>- 退回中间体或原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> 、数量；</dd></dl><dl><dd>- 退货原因；</dd></dl><dl><dd>- 退回中间体或原料药的使用或处置。</dd></dl><h2>15 投诉和召回</h2><p>15.10 应当按照书面规程对所有与质量相关的口头或书面投诉进行记录和调查。</p><p>15.11 投诉记录应当包括以下内容：</p><dl><dd>- 投诉者的名称和地址；</dd></dl><dl><dd>- 投诉提交人的姓名(必要时包括职务)和电话；</dd></dl><dl><dd>- 被投诉事件的描述(包括原料药的名称和批号);</dd></dl><dl><dd>- 收到投诉的日期；</dd></dl><dl><dd>- 最初采取的措施(包括日期和实施者的身份);</dd></dl><dl><dd>- 后续采取的措施；</dd></dl><dl><dd>- 对初始投诉者的回复(包括回复发送的日期);及</dd></dl><dl><dd>- 对中间体或原料药批次的最终处理意见。</dd></dl><p>15.12 应当保存投诉记录，以评估其趋势、与产品相关的发生频率和严重性，以便采取进一步的纠正措施，必要时应当立即采取纠正措施。</p><p>15.13 应当有书面规程明确在何种情况下考虑召回中间体或原料药。</p><p>15.14 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AC%E5%9B%9E">召回</a> 规程应当明确参与评估的人员、如何启动召回、应当通知的人员以及召回产品的处理方式。</p><p>15.15 若遇到严重的或可能危及生命的情况，必须报告当地、国家和/ 或国际监管机构并寻求其建议。</p><h2>16 受托生产商(包括实验室)</h2><p>16.10 所有受托生产商(包括实验室)应当遵循本指南规定的GMP 要求。应当特别注意防止交叉污染及确保 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E8%BF%BD%E6%BA%AF">可追溯</a> 。</p><p>16.11 委托方应当对受托生产商(包括实验室)进行评估，确保受托方场地的相关操作符合 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">GMP</a> 要求。</p><p>16.12 合同双方之间应当有经批准的书面合同或正式协议，详细规定各方的GMP责任，包括质量措施。</p><p>16.13 合同应当规定委托方有权对受托方的厂房设施进行审计以确认其GMP符合性。</p><p>16.14 如允许分包，未经委托方事先评估和核准， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%97%E6%89%98%E6%96%B9">受托方</a> 不得将协议规定的受托工作分包给第三方。</p><p>16.15 生产和实验室记录应当保存在现场，易于查阅。</p><p>16.16 未经 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A7%94%E6%89%98%E6%96%B9">委托方</a> 同意或批准，受托方不得擅自变更工艺、设备、检测方法、质量标准或协议要求的其它内容。</p><h2>17 代理商、中间商、贸易商、经销商、分包装商和重新贴签方</h2><h2>17.1 适用性</h2><p>17.10 本节适用于除原生产企业之外的参与中间体或原料药的购销、持有、重新包装、重新贴签、处理、发运或贮存的任何一方。</p><p>17.11 所有的代理商、中间商、贸易商、经销商、分包装商、重新贴签方均应当遵循本指南中的GMP 要求。</p><h2>17.2 已发运的原料药和中间体的可追溯性</h2><p>17.20 代理商、中间商、贸易商、经销商、分包装商或重新贴签方应当确保所发运原料药和中间体的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E8%BF%BD%E6%BA%AF">可追溯</a> 性。应当保存并能够获取的文件包括：</p><dl><dd>- 原生产企业的身份</dd></dl><dl><dd>- 原生产企业的地址</dd></dl><dl><dd>- 订单</dd></dl><dl><dd>- 提货单(运输文件)</dd></dl><dl><dd>- 收货文件</dd></dl><dl><dd>- 原料药或中间体的名称或命名</dd></dl><dl><dd>- 生产企业的批号</dd></dl><dl><dd>- 运输和发运记录</dd></dl><dl><dd>- 所有真实可信的检验报告，包括原生产企业提供的报告</dd></dl><dl><dd>- 复验期或有效期</dd></dl><h2>17.3 质量管理</h2><p>17.30 代理商、中间商、贸易商、经销商、分包装商或重新贴签方应当按照第2节的要求建立、记录并实施有效的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E4%BD%93%E7%B3%BB">质量管理体系</a> 。</p><h2>17.4 中间体和原料药的重新包装、重新贴签和存放</h2><p>17.40 中间体和原料药的重新包装、重新贴签和存放应当按照本指南的规定，遵循必要的GMP 要求，防止原料药或中间体的混淆或影响其纯度。</p><p>17.41 重新包装应当在合适的环境条件下进行，防止污染和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a> 。</p><h2>17.5 稳定性</h2><p>17.50 原料药或中间体重新包装所用包装材料和容器如果与原生产商所用类型不同，应当进行稳定性考察，确定其 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%8D%E9%AA%8C%E6%9C%9F">复验期</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%89%E6%95%88%E6%9C%9F">有效期</a> 。</p><h2>17.6 信息传递</h2><p>17.60 代理商、中间商、贸易商、经销商、分包装商或重新贴签方应当将所有质量、监管信息在原料药或中间体生产企业和客户之间进行传递。</p><p>17.61 代理商、中间商、贸易商、经销商、分包装商或重新贴签方将原料药或中间体提供给客户时，应当提供原生产企业的名称和批号。</p><p>17.62 药品监管部门如有要求，代理商应当提供原料药或中间体的原生产企业的证明性文件。根据原生产企业与其授权代理商之间的法律关系，原生产企业可直接或通过其授权的代理商与药品监管部门联系。(“授权”指由生产企业授权)</p><p>17.63 应当遵守第11.4章节中有关检验报告的要求。</p><h2>17.7 投诉和召回的处理</h2><p>17.70 代理商、中间商、贸易商、经销商、分包装商或重新贴签方应当按照第15章中的要求保存收到的所有投诉和召回记录。</p><p>17.71 必要时，代理商、中间商、贸易商、经销商、分包装商或重新贴签方应当与原料药或中间体的原生产企业共同审核投诉，确定是否应当与购买该原料药或中间体的其他客户或者药品监管部门共同采取进一步措施。应当由合适的一方调查并记录 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%95%E8%AF%89">投诉</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AC%E5%9B%9E">召回</a> 的原因。</p><p>17.72 针对原料药或中间体原生产企业的投诉，代理商、中间商、贸易商、经销商、分包装商或重新贴签方所保存的记录中应当包含所有来自于原生产企业的反馈信息(包括日期和具体内容)。</p><h2>17.8 退货处理</h2><p>17.80 退货应当按照第14.52章节的要求处理。代理商、中间商、贸易商、经销商、分包装商或重新贴签方应当保存原料药和中间体退货文件。</p><h2>18 细胞培养/发酵生产原料药专用指南</h2><h2>18.1 通则</h2><p>18.10 本章专门对采用天然或重组组织进行细胞培养或发酵生产的原料药或中间体提出了未在之前章节充分阐明的特殊控制要求。本章并非独立存在章节。总体上，本指南其它章节中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:GMP">GMP</a> 原则通常也适用于本章节。需要说明是，生产小分子化合物的“传统”发酵工艺与采用重组与非重组组织生产蛋白质和/或多肽的发酵工艺本质相同，只是控制程度有所不同。生产蛋白质和多肽的生物技术工艺的控制要求通常要严于传统发酵工艺，本章对此作出了规定/阐述。</p><p>18.11 “生物技术工艺”(生物技术)指使用细胞或组织生产原料药，这些细胞或组织可通过DNA重组技术、杂交瘤技术或其它技术得到。生物技术工艺生产的原料药通常由蛋白质、多肽等高分子物质组成，本章提出了相应的具体要求。某些低分子原料药，如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%97%E7%94%9F%E7%B4%A0">抗生素</a> 、氨基酸、维生素和碳水化合物也可使用DNA重组技术生产，其控制要求与传统 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%91%E9%85%B5">发酵</a> 工艺生产的原料药相似。</p><p>18.12 “传统发酵”指用自然存在和/或用传统方法(如辐照和化学诱变)改良的微生物来生产原料药的工艺。传统发酵生产的原料药通常为小分子化合物，如抗生素，氨基酸、维生素和碳水化合物等。</p><p>18.13 细胞培养或发酵生产原料药或中间体涉及如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%86%E8%83%9E%E5%9F%B9%E5%85%BB">细胞培养</a> ，或从活体组织提取和纯化等生物工艺。生产工艺中也可能存在某些附加工艺步骤，如物理化学修饰。所用原材料(如培养基、缓冲组分)可能导致微生物污染物的生长。根据原料药或中间体的来源、制备方法和预期用途，必要时，在生产过程中需要控制生物负荷、病毒污染和/ 或内毒素并在工艺的适宜阶段对其进行监测。</p><p>18.14 生产的所有阶段均应当建立必要的控制措施，确保中间体和/或原料药的质量。尽管本指南从细胞培养/发酵步骤开始，但前期步骤(如细胞库建立)也应当在适当的工艺控制下进行。本指南涵盖了从细胞库中取得生产用细胞开始的细胞培养/发酵过程。</p><p>18.15 应当采用适当的设备和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%8E%AF%E5%A2%83%E6%8E%A7%E5%88%B6">环境控制</a> 将污染的风险减至最低。环境质量的可接受标准和监控频率取决于生产的步骤和生产条件(开放、封闭或限制系统)。</p><p>18.16 通常，工艺控制应当考虑以下内容：</p><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E4%BD%9C%E7%BB%86%E8%83%9E%E5%BA%93">工作细胞库</a> 的维护(如适用);</dd></dl><dl><dd>- 合理的接种和扩增培养；</dd></dl><dl><dd>- 发酵和细胞培养过程中关键操作参数的控制；</dd></dl><dl><dd>- 细胞生长过程，活力(对绝大多数细胞培养工艺而言)和产率的适当监控；</dd></dl><dl><dd>- 去除细胞、细胞碎片和培养基组份，同时防止污染(特别是微生物污染)中间体或原料药，或影响质量的收获和纯化步骤；</dd></dl><dl><dd>- 在适当的生产阶段监控生物负荷，必要时监控 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%86%85%E6%AF%92%E7%B4%A0">内毒素</a> 水平；</dd></dl><dl><dd>- <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q5A_R2_Viral_Safety_Evaluation_of_Biotechnology_Products_Derived_from_Cell_Lines_of_Human_or_Animal_Origin_20231101">ICH 指南Q5A</a> 生物技术产品质量：源自人类或动物细胞系的生物技术产品的病毒安全性评估中所述的病毒安全问题。</dd></dl><p>18.17 应当证明已去除培养基组分、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%AE%BF%E4%B8%BB%E7%BB%86%E8%83%9E">宿主细胞</a> 蛋白质、其它工艺相关杂质、产品相关杂质和污染物(如适用)。</p><h2>18.2 细胞库的维护和记录保存</h2><p>18.20 只有被授权人员方可进入细胞库。</p><p>18.21 细胞库应当在适当条件下保存，保持其活性和防止污染。</p><p>18.22 应当保存细胞库的细胞领用记录、贮存条件记录。</p><p>18.23 细胞库应当视情况进行周期性监控，确定其可使用。</p><p>18.24 有关细胞库的详细内容参见 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q5D_Derivation_and_Characterisation_of_Cell_Substrates_Used_for_Production_of_Biotechnological/Biological_Products">ICH指南Q5D</a> 生物技术产品的质量：用于生物技术产品/生物制品生产的细胞基质的来源和鉴定。</p><h2>18.3 细胞培养/发酵</h2><p>18.30 在无菌操作条件下添加细胞基质、培养基、缓冲液和气体，应当采用密闭或限制系统。初始容器接种、转种或加料(培养基、缓冲液)使用敞口容器操作的，应当有控制措施和程序，最大限度降低污染的风险。</p><p>18.31 当微生物污染对原料药质量有影响时，敞口容器的操作应当在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%AE%89%E5%85%A8%E6%9F%9C">生物安全柜</a> 中或其他适当的控制环境下进行。</p><p>18.32 操作人员应当穿着适宜的工作服，并在处理培养物时采取特殊的防护措施。</p><p>18.33 应当对关键操作参数(如温度、pH值、搅拌速度、通气量、压力)进行监控，保证与规定的工艺一致。应当对菌体生长、活力(对多数细胞培养工艺而言)进行监控，视情形还应当对产率进行监控。各类工艺的关键参数有所不同，对于传统发酵工艺，某些参数(如细胞活力)可能无需监控。</p><p>18.34 细胞培养设备使用后应当清洁和灭菌。发酵设备应当视情形清洁、消毒或灭菌。</p><p>18.35 必要时，培养基使用前应当灭菌，保障原料药的质量。</p><p>18.36 应当制定监测各工序微生物污染、所采取措施的操作规程，包括评估 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%AE%E7%94%9F%E7%89%A9%E6%B1%A1%E6%9F%93">微生物污染</a> 对产品质量的影响，确定消除污染使设备恢复到后续正常生产的条件。应当视情况对发酵过程中发现的外源微生物进行鉴别，必要时评估其对产品质量的影响。处理生产的物料时应考虑该评估结果。</p><p>18.37 应当保存污染事件的记录。</p><p>18.38 共用(多产品)设备更换品种生产时，应当视情况进行必要的额外检测，将交叉污染的风险降低到最低程度。</p><h2>18.4 收获、分离和纯化</h2><p>18.40 收获步骤无论是通过去除细胞或细胞碎片的方式还是通过在细胞破碎后收集细胞组分的方式，均应当在设计合理的设备和区域中进行，将污染风险降低到最低程度。</p><p>18.41 收获和纯化的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%93%8D%E4%BD%9C%E8%A7%84%E7%A8%8B">操作规程</a> 中，应当详细规定去除或灭活生产用微生物、细胞碎片和培养基组分(同时尽可能确保减少降解、污染和对质量的影响)的操作，确保中间体或原料药具有持续稳定的质量。</p><p>18.42 设备使用后应当适当清洁，必要时清毒。如不影响中间体或原料药质量，可多批连续生产后清洁。</p><p>18.43 纯化过程使用开放系统的，应当在能够保证产品质量的环境中进行。</p><p>18.44 如果设备用于多品种生产，必要时应当采取更多的控制措施，例如采用专用层析树脂或进行更多的检测。</p><h2>18.5 病毒去除/灭活步骤</h2><p>18.50 有关详细内容参见 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q5A_R2_Viral_Safety_Evaluation_of_Biotechnology_Products_Derived_from_Cell_Lines_of_Human_or_Animal_Origin_20231101">ICH Q5A</a> 生物技术产品质量：源自人类或动物细胞系的生物技术产品的病毒安全性评价。</p><p>18.51 病毒去除和病毒灭活步骤在某些工艺中属于关键步骤，应当在验证的参数范围内进行。</p><p>18.52 应当采取必要的措施，防止 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%97%85%E6%AF%92%E5%8E%BB%E9%99%A4%E7%81%AD%E6%B4%BB">病毒去除和灭活</a> 前后步骤中可能的病毒污染。敞口操作区应当与其它操作区分开，并设独立的空气净化系统。</p><p>18.53 同一设备通常不得用于不同的纯化操作。如果使用同一设备，重新使用前应当采取适当的清洁和消毒措施，采取适宜的措施防止病毒从之前的步骤带入(通过设备或环境)。</p><h2>19 临床试验用原料药</h2><h2>19.1 通则</h2><p>19.10 本指南前述章节中的控制条款并不完全适用于研发阶段供研究用的新原料药的生产。本节提供针对此种情况的特定指南。</p><p>19.11 临床试验用原料药的生产控制应与含有该 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的制剂的研发阶段相一致。工艺和检验程序应当灵活，使其可在随着工艺知识的积累及药品临床测试从临床前阶段进展到临床阶段之后进行变化。一旦药品开发进展到临床试验药品用原料药的生产阶段，生产商应确保原料药在合适的设施中，采用能保证原料药质量的适当的生产和控制程序生产。</p><h2>19.2 质量</h2><p>19.20 应当将GMP 理念应用于临床试验用原料药的生产，并建立适宜的各批次原料药的批准机制。</p><p>19.21 应当建立独立于生产部门的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E7%AE%A1%E7%90%86%E9%83%A8%E9%97%A8">质量管理部门</a> 以执行每一批临床试验用原料药的批准或拒绝放行。</p><p>19.22 某些通常由质量管理部门执行的检测工作可由其它部门完成。</p><p>19.23 质量措施应当包括一套原料、包装材料、中间体和原料药的检测系统。</p><p>19.24 应当对出现的工艺和质量问题进行评估。</p><p>19.25 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床试验</a> 用原料药的标签应当进行适当控制，标签上应当标明物料仅供研究使用。</p><h2>19.3 设备和设施</h2><p>19.30 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 的所有阶段中，包括使用小规模的设施或实验室来生产临床试验用原料药，应当有相关的规程以确保设备的校准、清洁并适用于其预定用途。</p><p>19.31 设施的使用程序应当确保物料的操作能够以将污染和交叉污染风险降至最低的方式进行。</p><h2>19.4 原料的控制</h2><p>19.40 应当通过检测对临床试验用的原料药生产所需的原料进行评估，或凭供应商的检验报告接收并进行鉴别试验。对于危险物料，有供应商的检验报告即可。</p><p>19.41 某些情况下，原料的适用性可在使用前通过小规模的反应(即，使用试验)来决定，而不仅仅依据分析测试。</p><h2>19.5 生产</h2><p>19.50 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床试验</a> 用的原料药的生产应当以实验记录本、批记录或其它合适的方法加以记录，这些记录应当包括所用生产物料、设备、工艺和科学观察。</p><p>19.51 与商业生产相比，预期产量可能变化更大且更不确定，因而不必对产量变化进行相关调查。</p><h2>19.6 验证</h2><p>19.60 临床试验用原料药的生产通常不适合做工艺验证，因为原料药往往只生产一批，或由于原料药开发阶段的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%8F%98%E6%9B%B4">工艺变更</a> ，使得批次间重现困难或不精确。可采用控制、校准和必要的设备确认等联合手段来保证开发阶段的原料药质量。</p><p>19.61 当所生产的批次供商业化用途时，即便以中试或小试规模生产，均应按照第12章的要求进行 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E9%AA%8C%E8%AF%81">工艺验证</a> 。</p><h2>19.7 变更</h2><p>19.70 随着知识的积累和生产规模的扩大，开发过程会有变更。生产、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 或检验规程的任何变更均应当充分记录。</p><h2>19.8 实验室控制</h2><p>19.80 虽然用于评估某一批临床试验用原料药的分析方法可能尚未验证，但所用分析方法应科学合理。</p><p>19.81 应当建立一套针对所有批次的留样系统。该系统应当能确保在一项申请被批准、终止或中断之后，将足够量的每个留样保存一段适当的时间。</p><p>19.82 第11.6节中关于有效期和复验期的规定适用于现有的临床试验用原料药。对于新的原料药，第11.6节的规定通常并不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床试验</a> 的早期阶段。</p><h2>19.9 文件</h2><p>19.90 应当建立一套系统确保临床试验用原料药的开发和生产过程中所获得的信息资料均有记录，并可随时查阅。</p><p>19.91 临床试验用原料药放行所用 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 的开发和实施应当有适当的记录。</p><p>19.92 应当采用一套保存生产和控制相关记录和文件的系统。该系统应当确保申请批准、终止或中断后，相关记录和文件能够得以保存一段适当的时间。</p><h2>20 术语</h2><p><b>可接受标准</b></p><p>检测结果的可接受的数值限度、范围或其它合适的测定结果。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a></b> (<b>API)</b></p><p>任何作为活性成分且用于医药产品生产的物质或混合物质。该类物质旨在预防、诊断、处置、缓解或治疗疾病中发挥药理作用或其它直接效用，或者影响人体的结构和机能。</p><p><b>原料药的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%B7%E5%A7%8B%E7%89%A9%E6%96%99">起始物料</a></b></p><p>原料药生产所用的、且以主要结构单元的形式被结合进原料药结构中的原料、中间体或原料药。原料药的起始物料可以是市售商品，也可以通过合同或商业协议从一个或多个供应商处购得，或者企业自行生产。原料药的起始物料一般有其特定的化学性质和结构。</p><p><b>批</b> （ <b>批次</b> ）</p><p>经一个或若干加工过程生产的、具有预期均一质量和特性的一定数量的物料。在连续生产情况下，批必须与生产中具有预期均一特性的确定数量的产品相对应，批量可以是固定数量或固定时间段内生产的产品量。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> (批次号)</b></p><p>用于识别一个批次的具有唯一性的数字、字母和/或符号的组合，用以确认生产和销售的历史。</p><p><b>生物负荷</b></p><p>可能存在于原料、原料药的起始物料、中间体或原料药中的微生物的水平和种类(如致病或非致病的微生物)。生物负荷不应当视为污染，除非其水平超标，或检测到规定的致病微生物。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%A0%A1%E5%87%86">校准</a></b></p><p>在适当的量程范围内，将某一特定仪器或装置产生的结果与参比或可溯源标准器的对应结果进行比较，以证明该仪器或装置产生的结果在规定的限度范围内的过程。</p><p><b>计算机系统</b></p><p>由一系列硬件和软件设计和组装而成，用于执行某种特定功能或某一组功能。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AE%A1%E7%AE%97%E6%9C%BA%E5%8C%96%E7%B3%BB%E7%BB%9F">计算机化系统</a></b></p><p>某计算机系统集成的流程或操作。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%B1%A1%E6%9F%93">污染</a></b></p><p>在生产、取样、包装或重新包装、贮存或运输等操作过程中， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E8%BE%85%E6%96%99">原辅料</a> 、中间产品、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%BE%85%E5%8C%85%E8%A3%85%E4%BA%A7%E5%93%81">待包装产品</a> 、成品受到具有化学或微生物特性的杂质或异物的不利影响。</p><p><b>受托生产商</b></p><p>受原生产商委托进行某些阶段生产的生产商。</p><p><b>关键的</b></p><p>用来描述为确保原料药符合其标准而必须控制在预定标准范围内的工艺步骤、工艺条件、测试要求或其他有关参数或项目。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%BA%A4%E5%8F%89%E6%B1%A1%E6%9F%93">交叉污染</a></b></p><p>不同物料或产品之间发生的相互污染。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%81%8F%E5%B7%AE">偏差</a></b></p><p>与已批准的指令或既定标准的偏离。</p><p><b>医药产品</b></p><p>已完成最终包装用于销售的医用制剂产品(参见 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q1A(R2)_Stability_Testing_of_New_Drug_Substances_and_Products">Q1A</a>)</p><p><b>原料药(DS)</b></p><p>参见 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药(API)</a></p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9C%89%E6%95%88%E6%9C%9F">有效期</a></b></p><p>标注于容器/标签上的日期，用以表明在规定的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%AE%E5%AD%98">贮存</a> 条件下，原料药仍可符合质量标准的保存期限，超过后则不应当使用。</p><p><b><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E8%BF%87%E7%A8%8B%E6%8E%A7%E5%88%B6">中间过程控制</a> (或过程控制)</b></p><p>生产过程中为监测、必要时调节工艺和/或保证中间体或原料药符合质量标准而进行的检查。</p><p><b>中间体/ <a <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%89%B9%E5%8F%B7">批号</a> 。</p><p><b>生产</b></p><p>原料药的物料接收、生产过程、包装、重新包装、贴签、重新贴签、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%8E%A7%E5%88%B6">质量控制</a> 、放行、贮存、发运在内的所有操作及相关控制。</p><p><b>物料</b></p><p>用以表示原料(<a <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8A%A9%E6%BB%A4%E5%89%82">助滤剂</a> 、 <a
