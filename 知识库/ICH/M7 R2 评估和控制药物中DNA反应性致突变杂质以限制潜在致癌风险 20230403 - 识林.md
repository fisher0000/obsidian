协调指导原则</b></center><center><b>评估和控制药物中 DNA 反应性（致突变）</b></center><center><b>杂质以限制潜在致癌风险</b></center><center><b>M7(R2)</b></center><h2>1. 引言</h2><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8E%9F%E6%96%99%E8%8D%AF">原料药</a> 的合成涉及反应物、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AF%95%E5%89%82">试剂</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BA%B6%E5%89%82">溶剂</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%82%AC%E5%8C%96%E5%89%82">催化剂</a> 和其它加工助剂的使用。因化学反应和终产物降解，所有原料药及制剂中均存在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9D%82%E8%B4%A8">杂质</a> 。 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A(R2)《新原料药中的杂质》</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3B(R2)_Impurities_in_New_Drug_Products.">Q3B(R2)《新药制剂中的杂质》</a> （参考文献 1、2）为大多数杂质的定性和控制提供了指导，但其对 DNA 反应性杂质的指导有限。</p><p>本指导原则旨在为 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%87%B4%E7%AA%81%E5%8F%98%E6%9D%82%E8%B4%A8">致突变杂质</a> 的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%89%B4%E5%88%AB">鉴别</a> 、分类、定性和控制提供一个可行的框架方案，以控制其潜在的致癌风险。本指导原则意在补充 ICH Q3A(R2)、Q3B(R2)（注释 1）和 <a href="https://lib.shilinx.com/wiki/index.php?title=M3(R2)_Guidance_on_Nonclinical_Safety_Studies_for_the_Conduct_of_Human_Clinical_Trials_and_Marketing_Authorization_for_Pharmaceuticals">ICH M3(R2)《支持药物进行临床试验和上市的非临床安全性研究指导原则》</a> （参考文献 3）。</p><p>本指导原则强调，为把致癌风险控制在可忽略的程度，制订致突变杂质的限度时，需同时兼顾安全性和质量 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%A3%8E%E9%99%A9%E7%AE%A1%E7%90%86">风险管理</a> 。本指导原则考虑了用药人群的具体情况，为原料药或制剂中残留或可能残留的致突变杂质进行评估和控制给出了建议。</p><h2>2. 指导原则的适用范围</h2><p>本指导原则旨在新原料药和新制剂 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 和后续上市申报阶段提供指导。它也适用于已上市药品的上市后申请、以及在已上市制剂中使用的原料药用于新制剂时的上市申请，在以上两种情况下仅限于：</p><ul><li>变更了原料药合成工艺，导致产生了新杂质或放宽了已有杂质限度；</li></ul><ul><li>变更了处方、组分或生产 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA">工艺</a> ，导致产生了新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%99%8D%E8%A7%A3%E4%BA%A7%E7%89%A9">降解产物</a> 或提高了已有降解产物限度；</li></ul><ul><li>变更了 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%80%82%E5%BA%94%E7%97%87">适应症</a> 或给药方案，显著影响了可接受的致癌风险水平。</li></ul><p>本指导原则所述对杂质潜在致突变性的评估不适用于以下类型的原料药和制剂：生物/ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E7%89%A9%E5%88%B6%E5%93%81">生物技术制品</a> 、肽类、寡核苷酸、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%94%BE%E5%B0%84%E6%80%A7%E8%8D%AF%E5%93%81">放射性药物</a> 、发酵产品、草药、动物或植物来源的粗制品。</p><p>本指导原则不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=S9_Nonclinical_Evaluation_for_Anticancer_Pharmaceuticals">ICH S9</a> （参考文献 4）所定义的拟用于晚期癌症适应症的原料药和制剂。另外，某些情况下，拟用于其它适应症的原料药本身在治疗浓度下就存在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%9F%BA%E5%9B%A0%E6%AF%92%E6%80%A7">遗传毒性</a> ，且预期可能会增加致癌风险。在这些情况下，暴露于致突变杂质不会显著增加原料药的致癌风险。因此，杂质可以按非致突变性杂质进行控制。</p><p>本指导原则所述对杂质潜在致突变性的评估不适用于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%9F%AB%E5%91%B3%E5%89%82">矫味剂</a> 、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%9D%80%E8%89%B2%E5%89%82">着色剂</a> 和香料，以及已上市药物中使用的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%BE%85%E6%96%99">辅料</a> 。本指导原则不适用于药物 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8C%85%E8%A3%85%E6%9D%90%E6%96%99">包材</a> 相关的可浸出物，但如有必要，可使用本指导原则中描述的限制潜在致癌风险的安全风险评估原则。如有必要，首次在药品中使用的化学合成辅料中的杂质，可采用本指导原则的安全风险评估原则。</p><h2>3. 总则</h2><p>本指导原则关注焦点为较低水平即可直接造成 DNA 损伤，进而导致 DNA 突变，因此可能引发癌症的 DNA 反应性物质。这类致突变性致癌物通常由 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:Ames%E8%AF%95%E9%AA%8C">细菌回复突变</a> （致突变性）试验检测确定。其它类型的非致突变性遗传毒性物质通常有阈值机制，以一般杂质水平存在时通常不会对人造成致癌风险。因此，为限制人暴露于潜在致突变杂质相关的患癌风险，使用细菌致突变试验来评估杂质致突变的可能性及控制的必要性。根据已有知识，基于结构的评估有助于预测细菌致突变试验结果。这种评估有多种方法，包括综述已有文献和/或计算机毒理学评估。</p><p>本文采用了 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:TTC">毒理学关注阈值</a> （threshold of toxicological concerns,<a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:TTC">TTC</a> ）的概念。TTC 的概念来定义无研究数据的化学物质在引起可忽略的致癌或其他毒性作用风险时的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E6%8E%A5%E5%8F%97%E6%91%84%E5%85%A5%E9%87%8F">可接受摄入量</a> 。。确定 TTC 的方法总体上可认为是非常保守的，它是将 50%肿瘤发生率（TD <sub>50</sub> ）的剂量简单线性外推到百万分之一发生率而得出，且采用的 TD <sub>50</sub> 数据取自最敏感种属和对诱导肿瘤最敏感部位。在使用 TTC 评估原料药和制剂中致突变杂质的可接受限度时，宜采用 1.5 μg/天的杂质限度，对应理论上额外增加 10 <sup>-5</sup> 的终生患癌风险。已确定有些结构基团具有较高的致癌性，即使摄入量低于 TTC 水平，理论上仍会具有高致癌风险。这类高效致突变致癌物，被称为“关注队列”，包括黄曲霉毒素类、N-亚硝基化合物，以及烷基-氧化偶氮基化合物。</p><p>早期 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 阶段由于整体研发经验有限， <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8E%A7%E5%88%B6%E7%AD%96%E7%95%A5">控制策略</a> 和控制方法可能不完善。本指导原则基于已建立的风险评估策略制订致突变杂质的可接受摄入量。在早期研发阶段，将可接受风险设定为一个理论计算值，即致癌风险的增加约为百万分之一。 在研发后期及上市后，将可接受风险设定为一个理论计算值，即致癌风险的增加约为十万分之一。与人终生罹患各类癌症的概率（大于三分之一）相比，上述风险控制水平仅为理论上患癌风险的微小增加。需注意的是，现确立的癌症风险评估是基于终生暴露的情形建立的。在研发和上市期间短于终生（less than lifetime, LTL）的暴露时杂质可以有更高的可接受限度，并仍保持相同的风险水平。使用量化的患癌风险值（十万分之一），并将其转化为基于风险的剂量（TTC），是一种高度假设性的方法，不应作为实际风险的真实数值。尽管如此，TTC 的概念为致突变性化合物提供了一个安全 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9A%B4%E9%9C%B2%E9%87%8F">暴露量</a> 的估值。但是，考虑到推导 TTC 值时采用了保守的假设，超出 TTC 值并不一定会增加患癌风险。实际增加的癌症发生率远低于十万分之一。此外，如果致突变化合物在啮齿动物 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%87%B4%E7%99%8C%E8%AF%95%E9%AA%8C">致癌试验</a> 中显示为非致癌物，则预计致癌风险不会增加。基于上述这些原因，患者已暴露于某杂质，该杂质后来被鉴定为致突变的杂质时，患者的患癌风险不一定因此增加。应进行风险评估来决定是否需要采取进一步措施。</p><p>如果确定某杂质具有潜在风险，则需根据对工艺的理解开发适当的控制策略和/或分析控制方法，以确保致突变杂质不高于可接受的癌症风险水平。</p><p>某些情况下，某杂质同时也是原料药的代谢产物。此时对代谢产物的致突变风险评估可作为该杂质限度制订的依据。</p><h2>4. 已上市药品的注意事项</h2><p>本指导原则初衷不是用于回顾性分析（即本指导原则采纳前已上市的药品）。但是，一些上市后变更需重新评估致突变杂质相关的安全性。本节内容适用于在本指导原则采纳之前或之后药品的上市后变更。第 8.5 节（ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 管理）包含了本指导原则采纳后的上市药品的其它建议。</p><h2>4.1 原料药 CMC 部分的上市后变更</h2><p>原料药涉及 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:CMC">CMC</a> 的上市后变更，对 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%B7%E5%A7%8B%E7%89%A9%E6%96%99">起始物料</a> 之后的部分如合成路线、试剂、溶剂或工艺条件的变更可能对致突变杂质的潜在风险产生影响，申报资料中应对这些影响进行评估。尤其是应评估变更是否会产生任何新的致突变杂质或放宽了已知致突变杂质的可接受限度。对不受变更影响的杂质建议不再重新评估。例如，如果只变更部分生产工艺，则致突变杂质的风险评估应局限于该变更是否会产生新的致突变杂质，在受影响的步骤中致突变杂质是否增加，以及来自上游步骤中的已知致突变杂质的水平是否升高。与这些变更相关的申报参照第 9.2 节内容。变更原料药、中间体或起始物料的生产场地，或变更原材料供应商，不需要重新评估致突变杂质风险。</p><p>如拟提交一个新的原料药供应商，如有充分证据证明该供应商生产该原料药采用了与审评区域内已上市药品中所用的原料药相同 的合成路线，则可以说明关于致突变杂质的风险/获益是可以接受的，不需要根据本指导原则进行评估。如果不是这种情况，则需要根据本指导原则进行评估。</p><h2>4.2 制剂 CMC 部分的上市后变更</h2><p>制剂的上市后变更（例如组分、生产工艺、剂型的变更），应包括对新致突变 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%99%8D%E8%A7%A3%E4%BA%A7%E7%89%A9">降解产物</a> 或提高已知致突变降解产物的可接受限度相关的潜在风险进行评估。必要时，申报资料应包括更新后的控制策略。如果制剂所用的原料药没有变化，则不建议也不要求对该原料药重新进行评估。制剂生产场地变更不需要重新评估致突变杂质风险。</p><h2>4.3 已上市药品的临床用途变更</h2><p>变更已上市药品的临床用途时，需重新评估致突变杂质限度的情形包括临床使用剂量的显著增加、服药时间的增加（特别是先前适应症将致突变杂质限度设定为高于终生使用时的可接受摄入量，但该摄入量可能已不再适合于更长给药周期的新适应症），或者适应症由严重或危及生命的疾病改为不太严重的疾病，之前杂质可接受摄入量（第 7.5 节）较高，可能已不再适合新适应症。与新 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BB%99%E8%8D%AF%E9%80%94%E5%BE%84">给药途径</a> 或扩大患者人群（包括孕妇和/或儿童）相关的已上市药品临床用途的变更，如日剂量或服药时间没有增加，则不需要重新评估致突变杂质风险。</p><h2>4.4 已上市药品的其他注意事项</h2><p>如果存在需要特别关注的情形，本指导原则也可用于已上市的药品。除非杂质有“关注队列”（第 3 节）中的结构，否则仅凭杂质存在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%AD%A6%E7%A4%BA%E7%BB%93%E6%9E%84">警示结构</a> 不足以启动后续措施。但是一种需关注的情形是，在已建立了上市所需总体控制策略和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B4%A8%E9%87%8F%E6%A0%87%E5%87%86">质量标准</a> 后，获得了新的相关杂质的危害性数据（分为第 1 类和/或 2 类，第 6 节）。这些新的相关杂质的危害性数据应来自符合相关法规、指导原则的高质量的科学研究，且可提供试验数据或报告。同样地，已上市药品中发现新的 1 类或 2 类致突变杂质，也属于一种需要关注的情形。上述两种情形下，一旦 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%B3%E8%AF%B7%E4%BA%BA">申请人</a> 知晓这些新的信息，即需要遵循本指导原则进行评估。</p><h2>5. 原料药和制剂杂质评估</h2><p>应对新原料药合成和贮藏期间以及新制剂生产和贮藏期间实际产生的和潜在的杂质进行评估。</p><p>杂质评估包含两个阶段：</p><ul><li>已鉴定的实际存在的杂质应考虑其潜在致突变性。</li></ul><ul><li>对可能出现在最终原料药中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BD%9C%E5%9C%A8%E6%9D%82%E8%B4%A8">潜在杂质</a> 应该进行分析，以确定是否需要进一步评估其潜在致突变性。</li></ul><p>合成杂质和降解产物分别按照第 5.1 节和第 5.2 节描述的步骤遵照执行。</p><h2>5.1 合成杂质</h2><p>实际杂质包括原料药中观察到的超出 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A</a> <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%A5%E5%91%8A%E9%99%90">报告限度</a> 的杂质。如果实际杂质水平超过了 ICH Q3A 中所述的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%89%B4%E5%AE%9A%E9%99%90">鉴定阈值</a> ，则应进行鉴定。应当注意，有些低于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%89%B4%E5%AE%9A%E9%99%90">鉴定限度</a> 的杂质可能也已经进行了鉴定。</p><p>原料药中潜在杂质包括从起始原料到原料药的合成路线中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%B7%E5%A7%8B%E7%89%A9%E6%96%99">起始物料</a> 、试剂和中间体。</p><p>应对杂质残留到原料药中的风险进行评估，包括起始原料和中间体中 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%B2%E9%89%B4%E5%AE%9A%E7%9A%84%E6%9D%82%E8%B4%A8">已鉴定的杂质</a> ，以及从起始物料到原料药的合成路线中合理预测产生的副产物杂质。由于有些杂质被带入原料药的风险可以忽略（例如，很长的合成路线中较早的合成步骤中的杂质），因此，可以在工艺某一个步骤之后进行潜在致突变性杂质的评估，并应提供确定这一步骤的风险依据。</p><p>对于在原料药合成路线后期才引入的起始物料（并且起始物料合成路线已知），应评估起始物料合成的最后几步中的潜在致突变杂质。</p><p>结构已知的实际杂质和如上所述的潜在杂质应按第 6 节所述评估其潜在致突变性。</p><h2>5.2 降解产物</h2><p>原料药中实际的降解产物包括原料药在拟定的长期贮藏条件下以及带 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%86%85%E5%8C%85%E8%A3%85">内包装</a> 和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%A4%96%E5%8C%85%E8%A3%85">外包装</a> 贮藏期间观察到的超过ICH Q3A <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8A%A5%E5%91%8A%E9%99%90">报告限度</a> 的降解产物。制剂中实际的降解产物包括制剂在拟定的长期贮藏条件下以及带内包装和外包装贮藏期间观察到的高于ICH Q3B 报告限度的降解产物，还包括制剂生产过程中产生的杂质。如果实际降解产物水平超过 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3A(R2)_Impurities_in_New_Drug_Substances">ICH Q3A</a> / <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q3B(R2)_Impurities_in_New_Drug_Products.">Q3B</a> 所述的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%89%B4%E5%AE%9A%E9%99%90">鉴定限度</a> ，则应进行鉴定。应当注意，有些低于鉴定限度的降解产物可能也已经被鉴定。</p><p>原料药和制剂中潜在的降解产物是在长期贮藏条件下预期可能会生成的杂质。潜在的降解产物包括 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8A%A0%E9%80%9F%E7%A8%B3%E5%AE%9A%E6%80%A7%E8%AF%95%E9%AA%8C">加速稳定性研究</a> 中（例如，40°C/ 相对湿度 75% 6 个月）和 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q1B_Photostability_Testing_of_New_Drug_Substances_and_Products">ICH Q1B</a> （参考文献 5）所述的验证性光稳定性研究期间所生成且高于 ICH Q3A/Q3B 鉴定限度的杂质，但这些杂质在原料药和制剂带内包装长期贮藏条件下尚未产生。</p><p>降解途径的相关知识，例如化学降解原理、相关强制降解试验和研发期间的稳定性，可用来指导选择需评估的有致突变性的潜在降解产物。</p><p>实际的和潜在的降解产物可能存在于最终的原料药或制剂中，并且在结构已知的情况下，应按照第 6 节所述评估致突变性。</p><h2>5.3 临床研发的注意事项</h2><p>第 5.1 和第 5.2 节所述的杂质评估适用于临床研发中的产品。然而，众所周知，这一阶段可获得的信息是有限的。例如，在临床研发期间，可能无法获得长期稳定性和光照稳定性研究信息，因此关于潜在降解产物的信息可能很有限。此外，ICH Q3A/Q3B 中列出的杂质限度并不适用于临床阶段的产品，因此此时鉴定出的杂质较少。</p><h2>6. 危害性评估因素</h2><p>危害性评估首先通过对数据库和文献的检索获得的杂质致癌性和细菌致突变数据对实际和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%BD%9C%E5%9C%A8%E6%9D%82%E8%B4%A8">潜在杂质</a> 进行初步分析，根据表 1 将其归为 1 类，2 类或 5 类。如果无法获得这样的分类数据，则应进行预测细菌致突变性的构-效关系（SAR）评估。根据评估结果将杂质归为 3 类、4 类或 5 类。</p><div><table><caption><b>表 应采用（定量）构-效关系（(Q)SAR）方法学进行毒性评估，目的是预测细菌致突变试验（参考文献 6）的结果。应采用两种互补的(Q)SAR 预测方法。一种方法应基于专家知识规则，另一种方法应基于统计学。(Q)SAR 模型采用的这些预测方法学应遵循经济合作与发展组织（OECD）制订的一般的验证原则。</p><p>如果经两种互补的(Q)SAR 方法（专家知识规则和统计学）预测均没有警示结构，则足以得出该杂质没有致突变性担忧的结论，不建议做进一步的检测（归为表 1 中的 5 类）。</p><p>对计算机系统得到的任何阳性、阴性、相互矛盾或无法得出结论的预测结果，如有必要，申请人可根据专业知识进行综合评估，提供进一步支持性证据，合理论证并得出最终结论。</p><p>对具有警示结构（表 1 中的 3 类）的杂质可采取充分的控制措施，或者对该杂质单独进行细菌致突变试验。如果杂质规范的细菌致突变试验（注释 2）结果为阴性，则可以推翻任何基于结构的担忧，不建议进行进一步遗传毒性评估（注释 1）。这些杂质视为非致突变杂质（表 1 中的 5 类）。如果细菌致突变试验为阳性，则需要进行进一步的危害评估和/或采取控制措施（表 1 中的 2 类）。例如，如果杂质水平不能控制在一个适当的可接受限度，则建议进行体内基因突变试验，以了解在体内环境下细菌致突变试验结果的相关性。其它体内遗传毒性试验的选择应根据杂质的反应机理和预期靶组织暴露 （注释 3）的知识进行科学地论证。体内试验的设计应参考现行 ICH 遗传毒性试验相关指导原则。符合要求的体内测试结果可支持 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%89%B9%E5%AE%9A%E6%9D%82%E8%B4%A8">特定杂质</a> 限度的设定。</p><p>与原料药或相关化合物具有相似的警示结构（例如，在相同位置和相同化学环境下具有相同警示结构）的杂质，如果这些原料药或相关化合物细菌致突变试验结果为阴性， 则可视为非致突变杂质（表 1 中的 4 类）。</p><h2>7. 风险表征</h2><p>根据第 6 节所述的危害评估方法，每个杂质应按表 1 中的 5 个类别进行分类。本节介绍了推导 1、2、3 类杂质 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%AF%E6%8E%A5%E5%8F%97%E6%91%84%E5%85%A5%E9%87%8F">可接受摄入量</a> 的风险表征原则。</p><h2>7.1 基于 TTC 制定可接受摄入量</h2><p>基于 TTC 计算可接受摄入量时，单个致突变杂质每人每天摄入 1.5 μg 时其风险认为是可以忽略的（终生暴露情况下理论额外致癌风险小于十万分之一），这一值可以通用于大部分药物，作为可接受控制限度的默认值。TTC 方法一般用于长期治疗用（&gt;10 年）药物中存在且无致癌性数据（2 类和 3 类）的致突变杂质。</p><h2>7.2 基于特定化合物的风险评估数据制定可接受摄入量</h2><h3>7.2.1 具有阳性致癌性数据的致突变杂质（表 1 中的 1 类）</h3><p>如果有足够的致癌性数据，则应采用该特定化合物的风险评估数据来推导可接受摄入量，而非基于 TTC 的可接受摄入量。对于已知的致突变致癌物，其可接受摄入量可以根据致癌性强度计算，通常采用线性外推法计算。或者也可以采用其它已确认的风险评估方法。例如，可使用国际公认机构所采用的方法计算可接受摄入量，或直接使用监管机构已公布的测定数据（注释 4）。</p><p>对于与已知的某类致癌化合物在化学结构上相似的杂质，可以具体问题具体分析，对该特定化合物的可接受摄入量按照相似的致癌化合物的可接受摄入量进行计算，但前提是必须提供该杂质与已知化合物化学结构相似的理由以及支持性数据。（注释 5）。</p><h3>7.2.2 有实际阈值证据的致突变杂质</h3><p>现已逐渐认识到，一些作用机理可导致非线性或有阈值的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%89%82%E9%87%8F%E5%8F%8D%E5%BA%94">剂量反应</a> 关系。这包括与非 DNA 靶点相作用的化合物，以及 DNA 反应活性化合物，其效应可能存在一些调节机制，例如，在与 DNA 接触前被快速去除毒性，或产生的损伤可被有效地修复。。对这些化合物的监管，在可获得相关数据的前提下，可通过 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:NOEL">未观察到作用水平</a> （No Observed Effect Level, <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:NOEL">NOEL</a> ） 和使用不确定性因子（ 参见 <a href="https://lib.shilinx.com/reader/filedownload?name=Q3C_R5_Step4.pdf&amp;mode=view">ICHQ3C(R5)</a> ，参考文献 7）来计算 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">允许日暴露量</a> （permissible daily exposure, <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:PDE">PDE</a> ）。</p><p>化合物特定的经风险评估推算的可接受摄入量（第 7.2 节）可按后续章节（7.3.1 和 7.3.2）所述比例根据更短的使用时间进行调整，或应控制在不超过 0.5%，二者取较低者。例如，如果终生暴露时某化合物可接受摄入量为 15 μg/天，短于终生暴露时限度（表 2）可以增加至 100 μg（&gt;1-10 年治疗时长），200 μg（&gt;1-12 个月）或 1200 μg（&lt;1 个月）。但是，对于最大日服用剂量已知的药物-例如最大日服用剂量 100 mg，则&lt;1 个月时长的每日可接受摄入量应限制在 0.5%（500 μg），而不是 1200 μg。</p><h2>7.3 与短于终生（LTL）暴露的可接受摄入量</h2><p>已知致癌物的标准风险评估方法假定了癌症风险与累积剂量呈正相关。因此，终生以低剂量持续给药与在较短时间内给予相同累积 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%9A%B4%E9%9C%B2%E9%87%8F">暴露量</a> 的药物在患癌风险上是等同的。</p><p>基于 TTC 的可接受摄入量 1.5 μg/天被认为是安全的终生日暴露量。药品中致突变杂质的 LTL 暴露量可理解为可接受的累积终生摄入量（1.5 μg/天×25,550 天=38.3 mg）在 LTL 期间均匀分配在总暴露天数中。即允许致突变杂质日摄入量高于终生暴露时的日摄入量，对每日给药和非每日给药的剂量方案仍保持类似的风险水平。表 2 是从上述概念推导而得的数据，说明了临床研发阶段和上市阶段 LTL 个月以及超过 1 年的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%B4%E5%BA%8A%E7%A0%94%E7%A9%B6">临床研究</a> 直至完成Ⅲ期临床试验，推荐了致突变杂质的可接受摄入量（表 类，以及被列入关注队列的杂质控制在第 7 节所述的可接受限度内（参见 8 节）。所有其它杂质按非致突变杂质控制，其中包括 3 类杂质，因 I 期临床持续时间有限，无需进行评估。</p><h3>7.3.2 已上市药品</h3><p>对于已上市药品，绝大多数患者可按照表 2 中根据治疗时间确定可接受摄入量。所拟定的摄入量以及应用这些摄入量的各种情形均在注释 7 表 4 中进行了说明。在某些情况下，患者中一部分人群可能会延长治疗时间至超出该类药品预期使用时间的上限（例如，可接受摄入量为 10 μg/天的药品治疗时间超出 10 年，可能达到 15 年）。与大部分治疗 10 年的患者计算所得的整体风险相比，可以忽略治疗时间延长所导致的风险增加（如上例， 患癌风险增加至1.5/100000）。</p><h2>7.4 多个致突变杂质的可接受摄入量</h2><p>根据 特例和方法灵活性</h2><ul><li>如果某杂质通过其它途径在人体中的暴露量更大，如食品或内源性代谢物（例如甲醛），则更高的可接受摄入量可能是合理的。</li></ul><ul><li>在严重疾病、预期寿命缩短、迟发的慢性疾病或治疗选择有限的特殊情况下，可根据具体情况制订合理的可接受摄入量。</li></ul><ul><li>致突变物中某些结构的化合物存在高致癌风险（“关注队列”），即黄曲霉毒素类物质、N-亚硝基化合物、以及烷基-氧化偶氮结构化合物。如果药物中存在这些杂质，则这些高致癌风险化合物的可接受摄入量可能会显著低于本指导原则中定义的可接受摄入量。尽管仍可以使用本指导原则的策略，但是，通常应针对各案情况开发研究方法，例如，使用结构紧密关联的化合物的致癌数据（如果有）来证明制剂研发阶段和上市产品中可接受摄入量的合理性。</li></ul><p>第 7 节所述的以上风险控制方法适用于所有的给药途径，通常不需要对可接受摄入量进行修正。数据表明特定给药途径可能影响可接受摄入量时，应根据具体情况进行逐案评估。由于采用的风险评估方法较为保守，上述方法适用于所有患者人群。</p><h2>8. 控制</h2><p><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%8E%A7%E5%88%B6%E7%AD%96%E7%95%A5">控制策略</a> 是一套基于对当前产品和对工艺的理解而制订的有计划的控制方法，用以保证工艺性能和产品质量（ <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q10_Pharmaceutical_Quality_System">ICH Q10</a> ，参考文献 8）。控制策略包括但不限于以下内容：</p><ul><li>物料属性控制（包括原料、起始物料、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E4%BA%A7%E5%93%81">中间体</a> 、试剂、溶剂、内包材）；</li></ul><ul><li>设施和设备操作条件；</li></ul><ul><li>生产工艺设计中隐含的控制；</li></ul><ul><li><a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E8%BF%87%E7%A8%8B%E6%8E%A7%E5%88%B6">过程控制</a> （包括过程检测和 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%B7%A5%E8%89%BA%E5%8F%82%E6%95%B0">工艺参数</a> ）；</li></ul><ul><li>原料药和制剂的控制（如 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%94%BE%E8%A1%8C">放行</a> 检测）。</li></ul><p>如果一个杂质已经定性为表 1 中的 1、2 或 3 类杂质，为了保证该杂质在原料药和制剂中的水平低于可接受限度，构建一个专门的控制策略是至关重要的。构建适当的控制策略的基础是要全面理解原料药生产工艺相关的化学信息、制剂生产工艺以及原料药和制剂的整体 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%A8%B3%E5%AE%9A%E6%80%A7">稳定性</a> 。制剂中致突变杂质的控制策略与 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q9_Quality_Risk_Management">ICH Q9</a> （参考文献 9）中描述的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%A3%8E%E9%99%A9%E7%AE%A1%E7%90%86">风险管理</a> 过程应一致。基于对产品和工艺的理解以及风险管理原则，控制策略将工艺设计和控制与合适的分析测试相结合，可将控制点向上游移动并为尽量减少 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E6%88%90%E5%93%81">终产品</a> 测试提供可能性。</p><h2>8.1 工艺相关杂质的控制</h2><p>开发原料药控制策略有 4 种可能的方法：</p><p><b>方法 1</b></p><p>原料药质量标准中包含杂质检查项，采用合适的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%88%86%E6%9E%90%E6%96%B9%E6%B3%95">分析方法</a> 将杂质控制在可接受限度以内。</p><p>方法 1 的控制方式可以根据 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6A_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_New_Drug_Substances_and_New_Drug_Products:_Chemical_Substances_199910">ICH Q6A</a> （参考文献 10）进行定期确认性检测。如果在至少 6 个连续的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E8%AF%95">中试</a> 批次或 3 个连续的生产批次中，原料药中的致突变杂质水平均低于可接受限度的 30%，则可证明定期确认性检测是合理的。如果不满足该条件，则建议作为原料药质量标准中的常规检测项。更多考虑事项请参见第 8.3 节。</p><p><b>方法 2</b></p><p>在原料、起始物或中间体的质量标准中包含杂质检查项，或作为过程控制项，采用合适的分析方法将杂质控制在可接受限度以内。</p><p><b>方法 3</b></p><p>在原料、 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E8%B5%B7%E5%A7%8B%E7%89%A9%E6%96%99">起始物料</a> 或 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E4%B8%AD%E9%97%B4%E4%BA%A7%E5%93%81">中间体</a> 的质量标准中对杂质进行检测，或进行过程控制，制订一个高于原料药中该杂质可接受限度的标准，使用合适的分析方法并结合对杂质去向和清除的认知，及相关的工艺控制，保证原料药中的杂质的水平低于可接受限度而无需在后续工艺中再行检测。</p><p>对实验室规模试验（鼓励采用掺杂实验）的数据进行分析（必要时可以采用中试规模或商业规模批次数据加以佐证），如果原料药中杂质水平低于可接受限度的 30%，则采用该方法是合适的。见案例 1 和 2。也可以用其他方法来证明方法 3 的合理性。</p><p><b>方法 4</b></p><p>明确工艺参数及其对残留杂质水平（包括去向和清除知识）的影响，确信原料药中的杂质水平将会低于可接受限度，则建议无需对该杂质进行分析检测（即不需要将杂质订入任何质量标准中）。</p><p>如已充分了解对致突变杂质水平有影响的合成工艺和工艺参数，最终原料药中这些杂质残留量高于可接受限度的风险很小，则可用针对合成工艺的控制策略代替分析检测。很多情况下只需根据科学原理对控制方法进行论述即可。可以使用科学风险评估要素来论证 方法 4 的使用。风险评估可基于 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%90%86%E5%8C%96%E7%89%B9%E6%80%A7">物理化学性质</a> 和影响杂质去向和清除的工艺参数，如化学反应性、溶解性、挥发性、电离度和任何设计 用于去除杂质的物理过程。该风险评估的结果可表示为该工艺清除该杂质的预估清除因子（参考文献 11）。</p><p>方法 4 特别适用于那些自身不稳定的杂质（例如，与水迅速完全反应的二氯亚砜），或那些在合成路线早期引入并可被有效清除的杂质。</p><p>有些情况下，如果是成因已知的杂质或合成后期引入的杂质，也可采用方法 4；但此时需要提交针对该工艺的数据来论述采用该方法的合理性。</p><h2>8.2 控制方法的注意事项</h2><p>与采用方法 3 时一样，采用方法 4 时，如仅根据科学原理来进行论述是不够充分的，需提交分析数据来支持控制方法。提交的资料可包括：在下游化学反应中导致杂质结构改变的相关信息（去向）、中试批次的分析数据，某些情况下可提供实验室规模下加样研究（掺杂试验）结果。在这些情况下，证明该杂质的去向/清除的论据的可靠性是至关重要的，它能够自始至终地保证最终原料药中杂质残留量超过可接受限度的可能性可以忽略。如果清除因子是根据研发数据得到的，须注意评估其规模依赖性或非依赖性。如果用于研发阶段的小规模模型被认为不能代表商业规模，则需要确认中试规模和/ 或初始商业批次中所用的控制是适当的。根据由实验室或中试规模数据计算所得的清除因子大小、杂质引入步骤和对下游工艺清除工艺的理解，决定是否需要中试/商业批次数据。</p><p>如果使用方法 3 和 4 不能得到合理论证，则应在原料、起始物料或中间体的质量标准中对杂质进行检测或进行过程控制（方法 2）；或在原料药的质量标准中对杂质进行检测（方法 1），将杂质控制在可接受限度内。对于在最终步骤中引入的杂质，除另有论述外，一般应采用方法 1 的控制方法。</p><p>如果致突变杂质的水平低于可接受限度，则不必应用“合理可行的最低程度”（as low as reasonably practicable, ALARP）原则。同样地，也不必证明已摸索过可替代的合成路线。</p><p>如果通过控制仍不能将致突变杂质的水平降低至可接受限度以内，而杂质是“合理可行的最低程度”水平，则可根据风险/收益分析来论证更高限度的合理性。</p><h2>8.3 定期检测的注意事项</h2><p>上述这些方法中包括了建议将检测订入质量标准，但可能不必对每批产品均进行常规放行检测的情形。这种方法在 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6A_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_New_Drug_Substances_and_New_Drug_Products:_Chemical_Substances_199910">ICH Q6A</a> 中被称为定期检测或间隔检测，也可以称为“定期确认性检测”。如果能证明杂质形成/引入之后的工艺能清除该杂质，则该方法也是合理的。要注意的是，是否允许使用定期确认性检测取决于是否使用受控状态下的工艺（即，生产出的产品质量能持续满足质量标准，并符合已建立的合适的设施、设备、工艺和操作控制方案）。如果检测结果显示，致突变杂质的水平不符合定期检测所建立的可接受标准，则药品生产商要立即实施全面检测（即对每个批次的特定属性进行检测）直至最终找出失败的原因、实施了 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BA%A0%E6%AD%A3%E6%8E%AA%E6%96%BD%E5%92%8C%E9%A2%84%E9%98%B2%E6%8E%AA%E6%96%BD">纠正措施</a> ，并且书面证明该工艺已再次处于受控状态。正如 ICH Q6A 中所述，如果定期确认性检测失败，应通知监管机构评估先前放行未经检测的批次的风险/收益。</p><h2>8.4 降解产物的控制</h2><p>对于已经定性为具有致突变性的潜在降解产物，需了解其降解途径与原料药和制剂的生产工艺和/或其拟定的包装和贮藏条件的相关性。建议采用一个设计良好的在拟定的包装中的 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8A%A0%E9%80%9F%E7%A8%B3%E5%AE%9A%E6%80%A7%E8%AF%95%E9%AA%8C">加速稳定性试验</a> （例如，40°C/75%相对湿度，6 个月），采用适当的分析方法来确定潜在降解产物的相关性。也可以采用一个设计良好的动力学等效的时间更短、温度更高的稳定性试验，使用拟定的商业包装，在开始 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%95%BF%E6%9C%9F%E7%A8%B3%E5%AE%9A%E6%80%A7%E8%AF%95%E9%AA%8C">长期稳定性试验</a> 前确定降解途径的相关性。对于那些依据潜在降解途径的知识推断应该存在，但在产品中尚未观察到的潜在降解产物，这类研究对明确其相关性尤其适用。</p><p>根据这些加速试验的结果，如预计降解产物将在拟定包装和贮藏条件下形成，且接近可接受限度水平，则应采取措施控制降解产物的形成。这种情况下，除另有论述外，应在拟定贮藏条件下（采用拟定的商业包装），监测原料药或制剂中长期稳定性研究中的降解产物。该致突变降解产物的质量标准限度是否合理通常取决于这些稳定性研究的结果。</p><p>如果预计制剂研发和包装设计的选择不能将致突变降解产物水平控制在可接受限度以下，而杂质是“合理可行的最低限度”水平， 则可以根据风险/收益分析来论证更高限度的合理性。</p><h2>8.5 生命周期管理</h2><p>本部分适用于本指导原则颁布后批准的药品。</p><p><a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q10_Pharmaceutical_Quality_System">ICH Q10</a> 所述的质量体系要素和管理职责意在鼓励在 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%94%9F%E5%91%BD%E5%91%A8%E6%9C%9F">生命周期</a> 各阶段使用基于科学和基于风险的方法以促进药品在整个生命周期中持续改进。产品和工艺知识应从研发阶段开始进行管理，贯穿产品的整个商业化生命周期，直至产品退市。</p><p>对原料药和制剂的生产工艺的研发和改进一般会在其生命周期中持续进行。生产工艺性能，包括控制策略的有效性，应定期进行评估。从商业化生产中所获得的知识可用于进一步提高对工艺的理解和工艺性能，并对控制策略进行调整。</p><p>拟对生产工艺的任何变更均应评估其对原料药和制剂质量产生的影响。该评估应当基于对生产工艺的理解，并应明确是否需要适当的检测来分析拟定变更的影响。另外，对分析方法的改进可能促进对杂质进行结构鉴定。在这种情况下，需要按本指导原则的要求对新结构的杂质致突变性进行评估。</p><p>药品整个生命周期中，当工艺发生了计划内或计划外变更时，需重点重新评估是否需进行检测。这适用于对可接受限度没有常规监控的情况（方法 3 或方法 4 的控制方法），或采用定期检测而不是逐批检测的情况。该项检测应在生产工艺中适当的中控点进行。</p><p>在某些情况下，应用统计过程控制和工艺参数测量值的变化趋势有益于改善工艺持续适用性和工艺控制能力，从而对杂质控制更加充分。即便不对杂质进行常规监控（例如方法 4），统计过程控制也可以基于对杂质形成或清除有影响的工艺参数进行。</p><p>所有的变更应作为质量体系（ <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q10_Pharmaceutical_Quality_System">ICH Q10</a> ）的一部分遵循内部 <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E5%8F%98%E6%9B%B4%E7%AE%A1%E7%90%86">变更管理</a> 流程。对已批准的申报资料中的内容进行变更应根据当地法规和指导原则的要求向监管机构进行报告。</p><h2>8.6 临床研发的注意事项</h2><p>众所周知，对产品和工艺的认知在研发过程中是不断增长的，因此，在临床试验阶段用于支持控制策略的数据会比上市注册阶段少。鼓励根据工艺的化学基础，基于风险，优先分析最可能出现在原料药或制剂中的杂质。出现可能性很低的杂质早期临床阶段可不要求分析数据；但相同情况下，上市申请时宜提供分析数据以支持其控制策略。商业化处方实际上是在临床研发后期才确定的，因此在早期阶段与制剂降解产物相关的研究会比较有限。</p><h2>9. href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E7%BD%AE%E4%BF%A1%E5%8C%BA%E9%97%B4">置信区间</a> 下限 10%（BMDL10，有95%的概率确信预估的最低剂量不会导致啮齿类动物的致癌率超过 10%）。通过简单地将 BMDL10 除以 10000 来线性外推至十万分之一的发生率（即终生风险水平），即得。</p><p>特定化合物的可接受摄入量也可以使用国际公认机构公布的推荐值，如世界卫生组织（WHO，国际化学品安全规划署[IPCS]癌症风险评估程序），以及其它合理使用的十万分之一终生风险水平的值来推导。一般来说，应用的监管限度应以最新的科学支持性数据和/或方法为基础。</p><p><i>注 5</i> 对于化学结构与已知致癌物类似的致突变杂质（没有致癌性数据），可以采用化合物特异性的方法来计算致突变杂质的可接受摄入量。例如，单官能团烷基氯化物为药物合成中常见物质，现已明确其致癌强度相关因素（参考文献 15），可用于制定所有单官能团烷基氯化物的安全可接受摄入量。与多官能烷基氯化物相比，单官能团烷基氯化物的致癌性较低，TD <sub>50</sub> 值介于 36 至 1810 mg/kg/天之间（n=15；具有两个明显不同官能团的表氯醇除外）。TD <sub>50</sub> 值 36 mg/kg/天则可以作为一个非常保守的特定类的效价参考点，来计算单官能团烷基氯化物的可接受摄入量。该效价水平至少比与默认的终生 TTC（1.5 μg/天）对应的TD <sub>50</sub> （1.25 mg/kg/天）低 10 倍，因此证明了单官能团烷基氯化物的终生和短于终生的每日摄入量是默认值的 10 倍。</p><p><i>注 6 </i>在建立临床研发阶段性 TTC 限度时，优先建立药品中致突变杂质短于终生的可接受摄入量（参考文献 16）。短于终生的可接受摄入量（ <a limit）：</b></p><p>从可接受摄入量和药物每日剂量推导而来的原料药或制剂中杂质的最大可接受浓度。</p><p><b>可接受标准（Acceptance criterion）：</b></p><p>为了接受特定分析方法的检测结果而设定的量化限度、范围或其他合适的测定手段。</p><p><b>控制策略（Control strategy）：</b></p><p>控制策略是基于对当前产品和对工艺的理解而制订的一套用以保证工艺性能和产品质量的有计划的控制方法。控制方法可包括与原料药、制剂物料和各组分相关的参数和属性、设施/设备的操作条件、过程控制、终产品质量标准、监测和控制的相关方法和频率。</p><p><b>累积摄入量（Cumulative intake）：</b></p><p>人体暴露一段时间后对某物质的总摄入量。</p><p><b>降解产物（ <a href="https://lib.shilinx.com/wiki/index.php?title=%E5%88%86%E7%B1%BB:%E9%99%8D%E8%A7%A3%E4%BA%A7%E7%89%A9">Degradation Product</a> ）：</b></p><p>药物分子受时间和/或受光、温度、pH、水或与辅料的作用和/或直接包装/密闭系统的影响发生化学变化而产生的物质。</p><p><b>DNA 反应性（DNA-reactive）：</b></p><p>通过与 DNA 发生化学反应直接诱导 DNA 损伤的潜在能力。</p><p><b>专家知识（Expert knowledge）：</b></p><p>在本指导原则中，专家知识可定义为综述已存在的数据并使用任何其他相关信息来评估计算机模型预测致突变性的精确度。</p><p><b>遗传毒性（Genotoxicity）：</b></p><p>一个广泛的术语，指的是遗传物质中的任何有害变化，而不考虑诱发该变化的机制。</p><p><b>杂质（Impurity）：</b></p><p>原料药或制剂中任何不是原料药或辅料的成分。</p><p><b>致突变杂质（Mutagenic impurity）：</b></p><p>已在合适的致突变试验模型（如细菌致突变试验）中确证具有致突变性的杂质。</p><p><b>定期确认性检测（Periodic verification testing）：</b></p><p>在 <a href="https://lib.shilinx.com/wiki/index.php?title=ICH_Q6A_Specifications_:_Test_Procedures_and_Acceptance_Criteria_for_New_Drug_Substances_and_New_Drug_Products:_Chemical_Substances_199910">ICH Q6A</a> 也称为定期检测或跨批检测。</p><p><b>(Q)SAR 和 SAR（(Q)SAR and SAR）：</b></p><p>在本指导原则中，是指使用（定量）构-效关系的实验数据说明化合物分子（亚）结构与致突变活性之间的关系。</p><p><b>清除因子（Purge factor）：</b></p><p>清除反映了一个工艺降低杂质水平的能力。清除因子的定义是指工艺中上游点的杂质水平除以工艺中下游点的杂质水平。清除因子可进行测定或预测。</p><p><b>警示结构（Structural alert）：</b></p><p>在本指导原则中，与致突变性相关的化学基团或分子（亚）结构。</p><h2>参考文献</h2><p>1. 是一个稳定的化合物，会被带入原料药。在实验室规模下，将不同浓度的杂质 A 加至中间体 X 中进行加样研究。如果研究发现即使在中间体 X 中有 1%的杂质 A，原料药中杂质 A 也能持续被清除至 TTC 限度的 30%以下。由于中间体 X 的形成离原料药只有 2 步，中间体 X 中杂质 A 的水平相对较高，通过检测多批中试规模批次原料药中的杂质 A 水平来确认了工艺的清除能力，结果均在 TTC 限度的 30%以下。因此，将中间体 X 中杂质 A 控制在 1.0% 的可接受限度是合理的，无需在原料药标准中对该杂质进行检测。</p><h3>案例 2：方法 3 控制策略的示例：基于采用标准分析方法的掺杂试验所获得的预期清除数据</h3><p>在一个 5 步合成工艺中，起始物料 Y 在第 3 步引入，采用标准分析方法在起始物料 Y 中常规检测杂质 B 均低于 0.1%。为了确定起始物料中 0.1%的标准是否可接受，在实验室规模下进行了清除研究，将不同浓度（最高 10%）的杂质 B 加至起始物料 Y 中，通过最后 3 步工艺步骤，确定清除因子&gt;500 倍。起始物料 Y 中杂质 B 的质量标准为 0.1%，应用该清除因子，原料药中杂质 B 的预期水平将低于 2 ppm。由于该值低于原料药中该杂质的 TTC 限度 50 ppm，起始物料 Y 中杂质 B 的质量标准为 0.1%是合理的，无需提交中试规模或商业规模批次的原料药批次数据。</p><h3>案例 3：方法 2 和方法 4 控制策略示例：结构相似的致突变杂质的控制</h3><p>在一个 5 步合成工艺中，第 1 步中间体是一个硝基芳香类化合物，可能含有低水平的杂质 C，杂质 C 是第 1 步中间体的位置异构体且也是硝基芳香类化合物。采用常规分析方法，第 1 步中间体中杂质 C 未检出，但杂质 C 可能以较低的水平存在。第 1 步中间体的细菌突变试验呈阳性。第 2 步氢化反应中第 1 步中间体 99%转化为相应的芳香胺。这通过过程检测予以确认。对残留的第 1 步硝基芳香类中间体的清除情况进行了评估，根据后续第 3 步和第 4 步工艺步骤中的清除点，预计对第 1 步中间体有一个较高的清除因子。预计第 5 步工艺步骤中不会清除第 1 步中间体，因此在第 4 步中间体中建立基于 TTC 限度的第 1 步中间体质量标准（方法 2 控制策略）。位置异构体杂质C 是预计通过与第1 步中间体相同的清除点来清除， 因此会一直远远低于第 1 步中间体的水平；因此，无需对杂质 C 进行检测，且杂质 C 应用方法 4 控制策略时也可由上述理由支持，无需提交额外的实验室规模或中试规模数据。</p><h3>案例 4：方法 4 控制策略示例：高活性杂质</h3><p>二氯亚砜是一种致突变性的高活性化合物。该试剂在一个 5 步合成工艺的第 1 步中引入。在合成过程中，多处使用了大量的水。由于二氯亚砜遇水立即发生反应，因此不可能有二氯亚砜残留至原料药中。这时适合选用方法 4 控制方式，而无需提交任何实验室规模或中试规模的数据。</p><h3>指导原则实施：</h3><p>鼓励 M7
