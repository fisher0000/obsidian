| 用法用量                                                                                                                                                                                                                                                                                 | 可供静脉和肌肉注射。成人常用量：轻中度感染，一次1-2g，每12小时一次；重度感染，一次2-3g，每8小时一次。接受血液透析时，透析后应予一次剂量。成人每日剂量一般不超过9g，但在免疫缺陷病人有严重感染时，剂量可加至每日12g。<br><br>小儿常用量：婴儿的剂量尚未确定。婴儿和儿童每日按体重接受25-200mg/kg的剂量未见副反应发生，但新生儿的t1/2时间延长，因此，剂量必需予以调整。用于预防手术感染时：应在手术开始前30-90分钟静脉给予1或2g，此剂量每12小时可重复一次，对于多数病例，预防给药持续时间应不超过24小时。本品的预防可持续至手术完成后72小时。                                                                                                                                                                                                            |     |
| 立题依据                                                                                                                                                                                                                                                                                 | 临床不可替代性：头孢哌酮钠作为第三代头孢菌素，对铜绿假单胞菌（绿脓杆菌）具有极强的抗菌活性，且约 70%~80% 经肝胆系统排泄，使其在肝胆系统感染（如急性胆囊炎、胆管炎）中拥有其他头孢菌素（如头孢他啶、头孢曲松）难以企及的药代动力学（PK）优势。<br><br>存量巨大的复方制剂市场：头孢哌酮钠主要以复方制剂（舒巴坦钠或他唑巴坦钠）形式应用。2025 年数据显示，该品种在中国公立医疗机构的年销售额仍维持在数十亿元至百亿元级别。立项原料药是保障下游大品种制剂供应稳定性的核心举措。<br><br>技术壁垒高：目前国内原料药普遍面临高分子聚合物（过敏原）控制不稳定的痛点。立项研发高纯度、低杂质（采用复合催化剂体系）的 API，可作为产品差异化的核心卖点，尤其在针对 ICU 重症患者的高端市场中更具竞争力。其化合物、晶型专利、制备专利均已过期，市场工艺成熟，可进行仿制。<br><br>（以上内容来自Gemini 3）                                                                                 |     |
