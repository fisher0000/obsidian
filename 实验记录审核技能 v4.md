## 6. 信息提取与填入

从识别内容中提取以下字段：

| 提取字段  | 表格字段   | 提取规则                                       |
| ----- | ------ | ------------------------------------------ |
| 日期    | 日期栏    | 匹配日期格式（YYYY-MM-DD、YYYY/MM/DD、YYYY年MM月DD日等） |
| 记录本编号 | 记录本编号栏 | 匹配"编号"、"记录本编号"等关键词后的内容                     |
| 实验地点  | 实验地点栏  | 匹配"实验地点"、"地点"后的内容                          |
| 实验主题  | 实验主题栏  | 匹配"实验主题"、"主题"后的内容                          |
| 实验人   | 实验人栏   | 匹配"实验人"、"操作人"、"记录人"后的内容                    |
| 页码    | 页码栏    | 匹配"第X页"、"Page X"等格式                        |
| 复核人   | 复核人栏   | 匹配"复核人"、"审核人"后的内容                          |

**填入操作：**

- 使用 `feishu_bitable_app_table_record` 工具的 `update` 操作

- 将提取的字段填入对应表格栏

- **注意：** 如果某字段识别不出内容，相应栏位留空（不填）

## 9. 执行16项审核检查

**（对所有识别到的内容均执行审核，不管是否匹配步骤7的项目）**

| 序号  | 审核项  | 检查内容                  | 检查方法                          |
| :-: | ---- | --------------------- | ----------------------------- |
|  a  | 物料信息 | 物料是否已注明所用的数量、来源以及批号   | 检查是否包含"物料"、"批号"、"数量"等关键词及对应数值 |
|  b  | 书写工具 | 是否使用蓝色水笔书写            | OCR无法直接判断，标记为"需人工核实"          |
|  c  | 环境记录 | 温度、湿度是否已书写            | 检查是否包含"温度"、"湿度"、"℃"、"RH"等关键词  |
|  d  | 基本信息 | 实验主题、实验地点是否已书写        | 检查步骤6是否成功提取对应字段               |
|  e  | 日期格式 | 日期格式是否正确 (YYYY-MM-DD) | 验证日期字段是否符合标准格式                |
|  f  | 表格规范 | 绘制表格线条是否工整            | OCR无法直接判断，标记为"需人工核实"          |
|  g  | 修改规范 | 书写有误是否正确修改            | 检查是否有"划改"、"修改"等标记及签名日期        |
|  h  | 续页标识 | 后接第二页的记录是否有标明         | 检查是否包含"续页"、"接上页"等标识           |
|  i  | 骑缝签名 | 黏贴纸张处是否书写骑缝名、日期       | 检查是否包含"骑缝"、"骑缝签名"等关键词         |
|  j  | 签名完整 | 操作人、复核人是否已签名          | 检查步骤6是否提取到实验人和复核人             |
|  k  | 空格处理 | 记录是否划掉空格              | OCR无法直接判断，标记为"需人工核实"          |
|  l  | 批号规范 | 批号是否编制正确              | 检查批号格式是否符合规范（如包含日期、项目代码等）     |
|  m  | 仪器记录 | 是否注明主要仪器设备的型号和编号      | 检查是否包含"仪器"、"设备"、"型号"、"编号"等关键词 |
|  n  | 数据真实 | 实验过程的数据是否如实记录         | 检查是否有数值记录、数据表格等               |
|  o  | 结果完整 | 实验结果和实验结论是否已书写        | 检查是否包含"结果"、"结论"、"实验结果"等章节     |
|  p  | 及时填写 | 实验记录是否按时填写（滞后一周内）     | 比较实验日期与记录日期，判断是否超过7天          |

**审核结果格式：**

  
  
  
实验记录审核结果：  
━━━━━━━━━━━━━━━━━━━━  
✅ 通过项：  

- 物料信息已注明
    

- 环境记录完整（温度、湿度已记录）
    

- 基本信息完整
    

- 日期格式正确...
    

❌ 不通过项：  

- 未注明仪器设备型号和编号
    

- 实验结论未书写
    

⚠️ 需人工核实项：  

- 是否使用蓝色水笔书写
    

- 表格线条是否工整━━━━━━━━━━━━━━━━━━━━
    

**操作：**

- 将审核结果文本填入多维表格的 **"审核结果"** 栏

- 使用 `feishu_bitable_app_table_record` 工具的 `update` 操作

## 异常处理

| 异常情况     | 处理方式                   |
| -------- | ---------------------- |
| 文件夹扫描失败  | 回复"无法访问云盘文件夹，请检查链接权限"  |
| OCR识别失败  | 记录该文件为"识别失败"，继续处理下一个   |
| 多维表格写入失败 | 记录错误日志，继续处理下一个文件       |
| 云文档创建失败  | 记录错误，跳过电子实验记录创建，继续审核流程 |
| 文件移动失败   | 记录错误，不影响审核流程继续执行       |
| 无符合条件的文件 | 回复"文件夹中没有找到图片或PDF文件"   |
